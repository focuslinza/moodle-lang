L'éditeur HTML incorpore une interface semblable à celle des traitements de texte qui permet de facilement éditer un texte sans utiliser directement les balises HTML.

En plus du formatage du texte, l'éditeur permet :

Coller du texte en provenance d'une autre application

    Vous pouvez copier et coller du texte formaté dans d'autres applications Windows, telle Word, directement dans cet éditeur et tout le formatage sera conservé. On utilise les commandes Copier et Coller du fureteur (Ctrl-C, Ctrl-V).

Insertion d'images

    Vous pouvez insérer des images déjà publiées dans un site Internet en cliquant sur le bouton «Insérer image» et en entrant l'adresse URL de l'image.

Insertion d'un tableau

    On peut insérer un tableau dans le texte en utilisant le bouton correspondant.

Insertion d'un lien hypertexte

    Pour insérer un lien hypertexte, tapez tout d'abord le texte qui servira de lien. Sélectionnez ce texte et cliquez sur le bouton pour l'insertion d'un lien. Entrez ensuite l'adresse URL désiré.

Insertion de binettes

    Pour incorporer des binettes à votre texte, cliquez sur l'icône sourire dans la barre d'icônes. Une fenêtre s'ouvrira qui fait la liste des binettes disponibles. Cliquez sur une des binettes ou notez le code correspondant que vous pourrez entrer directement dans votre texte.
