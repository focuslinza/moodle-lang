Tests

    Ce module permet au professeur de concevoir et d'inclure des tests dans son cours. Plusieurs types de questions sont offerts : choix multiples, question à appariements, vrai ou faux, réponse courte, réponse numérique, etc. Ces questions sont conservées dans une base de données classée par catégorie, et peuvent être utilisées à plusieurs reprises dans le cours. Les catégories de question peuvent même être publiées, ce qui les rend disponibles aux autres professeurs.

    Le professeur peut autoriser les tentatives multiples pour faire le test. Chaque tentative est automatiquement évaluée. Il peut choisir de faire afficher une rétroaction pour chaque question et de montrer les réponses correctes.
