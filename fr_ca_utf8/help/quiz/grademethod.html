Calcul de la note du test

Lorsqu'on permet à l'étudiant des tentatives multiples pour faire le test, vous devez choisir comment sera calculé la note finale de l'étudiant. Voici les choix possibles :

La note la plus élevée

    La note finale est la meilleure note obtenue dans toutes les tentatives.

Note moyenne

    La note finale est la note moyenne (moyenne simple) de toutes les tentatives.

Première note

    La note finale est la note obtenue à la première tentative.

Dernière note

    La note finale est la note obtenue à la dernière tentative.
