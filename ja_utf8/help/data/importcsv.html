CSVファイルからインポート

CSVは、Comma-Separated-Values ( カンマで区切られた値 ) を意味し、テキストインターチェンジで一般的なフォーマットです。

ファイルフォーマットは、最初のレコードにフィールド名の一覧がある通常のテキストファイルです。次に1行1レコードのデータが続きます。

フィールドデリミタは、デフォルトでカンマ、またフィールドエンクロージャはデフォルトでセットされません。フィールドエンクロージャは、それぞれのレコードにおいて各フィールド囲む文字です。

レコードは、新しい行で区切られます ( 通常、あなたのテキストエディタの RETURN または ENTER で生成されます )。タブは、\t で指定し、新しい行は \n で指定します。

サンプルファイル: 

  name,height,weight
  Kai,180cm,80kg
  Kim,170cm,60kg
  Koo,190cm,20kg

このスクリプトの初期のバージョンでは、シンプルなフィールドのみサポートしますが、将来的に全てのフィールドタイプが追加されます。
