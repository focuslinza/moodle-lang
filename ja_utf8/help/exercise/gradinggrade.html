学生が評価に使用する評点

これは、自分または他の学生の課題を評価するため、学生に与える最大評点です。つまり、学生が自己評価するときに使用する評点です。評価に使用する実際の評点は、同じ提出課題の教師による評点と比較しながら、エクササイズモジュールにより計算されます。この評点は「評定点」であり、課題に与えられる最大評点、「提出課題のための評点」ではありません。

学生のエクササイズに対する評点は、この評点と提出課題のための評点との合計となります。従って、学生の評価に関する ( 最大 ) 評点が20に設定され、提出課題の ( 最大 ) 評点が80に設定された場合、エクササイズの ( 最大 ) 評点は100になります。

この値は、いつでも変更することができます。変更後すぐに学生 ( および教師 ) が結果を見ることができます。
