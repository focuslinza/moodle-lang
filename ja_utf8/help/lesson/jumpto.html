リンク先へジャンプ

それぞれの ( 問題の ) 答え、または ( ブランチページの ) 説明は、ジャンプするリンク先を持ちます。この答えが選択された場合、その答えに対するレスポンスが表示されます。その後、学生にはジャンプしたリンク先が表示されます。このリンクは相対リンク、絶対リンクのどちらでも選択することができます。相対リンクは、このページ と 次のページです。 このページでは、学生に現在のページが再度表示されます。次のページでは、論理的順番に従って、このページに続くページが表示されます。絶対リンクのページは、ページのタイトルを選択することで指定します。

ページが移動された後では、( 相対的な ) 次のページへのジャンプはリンク先が異なることに注意してください。これに対して、ページタイトルを使うジャンプでは、ページが移動された後でも常に同じページが表示されます。

特別ジャンプ

ブランチ内の未閲覧の問題

このリンクでは、このブランチテーブルとレッスン終了または次のブランチ終了間の ( この受験で学生による ) 未閲覧の問題をランダムに選択します。

ブランチ内のランダム問題

このリンクでは、現在のブランチテーブルとレッスン終了または次のブランチ終了間の問題をランダムに選択します。学生がすでに問題を閲覧して受験回数が1回以上の場合、その問題で点数を取る更なるチャンスを得ます。受験回数が1回に設定された場合、問題はスキップされ、別のランダム問題が表示されます。

ランダムブランチテーブル

ここでは、現在のブランチテーブルおよびレッスン終了または次のブランチ終了間のランダムブランチへジャンプします。
