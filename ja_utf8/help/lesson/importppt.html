PowerPoint HTMLのインポート

使用方法

全てのPowerPointスライドは、前および次の答えを持った分岐テーブルとしてインポートすることができます。

  1. PowerPointプレゼンテーションを開いてください。
  2. ウェブページ ( 特別オプション無し ) として保存してください。
  3. ステップ3では、全てのスライドがウェブページにコンバートされ、htmlファイルおよびフォルダが作成されます。
    フォルダをzip圧縮してください。
  4. あなたのmoodleサイトにアクセスして、新しいレッスンを追加してください。
  5. レッスン設定の保存後、「最初に何をしますか?」の下に4つのオプションが表示されますので、「PowerPointのインポート」をクリックしてください。
  6. 「Browse...」ボタンをクリックして、ステップ3で作成したzipファイルを指定してください。そして、「ファイルをアップロードする」をクリックしてください。
  7. 処理が正常に行われた場合、次の画面では「続ける」ボタンが表示されます。

PowerPointにイメージが含まれる場合、イメージはコースファイルとしてmoddata/XYに保存されます。Xはレッスン名、Yはナンバー ( 通常 0 ) です。また、インポート処理中にmoodleデータディレクトリtemp/lessonの中にファイルが作成されます。これらのファイルは、現在のところimportppt.phpで削除されることはありません。
