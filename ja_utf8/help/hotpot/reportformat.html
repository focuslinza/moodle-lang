レポートフォーマット

ここではHot Potatoesレポートの異なる出力フォーマットを指定することができます。

Report Format

HTML
レポートはウェブブラウザの表示に適するようにHTMLフォーマットで生成されます。

Excel
レポートはMicrosoft Excelのようなスプレッドシートプログラムで開くことのできる「.xls」ファイルとして生成されます。

Text
レポートはテキストエディタで開くことのできるテキストファイルとして生成されます。

エンコーディング

スプレッドシートプログラムが必要とする場合、レポート値の変換に使用するエンコーディングリストです。 例えば、Microsoft Excelを日本語で使用する場合は「SJIS」エンコードをする必要があります。 利用可能なエンコーディングのリストはサイト管理者により、HotPotモジュールの「設定」ページで調整することができます。

データの折り返し

Yes
長いデータ値はテーブルセルのサイズに合うように折り返されます。 列が非常に長い場合もあります。

No
データ値は折り返されません。

問題内容の表示

Yes
メインレポートでは問題と答えは文字で表されます。どの文字が、どの問題または答えを表しているのか表示するため、問題内容が生成されます。

No
レポートでは問題と答えが完全な状態で表示され、問題内容は生成されません。
