ප්‍රවේශ වීමේ හැකියාව

මෙම අඩවිය විවිධ බ්‍රව්සර සහ screen reader මෘඳුකාංග සමඟ පරික්ෂණයට භාජනය කර ඇත.

අන් අය අතරින් screen reader පරිශීලකයන්ට මූඩ්ල් පද්ධතිය පහසුකම් සපයන්නේ, මූඩ්ල් Semantic Markup යොදා ගනිමිනි. — වෙබ් අඩවි, වෙබ් පිටු, කොටු (block) සිරස්තල <h1> <h2> සංචාලන කොටු ලැයිස්තුගත වීම <ul>ආදි පහසුකම් ඒ යටතට ගැනේ.

මූඩ්ල් World Wide Web Consortium's [වෙබ් අන්තර්ගතයන්ට ප්‍රවේශවීම සඳහා වූ උපදෙස් මාලා 1.0] පළමු මට්ටම සහ බොහෝ විට දෙවැනි මට්ටම [Double -A comformance] එමෙන් ම සමහර අවස්ථාවලදී තෙවැනි මට්ටමට ද අනුගත වන ලෙස නිර්මාණය කර ඇත. මේ මාර්ගෝපදේශය මඟින් අප බලාපොරොත්තු වන්නේ, ප්‍රාදේශීය වශයෙන් වෙනත් හැකියා ඇති අයවලුන් (disabilities-වෙනත් හැකියාවන් ඇති අය) සඳහා ඇති ප්‍රවේශවීමේ අයිතිය ආශ්‍රිත නීති රීතිවලට සරිලන ආකාරයට පද්ධතිය පවත්වා ගැනීමයි. දැනටමත් සමහර වගු සඳහා 'layout' යොදා ගනී. නමුත් සාමාන්‍ය ඉදිරිපත් කිරීම් වලදී ( CSS යෙදා ගනිමින්) අන්තර්ගතයෙන් වෙන් කර ඇත.

ඔබ මේ ගැන සැලකිල්ලක් දක්වන්නේ නම් හෝ එවැනි අපහසුතා ඇත්නම්, කරුණාකර අපගේ පද්ධති පරිපාලකට දන්වන්න. ඒ සඳහා මෙම සම්බන්ධකය යොදා ගන්න.

ප්‍රවේශ යතුර

මේ අවස්ථාවේ දී අප ප්‍රවේශ යතුර දක්වන්නේ, නැත. නමුත් අනාගත මූඩ්ල් සංස්කරණවලදී මේ ප්‍රවේශ යතුර යොදා ගැනීමට බලාපොරොත්තු වේ.

සම්බන්ධකයන්

දෙපස වූ කොටුවලට පෙර පිහිටා ඇති සම්බන්ධකයකි. මේ මඟින් screen reader පරිශීලකයන්ට, කොටු මඟහැරීම සඳහා මං පාදයි. මෙම සම්බන්ධකය රූපමය බහුමාධ්‍ය බ්‍රව්සරවලදී (උදාහරණ Internet Explore, Fire Fox ආදී) දක්නට නැත.

බොහෝ සම්බන්ධක සඳහා සිරස්තල විශේෂණ (attribute) ඇත. එම සම්බන්ධකය හොඳින් විස්තර කරන්නේ, සම්බන්ධ කර ඇති පෙළ එහි අන්තර්ගතය විස්තර නොවන විටෙක දී ය.

සම්බන්ධකයට ප්‍රවේශ යතුරක් පවතීනම්, මේ බව සම්බන්ධකයේ සිරස්තලය තුළින් නිවේදනය කරයි. හැකි සෑම විටෙක දී ම, සම්බන්ධකය පෙළ මඟින් එහි අන්තර්ගතය දක්වන පරිදි යතුරු ලියනය කළ යුතු යි. එහි javascript අන්තර්ගත වී නොමැති සම්බන්ධක (අසත්‍ය සම්බන්ධක-pseudo-links) විය යුතු ය.

රූපසටහන්

සෑම රූපයකට ම 'alt' විශේෂණය යොදා ගනිමින් රූපය විස්තර කරයි. රූපය සැරසීමට හෝ අලංකරණයට පමණක් තිබේනම්, 'alt' විශේෂණය හිස්ව තබන්න. සිතියම් ආදී සංකීර්ණ තොරතුරු රූපසටහන් සඳහා 'longdess' විශේෂණය යොදා ගනිමින්, එය විස්තර කරන පෙළවැලකට සම්බන්ධ කරන්න.

සම්මත Complaince

මෙම අඩවිය නිර්මාණය කර ඇත්තේ, අනුමත HTML කේතවලට අනුකූලව ය. එසේ ම දර්ශනය වීම සඳහා CSS යොදා ගනී.
