HTML ප්‍රවර්ගය

පහත ඒවා අනුගමනය කිරීමට ඔබට අවස්ථාව ඇත:

HTML නැත
සියලුම HTML-ටැග මගහරින්න. හැඩතල ගැන්වීම විකී වචනවලින් පමණක් සිදු කරයි. පෙරණයක් මේ ප්‍රවර්ගයේ දි භාවිත කරයි.
පරිස්සම් ලෙස HTML භාවිත කිරීම
සමහර ටැග පමණක් භාවිත කළ හැකි අතර, පෙරණය භාවිත නොකරයි.
HTML පමණයි
විකී වචන නොමැතිව, HTML පමණක් භාවිත කරයි. එහිදී HTML-සංස්කරණය භාවිත කිරීමට අවසර ඇත.
