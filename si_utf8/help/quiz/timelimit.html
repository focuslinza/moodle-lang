කාල සීමාව

පෙරනිමියයෙන් ප්‍රශ්නාවලියේ කාල සීමාව සීමා කර නැත. මේ නිසා සිසුන් ට අවශ්‍යය තරම් කාලය ගෙන ප්‍රශ්නාවලිය සම්පුර්ණ කළහැකි ය.

කාල සීමාව, සඳහන් කළවි‍ට ඔබ‍ කාරුණු කීපයක් ගැන අවධනය කළ යුතු ය. එම කරුණු නම්:

  • බ්‍රව්සරයේ Java Script වලට සහයෝගය දැක්විය යුතුයි.(වේලාගණකය නිවැරදිව ක්‍රියා කිරීම‍ට සහයෝගය ලබා දේ.)
  • පාවෙන වේලාගණකය පෙන්විය යුතුයි.
  • කාල සීමාව අවසන් වු පසුව ස්වයංක්‍රීයව ලබා දී ඇති පිළිතුරු පද්ධතිය ලබාගනී.
  • ‍කෙසේ හෝ ශිෂ්‍යයෙකුට ලබා දී අති අගය‍ට වඩා කාලය වංචා කර (තත්පර 60ට ‍‍වඩා) ඇති විට, සියලු ප්‍රශ්න සඳහා ලකුණු 0 ක් ලැබේ.
