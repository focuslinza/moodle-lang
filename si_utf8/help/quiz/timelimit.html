කාල සීමාව

පෙරණිමියේන් ප්‍රශ්නාවලියේ කාල සීමාව සීමා කර නැත. මේ නිසා සිසුන් ට අවශ්‍යය තරම් කාලය ගෙන ප්‍රශ්නාවලිය සම්පුර්ණ කල හැකිය.

කාල සීමාව, සදහන් කළවි‍ට ඔබ‍ට කාරණා කීපයක් ගැන අවධනය කළයුතුය. මේ කරුණු නම්:

    • බ්‍රව්සරයේ Java Script වලට සහයෝගය දැක්විය යුතුයි.(වේලාගණකය නිවැරදිව ක්‍රියා කිරීම‍ට සහයෝගය ලබා දේ.)
    • පාවෙන වේලාගණකය පෙන්විය යුතුයි
    • කාල සීමාව අවසන් වු පසුව ස්වයංක්‍රීයව ලබා දී ඇති පිළිතුරු පද්ධතිය ලබාගනී.
    • ‍කෙසේ හෝ සිසු‍වකු ලබා දී අති අගය‍ට වඩා කාලය වංවා කර( තත්පර 60 ‍‍වඩා)අති විට, සියලු ප්‍රශ්න සදහා ලකුණු 0 ලැබේ.
