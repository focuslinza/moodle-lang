පරිගණක ජාල ලිපිනයක් අවශ්‍යයි.

මෙය ද විකල්ප සැකසුමකි. ඔබ‍ට යම් උප-ජාලයක් (subnet) හරහා පමණක් ප්‍රශ්නාවලියට පිළිතුරු දීමට අවශ්‍යනම්, ඔබට අදාල IP ලිපිනයන් පූර්ණව හෝ කොටසක් ලබා දීමෙන් පහසුවෙන් ප්‍රශ්නාවලිය සීමා කළ හැකි ය.

මෙම පහසුකම "proctored" ප්‍රශ්නාවලිය සඳහා ඉතා ප්‍රයෝජනවත් වේ. එහිදී ඔබට මෙය සීමිත ස්ථානයක සිටින පිරිසකට පමණක් මෙය ඉදිරිපත් කළ හැකි ය.

තුන් ආකාරයක TP ලිපිනයන් භාවිත කළ හැකි ය.

1. පූර්ණ IP ලිපිනය උදාහරණ: 192.248.73.160 (එය යම් පරිගණකයක් සඳහා පමණක් වේ, නැතහොත් එය proxy සේවාදායකට අයත් ජාලයකි.)

2. අසම්පූර්ණ IP ලිපින: උදාහරණ 192.248. මෙහිදී මේ කොටස ගැලපෙන ඕනෑ ම IP ලිපිනයක් ඇති පරිගණකයක සිට ප්‍රශ්නාවලිය ලබා ගත හැක.
