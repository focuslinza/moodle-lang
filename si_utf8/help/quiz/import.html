නව ප්‍රශ්නමාලාවක් ආයාත කිරීම

මෙම ක්‍රියාකාරකම, බාහිර ගොනුවලින් ප්‍රශ්න සමූහයක් පද්ධතියට පෝරමයක් මාර්ගයෙන් හෝ ගොනු පෙදෙසේ දැනටමත් පවතින ගැලපෙන ගොනුවක් ලෙසින් ආයාත කිරීමට ඉඩ සලසයි.

අක්ෂර කේතනය

ඔබේ ගොනුවේ Ascii නොවන කේත අඩංගුවේ නම්, ගොනුව UTF -8 කේතනය කරන්න. එය පහසුවෙන් ගොනුව සුරක්ෂිත කරන අවස්ථාවේ දී හෝ ගොනුව සංස්කරණයෙන් නිර්යාත කිරීමේ දී කළ හැකි ය. XML පාකද වූ ගොනු මූඩ්ල් සේවාදායකය PHP S සහයෝගය සහිත ක්‍රියාත්මක වන විට UTF -8 බවට ස්වයංව කේතනය වේ.

සහයෝගය දක්වන හැඩතල

GIFT හැඩතලය

පෙළ ගොනුවක්( a text file) ලෙස මූඩ්ල් ප්‍රශ්නාවලියක් ආයාත(import) කිරීමට ඇති වඩාත් විශ්වාසදායි ක්‍රමය, මෙම හැඩතලය(format) වේ. GIFT  හැඩතලය, බහුවරණ, කෙටි පිළිතුරු, ගැලපීම් සහ අංක ගණිතමය ප්‍රශ්න මෙන්ම හිස්තැන් පිරවීම වැනි බහුවිධ ප්‍රශ්න වර්ගයන්ට සහයෝගය දක්වයි.‍ මෙම බහුවිධ ප්‍රශ්න, මිශ්‍රව තනි පෙළ ගොනුවක තැබීම, ප්‍රශ්නය නම් කිරීම, පසු විපරම් සහ ප්‍රතිශත - බර තැබීම් ( Percentage - weight) වැනි කාරණා සඳහා ද සහයෝගය දක්වයි.

උදාහරණ:

  •  {~ ඝණයක් ලෙස, ~ ද්‍රවයක් ලෙස, = වායුවක් ලෙස,} නයිට්‍රජන් කාමර උෂ්නත්වයේ දී පවතී. පිළිතුර නැවතී‍මේ තිතට පෙර පැවතියහොත් හිස්තැනක් ආදේශ වේ. (ඉහත උදාහරණය බලන්න.)
  • නයිට්‍රජන් කාමර උෂ්නත්වයේ දී, {~ ඝණයකි. ~ ද්‍රවයකි. = වායුවකි.}
  • වායුගෝලයේ වැඩිම ප්‍රතිශතයක් ලෙස පවතින වායුව කුමක් ද? {= N, = N2, = නයිට්‍රජන්}

වැඩි විස්තර

Aiken හැඩතලය

අයිකෙන් (Aiken) හැඩ තලය(format), බහුවරණ ප්‍රශ්න තැනීමේ දී, ඉතාමත් සරල ව කියවිය හැකි ලෙස යොදා ගත හැකි ක්‍රමයකි.
උදා: මේ ප්‍රශ්න ‍මේ ප්‍රශ්න සඳහා නිවැරදි පිළිතුර කුමක් ද?

A) මෙයයි?
‍B) මෙය විය හැකිය?
C) සමහර විට මෙය විට හැකිය?
D) මෙය ම වේ!

පිළිතුර : D

මේ ප්‍රශ්න තනි පේළියක තිබීම අවශ්‍යය වේ. සෑම පිළිතුරක් ම අරම්භ විය යුතේ තනි අකුරකිනි. පසුව තිතක් ද "." හෝ " ( )" වරහණක් හෝ ඉන් පසුව හිඩැසකින් සමන්විත විය යුතුයි.

නිවැරදි පිළිතුර ආරම්භ වියයුත්තේ "ANSWER:" සමඟය. පසුව අදාළ අකුර ලියනු ලබයි.

වැඩි විස්තර

මූඩ්ල් XML හැඩතලය

එක ම හැඩතලයකින් පෙ‍ර නිර්යාත කළ ගොනු නැවත ආයාත කිරීම සඳහා පහසුකම් සපයයි. මෙම හැඩතලය, කාර්යක්ෂම, කැටගරියක් මූඩ්ල් XML නිර්යාත කිරීමට විශ්වාසදායක ම ක්‍රමයක් වේ. මෙම XML හැඩතලයෙන් රූප සටහන් ද ආයාත කිරීමේ හැකියාව ද පවතී.

මේ අයුරින් ම WebCT, IMS, QFI වැනි නව හැඩතලයන් සඳහා ද මොඩියුල් පැමිණීමට නියමිත ය.

වැඩි විස්තර

බ්ලැක්බෝර්ඩ් (Blackboard)

‍කළුලෑල්ල නම් වූ මෙම නිර්යාත හැඩතලයෙන් සුරක්ෂිත කළ ගොනුවක් ආයාත කළ හැකි ය. එය රඳා පවතින්නේ මූඩ්ල් සේවාදායකයට XML කෘත්‍යය PHP සම්පාදනය (Compile) කර ඇති විටයි. මෙය Blackboard සංස්කරණ 6 සහ පසු ඒවායේ ආයාත කිරීම සඳහා සහයෝගය නොදක්වයි.

බ්ලැක්බෝර්ඩ් වී සික්ස් ප්ලස් (Blackboard V6+)

මෙම මොඩියුලය, Blackboard සංස්කරන 6 හෝ ගොනු ආයාත කිරීමේ පහසුකම සීමිතව සලසා දෙයි. එය ද PHP සමඟ XML සම්පාදනය මත රඳා පවතී. එය ඔබේ පද්ධතියට උඩුගත කිරීමේ දී හකුලන ලද Zip වර්ගයේ ගොනුවක් ලෙස උඩුගත කළ යුතු ය.

වෙබ් සී ටී (Web CT)

මෙම මොඩියුලය Web CT පෙළ පාදක කරගත් හැඩතල ආයාත කිරීමට ඉඩ සලසයි.

වැඩි විස්තර

පාඨමාලාවක පරීක්ෂණ කළමනාකරු (Course Test Manager) CTM

මෙම මොඩියුලය, CTM තැන්පතුවලින් සුරක්ෂිත කරන ලද ආයාත කිරීමට පහසුකම් සපයයි. එම දත්ත ගොනු පවතින්නේ, මයික්‍රොසොෆ්ට් ඇක්සෙස් දත්ත සමුච්ඡයේ නම්, එය CTM මොඩියුලයේ සමුච්ඡයට ළඟා වීමට ඇති මාර්ගය මත ආයාත කිරීම රඳා පවතී.

මූඩ්ල් සේවාදායකය වින්ඩෝස් මෙහෙයුම් පද්ධතියේ ක්‍රියාත්මක නම්, පහසුවෙන් ඇක්සෙස් දත්ත සමුච්ඡයෙන් ගොනු ආයාත කළ හැකි ය.

මෙහෙයුම් පද්ධතිය ලිනක්ස් (Linux) නම් CTM බැංකුව පවතින්නේ වින්ඩෝස් මෙහෙයුම් පද්ධතියක නම් මේ සේවාදායකයෙන් දෙදෙනා ම එකම ජාලයක ජාලගත කරන්න. පසුව ODBC Socket Server මෘදුකාංගය හරහා ඒ දෙක අතර දත්ත හුවමාරුව සඳහා පහසුකම් සැලසිය හැකි ය. එහිදී XML භාවිතයෙන් දත්ත, මූඩ්ල් ලිනක්ස් මෙහෙයුම් පද්ධතියට මාරු කළ හැකි ය.
