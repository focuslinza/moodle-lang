ආරක්ෂිත කවුළුවක් (Window) විවෘත කිරීම

ආරක්ෂිත කවුළුව, යම් ප්‍රමානයකට ප්‍රශ්නාවලියට ආරක්ෂාවක් ලබාදේ. (කොපි කිරීම සහ වංචා සීමා කර තිබේ.)‌ ‍මෙහි දී ශිෂ්‍යයින්ට වෙබ් බ්‍රව්සරයේ සමහර ක්‍රියාකාරකම් අත්හිටුවා ඇත.

‍මෙහි දී,

  • බ්‍රව්සරයේ Java Script ක්‍රියා කාරීවීම අවශ්‍යතාවයකි.
  • ප්‍රශ්නාවලිය සම්පුර්ණ කවුළුවක විවෘත වේ.
  • පෙළ මත, මූසිකයේ ක්‍රියාකාරකම් සීමා කර තිබේ.
  • යතුරු පුවරුවේ,සමහර ක්‍රියාකාරකම් සීමා කර තිබේ.

සටහන: මේ ආරක්ෂාව,100% සාර්ථක නැත. මේ මත ම විස්වාස නොතබන්න. වෙබ් පරිසරයේ දී අරක්ෂාව සැපයීම තරමක් අපහසු වේ.මේ නිසා ඔබ ආරක්ෂාව ගැන දැඩි අවධානයෙන් සිටී නම් මේ මතම විස්වාසය නොතබන්න.ඔබට වඩාත් විසාල ප්‍රශ්න දත්ත සමුදායක් යොදා ගනිමින් ප්‍රශ්නාවලිය ඉදිරිපත් කිරීම තවත් උපායකි. ඵසේ ම වඩාත් නිර්මණශීලී ලෙස ප්‍රශ්නාවලියට අවධාණය අ‍ඩුකර වෙනත් ක්‍රියාකාරකම් වලට ( උදාහරණ:‍ෆොරම් සාකච්ඡා (forum discussions), පද මාලා තැනීම (glossary building), විකී ලිවීම(wiki writing), වැඩමුළු (workshops),පැවැරුම් (assignments) )
