ආරක්ෂිත කවුළුවක්(Window) විවෘත කිරීම

ආරක්ෂිත කවුළුව, යම් ප්‍රමානයකට ප්‍රශ්ණාවළියට ආරක්ෂාවක් ලබාදේ(කොපි කිරීම සහ වංචා සීමා කරතිබේ)‌ ‍මෙහි දී ශිෂ්‍යන්ට වෙබ් බ්‍රව්සරයේ සමහර ක්‍රියා තාරකම් අත්හිටුවා ඇත.

‍මෙහි දී,

  • බ්‍රව්සරයේ Java Script ක්‍රියා කාරීවීම අව්‍ශ්‍යය තාවයකි
  • ප්‍රශ්ණාවළිය සම්පුර්ණ කවුළුවක විවෘත වේ.
  • පෙළ මත, මූසිකයේ ක්‍රියාකාරී කම් සීමා කර තිබේ.
  • යතුරු පුවරුවේ,සමහර ක්‍රියා කාරීකම් සීමාකර තිබේ.

සටහන: මේ ආරක්ෂාව,100% සාර්තක නැත. මේ මත ම විස්වාස නොතබන්න. වෙබ පරිසරයේ දී අරක්ෂාව අරක්ෂාව සැපයීම තරමක් අපහසු වේ.මේ නිසා ඹබ ආරක්ෂාව ගැන දැඩි අවදානයෙන් සිටී නම් මේ මතම විස්වාසය නොතබන්න.ඹබට වඩාත් විසාල ප්‍රශ්ණ දත්ත සමුච්දයක් යොදා ගනිමින් ප්‍රශ්ණාවළිය ඉදිරිපත් කිරීම තවත් උපායකි. ඵසේ ම වඩාත් නිර්මණ ශීලී ලෙස ප්‍රශ්ණාවළියට අවධාණය අ‍ඩුකර වෙනත් ක්‍රියාකාරීකම් වලට ( උදා:‍ෆොරම් සාකඡචා (forum discussions), glossary building(පද මාලා තැනීම), විකී ලිවීම(wiki writing), වැඩමුළු (workshops),පැවැරැම් (assignments) )
