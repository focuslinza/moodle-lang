html ලිවීම සඳහා උදව් කිරීම

මූඩ්ල් පද්ධතියේ HTML භාවිත කරන විට‍, ඔබට ඕනෑම HTML ටැගයක් භාවිත කළ හැක.

එහිදී Javascript හෝ VB Script අන්තර්ග කිරීමට අවසර නැත. ඒවා ස්වයං ව ඉවත් කෙරේ.

කේත, සාමාන්‍යයෙන් පිටුවේ වූ වගු මත මුද්‍රණය වේ.

ඔබට <HEAD>හෝ <BODY> ටැග භාවිත කිරීමට අවශ්‍ය නැත.

නොගැලපුණ </TABLE> ටැග් ඉවත් කිරීමට වග බලා ගන්න.
