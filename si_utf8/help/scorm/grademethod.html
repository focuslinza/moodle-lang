පංති ඇගයිම් ක්‍රම

ලකුණු දැක්වෙන පිටුවල, SCORM/AICC පැකේඡයේ ක්‍රියාකාරකම් වල ප්‍රතිඵලයන්, ක්‍රම කීපයකට ඇගයිය හැකිය:

Learning Objects

‍මෙම ප්‍රවර්ගය, සම්පුර්ණකල/සමත්වු ක්‍රියාකාරකමේ වල යෙදුන Learning Objects වල සංඛාව පෙන්වයි. වැඩිම අගය, පැකේඡයේ Learning Objects ඇති සංඛාව වේ.

  • වැඩිම පංතිය

    සියලුම සමත්වු Learning Objects වලන් වැඩිම ලබාගත් ලකුණු සංඛාව වේ.

  • සාමාන්‍ය පංතිය

    සම්පුර්ණකල/සමත්වු ක්‍රියාකාරකමේ වල යෙදුන Learning Objects වලන් ලබාගත් ලකුණුවල සාමාන්‍ය අඟය වේ.

  • පංතිවල ඵකතුව

    සියලු Learning Objects වලන් ලබාගත් ලකුණුවල මුලු ඵකතුව වේ
