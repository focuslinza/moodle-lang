පැකේඡයේ ගොනු

පැකේඡතුල zip/pi‍ දිගුව සහිත ගොනු ‍වලින් සමන්විත වන අතර ඵ්වායේ පිළගත හැකි AICC හෝ SCORM පාඨමාලා නිර්වචන ගොනු අංතර්ගතවේ.

SCORM පැකේඡයේ ඇති imsmanifest.xml ගොනුව SCORM පාඨමාලාවේ සැකිල්ල , සම්පත් ඇතිස්ථාන ස‍හ තවන් දෑ ගොනු වී ඇත.

AICC පැකේඡය සැදීතිබෙන්නේ ගොනු කීපය(5-6) සමඟයි . මේ දැක්වෙන්නේ ගොනුවල දිගුවන්ය

  • CRS - Course Description file (අත්‍යවශයයි)
  • AU - Assignable Unit file (අත්‍යවශයය)
  • DES - Descriptor file (අත්‍යවශයය)
  • CST - Course Structure file (අත්‍යවශයය)
  • ORE - Objective Relationship file (විකල්පයක්)
  • PRE - Prerequisites file (විකල්පයක)
  • CMP - Completition Requirements file (විකල්පයක්)
