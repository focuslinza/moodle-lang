පැකේජයේ ගොනු

මෙම පැකේජය, zip/pi‍ දිගුව සහිත ගොනුවලින් සමන්විත වන අතර ඒවායේ පිළි ගත හැකි AICC හෝ SCORM යනුවෙන් වූ පාඨමාලා ගොනු අන්තර්ගත වේ.

SCORM පැකේජයේ ඇති imsmanifest.xml ගොනුව SCORM පාඨමාලාවේ සැකිල්ල , සම්පත් ඇතිස්ථාන ස‍හ තවන් දෑ ගොනු වී ඇත.

AICC පැකේජය සැදී තිබෙන්නේ පහත දක්වා ඇති ගොනු කීපය (5-6) සමඟයි . මේ දැක්වෙන්නේ ඒ ඒ ගොනුවල දිගුවන් ය.

  • CRS - Course Description file (අත්‍යවශයයි)
  • AU - Assignable Unit file (අත්‍යවශයයි)
  • DES - Descriptor file (අත්‍යවශයයි)
  • CST - Course Structure file (අත්‍යවශයයි)
  • ORE - Objective Relationship file (විකල්පයක්)
  • PRE - Prerequisites file (විකල්පයක)
  • CMP - Completition Requirements file (විකල්පයක්)
