从CSV文件导入

CSV 是 Comma-Separated-Values的缩写，它是文本交换通用的格式 。

The expected file format is a plain text file with a list of field names as the first record. The data then follows, one record per line.

字段定界符默认是一个半角逗号,单引号不是默认的。

Records should be delimited by new lines (usually generated by pressing RETURN or ENTER in your text editor). Tabs can be specified using \t and newlines by \n. 每条记录以新的一行作为定界（通常按回车或者RETURN键就会自动生成）。制表符可以利用\t来表示，新的一行可以通过\n表示。

例如:

  name,height,weight
  Kai,180cm,80kg
  Kim,170cm,60kg
  Koo,190cm,20kg

注意: 并不是支持左右的字段。
