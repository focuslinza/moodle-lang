可访问性

可访问声明

这个站点已经在一系列浏览器和屏幕阅读软件中测试过。

Moodle uses semantic markup to assist screen reader users among others — site, page and block headings use <h1>, <h2>, navigation blocks are lists <ul> and so on.

Moodle遵守万维网协议的第一层，第二层的大部分，和第三层的一部分。[Web Content Accessibility Guidelines 1.0] [about Double-A conformance] 通过这些帮助我们打算依照本地的法律去做。我们当前的工作是为了提高Moodle的可访问性和使用性。目前仍然有一些表单是用来设计的，但是总体来说（使用风格表单）是令人不满意的。

我们欢迎反馈，特别是关于任何不满的报表。请[message the administrator].

进入密码

目前我们不能制定密码，但是我们打算在发布Moodle的时候实现这个功能。

链接

每一个站点的版块都有一个链接，以方便读者在版块之间跳转。注意，链接是隐藏在浏览器里的。

许多链接是有描述链接细节的标题和属性的，除非链接的文字已经是描述的内容（如，文章的题目）。如果一个链接有访问密码，将会在链接的标题中提示。

在任何时候，都有可能链接意义与上下文无关。没有javascript：假链接

图像

所有的图像使用高质量的在于提供了一个合适的文本转换；纯粹装饰的图像则没有高质量的属性。图像，像地图，展示一个复杂的信息拥有一个longdesc属性链接着一个文本的描述。
