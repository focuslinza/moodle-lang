成绩类别汇总

您可以通过这个菜单选择一种成绩汇总方式，此方式将用于计算本类别中每个参与者的总体成绩。下面是对各不同选项的解释。

首先，将成绩修改成百分制（从0~1之间，称之为规范化），然后运用以下功能中的一个进行汇总，最后将其转换为要求的范围相关的类项目.(最低等级 和 最高等级）。

重要提示: 一个空成绩可以简单地看做一个看不见的成绩条目册。意味着不同的事物。例如：它可以是指还没有提交作业的学生，或是教师还没有评分的已提交的作业，或是已被成绩册管理者手动删除的成绩。因而建议慎重解释理解这些“空成绩”。

成绩的平均数
所有成绩的总和除以成绩总个数。比如：
A1 70/100, A2 20/80, A3 10/10, 类别最多为 100:
(0.7 + 0.25 + 1.0)/3 = 0.65 --> 65/100
加权平均数
每个成绩项都赋予一个权重，并运用在算术平均数的汇总中。这将影响到总平均数中每一项的重要性。
A1 70/100 weight 10, A2 20/80 weight 5, A3 10/10 weight 3, category max 100:
(0.7*10 + 0.25*5 + 1.0*3)/18 = 0.625 --> 62.5/100
简单的加权平均数
与加权平均数不同的是 这个权重可以作为每一项的 最大成绩 - 最小成绩 来计算。 100分的作业权重为100，10分的作业权重为10，比如
A1 70/100, A2 20/80, A3 10/10, 类别最多为100:
(0.7*100 + 0.25*80 + 1.0*10)/190 = 0.526 --> 52.6/100
成绩平均数（包括课外学分）
算术平均数有一个不够完善的地方，只能兼容以前（低功能）的活动，而不支持这里提供的汇总方式。
成绩的中间值
中间值是由呈现的所有成绩以及处于成绩中间部分的分数（或者说两个中间分数的平均数，如果分数是偶数个）计算得来。相对平均数而言，这样做的好处就是不受局外人的影响（成绩与平均值很不相同）
A1 70/100, A2 20/80, A3 10/10, 类别最多为 100:
0.7 + 0.25 + 1.0 --> 0.25 --> 25/100
最低成绩
结果是规范化后的最低值。它通常只与汇总非空成绩结合起来使用。比如：
A1 70/100, A2 20/80, A3 10/10, 类别最多为 100:
最小(0.7 + 0.25 + 1.0) = 0.25 --> 25/100
最高成绩
结果是规范化后的最高值。
A1 70/100, A2 20/80, A3 10/10, 类别最多为100:
最大(0.7 + 0.25 + 1.0) = 1.0 --> 100/100
成绩模式
成绩模式是指出现最频繁的成绩。一般是指非数字类的成绩。相对平均数而言，这样做的好处是不受局外人影响（成绩与平均数有很大不同）。然而，一旦有多于一个最频繁出现的成绩的时候，或者所有成绩都不同于其他成绩的时候，成绩模式就失去它的意义了（只能允许存在一个）。
A1 70/100, A2 35/50, A3 20/80, A4 10/10, A5 7/10 类别最多为100:
模式(0.7; 0.7; 0.25; 1.0; 0.7) = 0.7 --> 70/100
成绩总和
所有成绩的总和。与成绩的比例无关。无需转换成百分制的类型（规范化）。相关联类别项的最大成绩将作为所有汇总项中的最大值的总和自动计算。
A1 70/100, A2 20/80, A3 10/10:
70 + 20 + 10 = 100/190
