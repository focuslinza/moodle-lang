类别成绩汇总

您可以通过这个菜单选择一种成绩汇总方式，此方式将用于计算该类别中每个参与者的总成绩。下面是对各不同选项的解释。

分数先被转换为百分比（从0~1之间，称之为标准化），然后运用下列功能之一进行汇总，最后将其转换到对应类别项的分数范围内(在最低分和最高分之间）。

重要提醒: 简单说，空成绩就是成绩单中漏掉的成绩项，它可能意味着不同的情况。例如：可能是学生还没有提交作业，或是教师还没有对已提交的作业评分，或是分数被成绩单管理员手动删除了。因而，建议慎重对待这些“空成绩”。

平均分
所有分数的总和除以分数个数。
A1 70/100, A2 20/80, A3 10/10, 类别最高分100:
(0.7 + 0.25 + 1.0)/3 = 0.65 --> 65/100
加权平均分
每个成绩项都被赋予一个权重，并应用在算术平均分的汇总中。这将影响到总平均分中每一项的重要性。
A1 70/100 权重 10, A2 20/80 权重 5, A3 10/10 权重 3, 类别最高分100:
(0.7*10 + 0.25*5 + 1.0*3)/18 = 0.625 --> 62.5/100
简单加权平均分
与加权平均分不同的是，每项的权重通过最高分 - 最低分来计算。 100分的作业权重为100，10分的作业权重为10。
A1 70/100, A2 20/80, A3 10/10, 类别最高分100:
(0.7*100 + 0.25*80 + 1.0*10)/190 = 0.526 --> 52.6/100
平均分（包括额外加分）
调整过的算术平均分。这是一个旧的，已经不再支持的汇总方式。这里提供它只是为了向后兼容旧活动。
中位数
把所有分数按大小排序后，处于中间位置的数值（或两个中间分数的平均分）。相对平均分而言，这样做的好处就是不会受到异常值（与平均分相差很大的分数）的影响。
A1 70/100, A2 20/80, A3 10/10, 类别最高分100:
median(0.7 ; 0.25 ; 1.0) = 0.7 --> 70/100
最低分
取标准化后的最低分。它通常与只汇总非空成绩结合起来使用。
A1 70/100, A2 20/80, A3 10/10, 类别最高分100:
min(0.7 + 0.25 + 1.0) = 0.25 --> 25/100
最高分
取标准化后的最高值。
A1 70/100, A2 20/80, A3 10/10, 类别最高分100:
max(0.7 + 0.25 + 1.0) = 1.0 --> 100/100
众数
众数是出现最频繁的分数。它经常被用于非数字式的成绩。相对平均分而言，这样做的好处就是不会受到异常值（与平均分相差很大的分数）的影响。然而，一旦有多个最频繁出现的成绩时（只保留一个），或者所有成绩都与其它成绩不同时，众数就失去了意义。
A1 70/100, A2 35/50, A3 20/80, A4 10/10, A5 7/10 类别最高分100:
mode(0.7; 0.7; 0.25; 1.0; 0.7) = 0.7 --> 70/100
总分
所有分数的总和。等级类型的成绩被忽略。这是仅有的一种无需在内部把分数转换成百分比（标准化）的方式。对应类别的最高分自动计算自所有汇总项的最高分的总和。
A1 70/100, A2 20/80, A3 10/10:
70 + 20 + 10 = 100/190
