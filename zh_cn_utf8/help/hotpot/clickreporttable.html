点击跟踪

这个表呈现了每个学生在做任何一个Hot Potatos测验时选择时选项和完成情况。

注意这个报表涉及得非常广泛，并且很长。因此可以下载到电脑上用电子表格软件查看（如Microsoft Excel）

表中的每行包括一下信息：

点击了那一个?
下面的细节描述了哪个学生在哪个测验中尝试点击了哪个选项等：

  • 课程名称, 第几节, 习题数, 习题名称,习题类型, 问题总数
  • 学生全名, 登录日期, 登录时间, 登离时间
  • 尝试次数, 尝试开始时间, 尝试结束时间
  • 点击数,点击时间, 点击类型（类别）

哪些问题受到点击（影响）
下面细节描述了哪些问题受到了点击（影响）：

  • 被选择过的问题或者给出当前点击的线索
  • 当前点击后每个问题的状态
    • 0 表示 "正确"
    • X 表示 "错误"
    • - 表示 "未尝试"
  • 当前点击后接受到的新答案
  • 当前点击后问题中接受到哪个新答案
  • 当前点击后问题中接受到哪个新答案, 或者给出哪些线索

多少问题受到影响？
下面细节中呈现出总共有多少道题目受到了点击的影响和目前测验中题目的总数：

  • 尝试数the number of questions attempted
  • 未尝试数the number of questions not attempted
  • 正确的题目数the number of questions whose status is "right"
  • 错误的题目数the number of questions whose status is "wrong"
  • 有提示的题目数the number of questions for which hints have been given
  • 有线索的题目数the number of questions for which clues have been given
  • 已接受新答案的题目数the number of questions for which new answers have been received
  • 新答案已接受或者提示、线索接受的题目数

分数
以下描述的是到目前位置测验中的得分情况：

  • 丢分
  • 最高分
  • 分数百分比
