Prolongar a inscrição

Esta página permite-lhe prolongar o período de inscrição dos participantes. O novo período de inscrição será calculado com base numa data inicial de referência e um período de extensão a partir dessa data. De acordo com a combinação de dados que usar, poderá efectivamente restringir o período de inscrição dos participantes.

As datas iniciais de referência são:

  • Data de início da disciplina
  • Início da inscrição - a data de início da inscrição do participante (esta opção estará disponível apenas para utilizadores com data de inscrição conhecida)
  • Fim da inscrição - a data em que termina a inscrição do participante (esta opção estará disponível apenas para utilizadores inscritos por um período limitado)
  • Início do período de inscrições - a data de início das inscrições definida para a disciplina (esta opção estará disponível apenas para disciplinas com data de início de inscrições definida)
  • Fim do período de inscrições - a data do fim do período de inscrições definida para a disciplina (esta opção estará disponível apenas para disciplinas com data de fim de inscrições definida)

Se o período de extensão for definido como 'Inalterado' ou 'Sem título', a data de referência será ignorada.
