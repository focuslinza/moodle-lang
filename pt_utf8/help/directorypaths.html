Caminhos de Directórios

A sintaxe de um caminho de directório completo depende do seu sistema operativo:

Num Sistema Windows use algo como:

  • c:\program files\data\moodle
  • c:\documents and settings\moodle\temp\saved

Em Sistemas Unix use algo como:

  • /usr/data/moodle
  • /home/moodle/tmp/saved

Note que é normalmente uma boa ideia conferir que o directório já exista. Algumas vezes o Moodle consegue criar o directório quando necessário e pedido mas nem sempre isso é possível.

Finalmente, assegure-se que o texto é o certo (Maiúsculas Vs Minúsculas) e nunca inclua nenhuma barra ("\" ou "/") no fim.
