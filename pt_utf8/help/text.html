Instruções para escrever texto

O texto escreve-se em Moodle na forma normal como se faz num editor de texto, excepto que o texto pode incluir carinhas (símbolos para representar gestos), "endereços da Web" e algumas etiquetas HTML.

Carinhas

Para incluir um destes símbolos no seu texto, escreva o código correspondente. Os códigos parecem como pequenas representações do símbolo, se olhar para eles com a sua cabeça virada para a esquerda.

sorriso :-)
grande sorriso :-D
a piscar o olho ;-)
misturado :-/
pensativo V-.
língua de fora :-P
calmo B-)
tolerante ^-)
olhos abertos 8-)
surpreendido 8-o
triste :-(

mau }-]

URLs

Qualquer palavra que começar por www. ou http:// será transformada automaticamente num vínculo para a respectiva página web.

Por exemplo: www.moodle.com ou http://curtin.edu

Etiquetas HTML

Pode ainda usar algumas etiquetas de HTML para dar ênfase ao seu texto.

etiqueta HTMLResultado
<b>bold </b>texto em negrito
<i>itálica </i>texto em itálico
<u>sublinhada </u> texto sublinhado
<font size="1">pequeno</font>letra pequena
<font size="4">grande </font>letra grande
<font color="green">cores</font>texto a cor
<ul><li>um</li><li>dois</li></ul>
  • um
  • dois
