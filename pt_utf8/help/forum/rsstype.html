Listagem RSS para um fórum

Esta opção permite-lhe a possibilidade de criar uma listagem RSS para um fórum.

Pode seleccionar entre dois tipos de listagem:

  • Temas: Utilizando este tipo, as listagens geradas irão os últimos temas iniciados no fórum junto das suas mensagens iniciais.
  • Mensagens: Utilizando este tipo, as listagens gerados irão incluir as últimas mensagens enviadas para o fórum.
