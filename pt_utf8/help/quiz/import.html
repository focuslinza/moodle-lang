Importação de novas perguntas

Esta função permite-lhe importar perguntas de ficheiros de texto externos, enviados através de um formulário, ou seleccionando um ficheiro apropriado já existente na área de ficheiros da disciplina.

Nota sobre codificação de caracteres

Se o seu ficheiro contem caracteres não ascii, deverá usar a codificação UTF-8. O seu editor de texto favorito deverá permitir converter para UTF-8, ou poderá existir como opção no software de exportação. Os formatos de importação baseados em XML deverão fazer a conversão automaticamente, sempre e quando o servidor Moodle estiver a usar PHP 5.

Aceitam-se vários formatos de ficheiros:

Formato GIFT

    GIFT é o formato de importação mais completo disponível para importar perguntas de mini-teste de um ficheiro em texto para o Moodle. Foi concebido para ser um método fácil para os professores escreverem perguntas como um ficheiro de texto. Aceita Escolha múltipla, Verdadeiro/Falso, Resposta curta, Correspondência correcta, Perguntas numéricas assim como a inserção de um campo ________ para o formato de palavra em falta. frase. Vários tipos de perguntas podem ser misturadas num único ficheiro de texto e este formato também aceita comentários, nomes de perguntas, "feedback" e pesos em percentagem para as respostas. Seguem-se alguns exemplos:

    Quem está enterrado no túmulo de Grant? {~Grant ~Jefferson ~ninguém}

Grant está  {~enterrado ~sepultado ~vivo} no túmulo de Grant.

Grant está enterrado no túmulo de Grant. {FALSE}

Quem está enterrado no túmulo de Grant? {=nenhuma pessoa =ninguém}

Quando nasceu Ulysses S. Grant? {#1822}

Formato Aiken

    O formato Aiken é uma forma simples de formular perguntas de escolha múltipla utilizando um formato muito claramente legível. Exemplo:

    Qual a finalidade dos primeiros socorros?
A. Salvar a vida, evitar danos posteriores, preservar a saúde
B. Providenciar tratamento médico a uma pessoa lesada ou ferida
C. Evitar danos posteriores
D. Ajudar vítimas que procuram ajuda
RESPOSTA: A

Palavra em falta

    Este formato apenas admit perguntas de escolha múltipla e perguntas de resposta curta, dependendo do número de respostas. Uma resposta produz uma pergunta de resposta curta, e duas ou mais respostas produzem uma pergunta de resposta múltipla. Cada resposta está separada por um til (~) e a resposta correcta é precedida por um sinal de igual " = ". Exemplo de pergunta de escolha múltipla:

    Mal começamos a explorar as partes do nosso corpo em criança tornamo-nos alunos de {=anatomia e fisiologia ~reflexologia ~ciência ~experiência}, e de certa forma ficamos alunos para toda a vida.

    Mais informação sobre o formato "Falta de palavra"

Aqui segue um exemplo de pergunta de resposta curta:

Mal começamos a explorar as partes do nosso corpo em criança tornamo-nos alunos de {=anatomia e fisiologia}, e de certa forma ficamos alunos para toda a vida.

AON

    Este formato é o mesmo de "Palavra em falta", excepto o facto de após a importação das perguntas, todas as de Resposta Curta se transformam, quatro de uma só vez, em Perguntas de Correspondência Correcta.

    Além disso, as respostas às perguntas de escolha múltipla são aleatóriamente misturadas na importação das mesmas.

    Este nome provem da organização que patrocinou o desenvolvimento de muitas opções dos mini-testes.

Blackboard

    Este módulo tem a capacidade de importar perguntas gravadas no formato de exportação do "Blackboard". Depende de queas funções do XML tenham sido compiladas no seu PHP. Repare que esta versão não funciona com as versões 6 ou anterior do Blackboard.

    Mais informação sobre o formato "Blackboard"

Blackboard V6+

    Este módulo pode importar perguntas gravadas no formato de exportação do Blackboard. Proporciona suporte limitado para as versões 6 e 7. Depende de que as funções do XML tenham sido compiladas no seu PHP. Terá que enviar para o Moodle o ficheiro zip exportado do Blackboard.

WebCT

    Este módulo tem a capacidade de importar perguntas gravadas no formato de texto do WebCT.

    Mais informação sobre o formato "WebCT"

Course Test Manager

    Este módulo tem a capacidade de importar perguntas gravadas num banco de testes do Test Course Manager. Apoia-se em diferentes formas de acesso ao banco de testes, que é uma base de dados Acess da Microsoft, dependendo se o Moodle está a funcionar num servidor Web Windows ou Linux.

    O Windows permite-lhe enviar a base de dados Access tal como qualquer outro ficheiro de importação de dados.

    No Linux, tem de configurar um sistema operativo Windows na mesma rede onde estiver a base de dados do Test course Manager, e um software chamado "Servidor ODBC Socket", que usa XML para transferir dados para o Moodle no servidor Linux.

    Leia, por favor, todo o seguinte ficheiro de ajuda, antes de utilizar este tipo de importação.

    Mais informação sobre o formato "CTM"

Respostas embebidas (Cloze)

    Este formato tem a finalidade específica de apenas importar um tipo de pergunta, o formato das Respostas Embebidas (também conhecidas por "Cloze")

    Mais informação sobre o formato "Cloze"

Formato Learnwise

    Este formato permite importar perguntas de escolha múltipla no formato XML do Leranwise.

Examview format

    Este formato permite importar perguntas dos ficheiros XML do Examview 4 XML files. De salientar que não aceita versões anteriores do Examview.

Formato XML do Moodle

    Este formato esopecífico do Moodle permite importar mini-testes que tenham sido exportados previamente no mesmo formato. O formato é bastante claro e a melhor forma de mostrá-lo é exportando uma categoria no formato XML do Moodle. O formato XML permite importar ficheiros de imagens..
