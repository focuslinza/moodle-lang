Teclas de Atalho do Editor de HTML

As seguintes opções estão acessíveis através de teclas de atalho:

Ctrl-O Alterar o tipo de letra
Ctrl-P Alterar o tamanho da letra
Ctrl-H Alterar o formato do texto (cabeçalhos, etc.)
Ctrl-1 até Ctrl-6 Alterar o nível do cabeçalho
Ctrl-= Altera a linguagem do texto para leitores do ecrã, ou especifica línguas para o filtro Multi línguas (estes aparecem depois da opção 'Multi' no menu).
Ctrl-B Alterar texto para negrito
Ctrl-I Alterar texto para itálico
Ctrl-U Sublinhar texto
Ctrl-S Riscar texto
Ctrl-, Subíndice
Ctrl-. Superíndice
Ctrl-0 Limpar HTML criado pelo Word
Ctrl-Z Desfazer
Ctrl-Y Refazer
Ctrl-L Alinhar à esquerda
Ctrl-E Centrar
Ctrl-J Preencher as linhas
Ctrl-/ Texto corre da esquerda para a direita
Ctrl-| Texto corre da direita para a esquerda
Ctrl-; Diminuir identação
Ctrl-' Aumentar identação
Ctrl-G Alterar cor do texto
Ctrl-K Alterar cor de fundo
Ctrl-F Encontrar e substituir
Ctrl-` Mudar para vista da fonte HTML
Ctrl-M Mudar editor para janela separada
Ctrl-Alt-O

 Inserir lista ordenada
Ctrl-Alt-U Inserir lista não ordenada
Ctrl-Alt-R Inserir barra horizontal
Ctrl-Alt-A Criar rótulo
Ctrl-Alt-L Criar apontador
Ctrl-Alt-D Retirar apontador
Ctrl-Alt-N Não criar apontadores automaticamente
Ctrl-Alt-I Inserir imagem
Ctrl-Alt-T Inserir tabela
Ctrl-Alt-S Inserir risonhos
Ctrl-Alt-C Inserir caracteres especiais

Limitações actuais:

  • Actualmente o atalho Ctrl-` não permite regressar para o modo WYSIWYG após ter passado para o código fonte HTML.
  • Não existem atalhos para operações de tabela (só disponíveis em modo de edição de janela separada). De qualquer forma não são "amigáveis" com o teclado já que dependem do uso do rato para a seleccionar as células da tabela nas quais se quer operar.
