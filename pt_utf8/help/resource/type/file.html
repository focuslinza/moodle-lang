Ficheiros e páginas Web

Este tipo de recurso permite ligação a qualquer página web ou a qualquer outro ficheiro web. Permite tambem ligação a qualquer outra página web ou a outro ficheiro que enviou para a área de ficheiros da sua disciplina a partir do seu computador.

As usuais páginas web são vizualizadas tal como são, enquanto que os ficheiros multimédia são processados mais hábilmente e podem ser integrados numa página web. Por exemplo, os ficheiros MP3 aparecem utilizando um programa corrente, bem como filmes, animações em flash, etc.

Existem várias opções em janelas quer normais quer enquadradas, etc.

Particularmente, se o seu recurso é uma aplicação web ou qualquer outro tipo de conteúdo que permita aceitar parâmetros, pode escolher enviar informação para o seu recurso tal como o nome do utilizador,o seu e-mail, a disciplina em que estão integrados, etc.
