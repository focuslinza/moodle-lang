Limite de respostas

Esta opção permite limitar o número de participantes que podem seleccionar cada uma das opções particulares.

Uma vez habilitados os limites, poderá ser estipulado um limite para cada opção. Quando esse limite for alcançado, mas ninguém podera seleccionar essa opção.

Se os limites forem deshabilitados, qualquer número de participantes poderão seleccionar cada uma das opções.
