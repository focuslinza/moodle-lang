Mudança de método de autenticação

Este menu permite alterar o método do autenticação para este utilizador em particular.

Mas tenha em conta que as possibilidades aqui dependerão dos métodos de autenticação que tiverem sido definidos para o sítio.

Um erro aqui pode fazer com que o utilizador não consiga entrar mais no sítio ou até que a sua conta seja apagada. Assim, só use este menu se souber perfeitamente o que está a fazer.
