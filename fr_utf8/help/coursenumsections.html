Nombre de sections

Ce paramètre n'est utilisé que pour les cours hebdomadaires ou thématiques.

Dans le format hebdomadaire, il indique en semaines la durée du cours, à partir de sa date de début.

Pour un cours thématique, il détermine le nombre de sections qui le composent.

Dans les deux cas, ce nombre conditionne la quantité de « boîtes » (sections) affichées dans la colonne centrale de la page d'accueil du cours.
