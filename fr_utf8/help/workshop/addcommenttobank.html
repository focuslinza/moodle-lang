Ajouter un commentaire à une banque de commentaires types

Des commentaires souvent formulés peuvent être ajoutés dans un formulaire d'évaluation de telle sorte qu'ils ne doivent pas être saisis à nouveau s'ils doivent être notifiés dans une évaluation ultérieure. Ce « stock » de commentaires apparaît sous la boîte de commentaire d'évaluation. Cliquer sur le texte de commentaire l'ajoute au champ de saisie de commentaire. Les étapes pour mémoriser un commentaire à un formulaire d'évaluation sont :

  1. tapez le commentaire dans la boîte de saisie de commentaire d'évaluation ;
  2. cliquez sur le bouton « Ajouter un commentaire ».

Le formulaire d'évaluation est rechargé. Le nouveau commentaire est présent à la fois dans le champ de saisie et sous forme de lien sous ce champ. Vous pourrez directement disposer de ce texte pour des évaluations ultérieures en cliquant sur ce lien.

Les commentaires mémorisés sont toujours ajoutés à la fin du texte présent dans le champ de saisie, séparés de ce dernier par un espace.
