Ajouter un commentaire

Les différents intervenants peuvent ajouter des commentaires :

  1. L'étudiant qui évalue le travail (le correcteur) peut ajouter un commentaire pour compléter ou clarifier son évaluation.
  2. L'étudiant dont le travail est évalué peut ajouter un commentaire pour remettre en cause les affirmations qui ont été faites dans l'évaluation.
  3. L'enseignant peut ajouter un commentaire pour remplir son rôle de médiateur entre les deux étudiants : celui qui se fait évaluer et celui qui est évalué.

Le but des commentaires est d'obtenir un consensus sur l'évaluation, le tout dans un climat qui favorise un meilleur apprentissage.

Si le travail est réévalué, les commentaires de l'évaluation précédente sont effacés et l'évaluation est recommencée à zéro.
