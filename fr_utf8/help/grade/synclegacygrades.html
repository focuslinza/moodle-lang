Synchroniser les notes d'anciennes versions

Le terme « notes d'anciennes versions » fait référence aux notes attribuées dans des versions de Moodle antérieures à la version 1.9. Lors de la mise à jour vers la version 1.9, toutes les notes d'anciennes versions doivent être incluses dans le carnet de note de Moodle 1.9. Le bouton « Synchroniser les notes d'anciennes versions » vérifie si c'est bien le cas, et si nécessaire ajoute toutes les notes d'anciennes versions au carnet de notes de Moodle 1.9.
