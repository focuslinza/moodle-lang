Tendance centrale de la catégorie

Ce menu vous permet de choisir quelle stratégie sera utilisée pour le calcul d'une tendance centrale pour la note globale des participants, dans cette catégorie. Les différentes options sont expliquées ci-dessous.

Important : une « note vide » est considérée simplement comme une entrée manquante du carnet de note. Elle pourrait cependant signifier différentes situations. Par exemple, ce pourrait être une participant qui n'a pas encore rendu un devoir, un devoir rendu, mais pas encore évalué par l'enseignant, ou une note ayant été supprimée manuellement par le gestionnaire du carnet de notes. Il convient d'être prudent lors de l'interprétation de ces « notes vides.»

Stratégie Description
Moyenne de toutes les notes Toutes les notes sont additionnées, puis la somme est divisée par le nombre total des notes. Les notes vides sont incluses dans le calcul (elles sont interprétées comme la valeur minimale de l'élément d'évaluation).
Moyenne des notes non vides Même chose que ci-dessus, sauf que les notes vides sont exclues du calcul.
Médiane de toutes les notes La médiane est calculée en comptant toutes les notes triées et sélectionnant la note du milieu (ou la moyenne des deux notes du milieu s'il y a un nombre pair de notes). L'avantage de la médiane sur la moyenne est qu'elle est beaucoup moins influencée par les valeurs extrêmes (notes anormalement éloignées de la moyenne). Les notes vides sont incluses dans le calcul.
Médiane des notes non vides Même chose que ci-dessus, sauf que les notes vides sont exclues du calcul.
La plus basse des notes non vides Seule la note la plus basse est conservée. Les notes vides sont exclues du calcul.
La plus haute des notes non vides Seule la note la plus haute est conservée. Les notes vides sont exclues du calcul.
Mode de toutes les notes Le mode est la note qui survient le plus fréquemment. Il est utilisé le plus souvent avec les notes non numériques. L'avantage du mode sur la moyenne est qu'il est beaucoup moins influencé par les valeurs extrêmes (notes anormalement éloignées de la moyenne). Les notes vides sont incluses dans le calcul.
Mode des notes non vides Même chose que ci-dessus, sauf que les notes vides sont exclues du calcul.
Moyenne pondérée de toutes les notes Chaque élément d'évaluation peut être associé à un coefficient qui sera utilisé lors du calcul de la moyenne arithmétique pour influencer l'importance de chacun des éléments dans la moyenne globale. Les notes vides sont incluses dans le calcul.
Moyenne pondérée des notes non vides Même chose que ci-dessus, sauf que les notes vides sont exclues du calcul.
Moyenne de toutes les notes (points supplémentaires) La moyenne arithmétique, avec un facteur de correction. Cette option n'est plus supportée et n'est fournie ici que pour être compatible avec d'anciennes activités. Les notes vides sont incluses dans le calcul.
Moyenne des notes non vides (points supplémentaires) Même chose que ci-dessus, sauf que les notes vides sont exclues du calcul.
