Barème de lettre

Vous pouvez définir un barème de lettre ici. Au départ un barème « proposé » est présent et la seule chose à faire pour utiliser ce barème est de cliquer sur « Enregistrer les modifications ». Cependant, si vous n'appréciez pas ce barème, il vous suffit de modifier les entrées que vous désirez et de cliquer sur « Enregistrer les modifications ». Après ce paramétrage initial, votre barème sera sélectionné. Laisser une entrée vide pour ne pas l'inclure ou supprimez-la du barème d'évaluation.

Erreurs que vous pouvez trouver dans le carnet de note :

  • Aucun barème défini : cela signifie que le barème d'évaluation initial n'a pas été défini. Tout ce que vous avez à faire (si le barème suggéré vous convient) est de cliquer sur « Enregistrer les modifications »
  • Pas de note lettre pour X : indique qu'il n'y pas de note lettre appropriée pour le niveau X.
