Echelle de lettre

Vous pouvez définir une échelle de lettre ici. Au départ une échelle "proposée" est présente et la seule chose à faire pour utiliser cette échelle et de cliquer sur "Enregistrer les modifications". Cependant si vous n'appréciez pas cette échelle, il vous suffit de modifier les entrées que vous désirez et de cliquer sur "Enregistrer les modifications". Après ce paramétrage initial vous verrez que votre échelle est sélectionnée. Laisser une entrèe vide pour ne pas l'inclure ou supprimez la de l'échelle de notation.

Erreurs que vous pouvez trouver dans le carnet de note :

  • Aucunes échelle définies : cela signifie que l'échelle de notation initial n'a pas été définie. Tout ce que vous avez à faire (si l'échelle suggérée vous convient) est de cliquer sur "Enregistrer les modifications"
  • Pas de lettre de notation pour X : indique qu'il n'y pas de lettre de notation appropriée pour le niveau X.
