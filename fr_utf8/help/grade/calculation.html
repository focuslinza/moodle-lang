Calculs

Les calculs du carnet de notes utilisent une syntaxe similaire à celle des formules ou fonctions des tableurs. Ils commencent par un signe d'égalité (=) et emploient des opérateurs et fonctions mathématiques usuel pour produire un résultat numérique. Ce résultat est ensuite utilisé comme valeur calculée pour l'élément d'évaluation que vous modifiez.

Les fonctions permises dans les calculs sont présentées ci-dessous. Le caractère point-virgule (;) est utilisé pour séparer les paramètres, à l'intérieur des parenthèses des fonctions. Attention : si vous utilisez d'autres langues dans Moodle, le séparateur sera adapté aux habitudes de la langue. Pour la langue anglaise par exemple, le séparateur est la virgule (,).

  • average([[param1]]; [[param2]]; ...) : calcule la moyenne des nombres fournis
  • max([[param1]]; [[param2]]; ...) : calcule la valeur maximale des nombres fournis
  • min([[param1]]; [[param2]]; ...) : calcule la valeur minimale des nombres fournis
  • mod(dividende; diviseur) : calcule le reste de la division
  • pi() : donne la valeur du nombre π
  • power(base; puissance) : élève un nombre à une puissance d'un autre
  • round(nombre; précision) : arrondit un nombre à la précision donnée
  • sum([[param1]]; [[param2]]; ...) : calcule la moyenne des nombres fournis

Il est possible d'inclure dans vos formules les valeurs d'autres éléments d'évaluations en utilisant leur identifiant comme référence. L'identifiant doit alors être entouré de deux paires de crochets [[...]], afin d'éviter que l'analyseur de formules ne soit trompé, si l'identifiant comporte des caractères spéciaux. Par exemple, si un élément d'évaluation a l'identifiant « Test.3 », vous vous y référerez par [[Test.3]] dans votre formule.

Au-dessous du champ de calcul est affichée une arborescence de vos cours avec leurs catégories de notes et leurs éléments d'évaluation. Près du total de chaque catégorie ou élément est affiché l'identifiant que vous pouvez utiliser dans votre formule (déjà entouré de crochets). Néanmoins, puisque l'identifiant est optionnel, certains éléments n'en auront pas. Ces éléments sans identifiant ont en lieu et place un champ vous permettant de saisir directement l'identifiant, de sorte que vous puissiez l'utiliser dans votre formule sans changer de page. Toutefois, ce formulaire ne vous permet pas de modifier les identifiants existants. Pour ce faire, vous devrez modifier directement l'élément d'évaluation en question.

Une fois que vous aurez spécifié les identifiants dont vous avez besoin, vous pourrez cliquer « Ajouter les identifiants » ; la page se rechargera et vous montrera la même arborescence avec les identifiants que vous aurez saisi. Vous pourrez alors utiliser ces références dans vos formules.

Voici quelques exemples de formules que vous pouvez utiliser dans votre carnet de notes :

  • =average([[Test.1]]; [[Test.4]]; [[Assignment.1]])
  • =average(max([[Test.1]]; [[Test.4]]; [[Devoir.1]]); min([[Test.1]]; [[Test.4]]; [[Devoir.1]]))
