Importer des objectifs

Il est possible d'importer des objectifs sous la forme de fichiers CSV, avec séparateur point-virgule (;). Le format de ces fichiers doit être le suivant :

Nom du champ Description Requis Format
outcome_name Le nom complet de l'objectif OuiString
outcome_shortnameLe nom abrégé de l'objectif Oui String
outcome_descriptionLa description de l'objectif Non String
scale_nameLe nom du barème utilisé Oui String
scale_itemsUne liste des éléments du barème, séparée par des virgules Oui String
scale_descriptionLa description du barème Non String

Voici un exemple :

outcome_name;outcome_shortname;outcome_description;scale_name;scale_items;scale_description
Participation;participation;;Barème de participation;"Peu ou pas de participation, Participation satisfaisante, Participation active";
