Types de devoir

De plus en plus de types de devoirs sont disponibles. En voici la liste actuelle.

Activité hors ligne

Un tel devoir est utile lorsque l'activité n'est effectuée dans Moodle. Ce peut être par exemple une activité effectuée face à face ou ailleurs sur le web.

Les participants peuvent voir une description de la tâche à effectuer, mais ne peuvent ni déposer un fichier, ni écrire quoi que ce soit. L'évaluation se passe normalement et les participants reçoivent notification de leur évaluation.

Texte en ligne

Ce type de devoir demande aux participants de taper un texte à l'aide des outils habituels d'édition de texte. Les enseignants peuvent ensuite les évaluer en ligne et même ajouter des commentaires ou des modifications dans le texte.

(Si les anciennes versions de Moodle vous sont familières, vous reconnaîtrez dans ce type de devoir l'ancienne activité Journal.)

Déposer un fichier

Ce type de devoir permet à chaque participant de déposer un (unique) fichier de n'importe quel type.

Par exemple, un fichier OpenOffice ou Word, une image, un dossier de documents compressé (ZIP), ou n'importe quoi d'autre.

Dépôt avancé de fichiers

Ce type de devoir permet à chaque participant de déposer un ou plusieurs fichiers de tous formats. Par exemple, un fichier OpenOffice ou Word, une image, un dossier de documents compressé (ZIP), ou n'importe quoi d'autre.

Il vous permet aussi de déposer plusieurs fichiers de solutions. Ces fichiers peuvent être déposés avant la remise du devoir par les participants, leur donnant ainsi à chacun un fichier différent sur lequel travailler.

Les participants peuvent aussi taper des remarques décrivant les fichiers déposés, le progrès de leur travail ou toute autre information textuelle.

La remise de ce type de devoir doit être finalisée manuellement par les participants. En attendant cette finalisation, les devoirs non terminés sont indiqués comme brouillons. Les participants peuvent remettre un fichier finalisé dans l'état de brouillon tant que l'évaluation n'est pas effectuée.
