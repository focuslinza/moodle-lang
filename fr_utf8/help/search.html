Comment effectuer des recherches ?

La recherche du texte intégral offre diverses options indiquées ci-dessous. Elles peuvent être combinées afin d'affiner vos recherches.

rechercher ces mots Pour une recherche simple d'un ou de plusieurs mots n'importe où dans le texte, il suffit de les taper, en les séparant par des espaces. Les mots de moins de 3 caractères sont ignorés.
rechercher +ces +mots L'exemple précédent aurait notamment permis de trouver le terme « marmots », qui contient les lettres « mots ». Pour faire en sorte que seuls des mots entiers soient trouvés, on utilise le symbole plus (+).
+moteur -recheche L'utilisation du signe moins (-) permet d'exclure des termes de la recherche.
"moteur de recherche" Pour rechercher une expression, on la placera entre guillemets droits (").
user:Nicolas Pour rechercher les textes d'un utilisateur particulier, on placera l'expression « user: » devant un des mots de son nom.
userid:6 Si vous connaissez l'identifiant (Moodle ID) d'un utilisateur, vous pouvez rechercher ses contributions en utilisant la syntaxe « userid: », suivie de l'identifiant.
subject:composition Pour rechercher un terme apparaissant dans le sujet ou le titre d'un texte, placer l'expression « subject: » devant le terme.

Pour des critères de recherche plus pointus, cliquez sur le bouton sans taper quoi que ce soit dans les différents champs. Un formulaire complet s'affichera, vous permettant de spécifier plus précisément votre recherche.
