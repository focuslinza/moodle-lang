Catégories de questions

Plutôt que de conserver toutes vos questions dans une longue liste, vous avez la possibilité de créer des catégories pour les classer.

Les catégories peuvent être créées ou supprimer selon votre gré. Cependant,

  • il doit y avoir au moins une catégorie dans chaque contexte. Vous ne pouvez donc pas supprimer la dernière catégorie d'un contexte ;
  • si vous essayez de supprimer une catégorie contenant des questions, il vous sera demandé d'indiquer une autre catégorie dans laquelle déplacer ces questions.

Vous pouvez classer vos catégories dans une hiérarchie afin de les gérer plus facilement. La modification des catégories s'effectue dans l'onglet « Catégories » de la banque de questions.

  • Sur la page principale de l'onglet « Catégories » de la banque de questions,
    • les flèches haut et bas modifient l'ordre dans lequel les catégories dans le même niveau sont affichées ;
    • il est possible de déplacer une catégorie vers un nouveau contexte avec les flèches haut et bas ;
    • Les flèches gauche et droite permettent de modifier la catégorie mère d'une catégorie.
  • Une manière plus rapide de déplacer des catégories est de cliquer sur l'icône d'édition de l'onglet « Catégories » de la banque de question et d'utiliser ensuite le menu déroulant pour choisir une catégorie mère.
