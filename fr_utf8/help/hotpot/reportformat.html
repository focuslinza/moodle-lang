Format du rapport

Vous pouvez choisir parmi différents formats de rapports.

Format du rapport

HTML
le rapport sera généré en format HTML, adapté à l'affichage dans un navigateur internet.

Excel
le rapport sera créé sous la forme d'un fichier au format « .xls », qui peut être ouvert avec un tableur tel que Microsoft Excel ou OpenOffice.Org Calc.

Texte
le rapport sera généra sous la forme d'un fichier texte qui peut être ouvert à l'aide d'un éditeur de texte.

Encodage

La liste des encodages possibles permet de forcer la sortie de vos données dans un encodage particulier. Par exemple, Microsoft Excel nécessite l'encodage « SJIS » pour l'écriture japonaise. La liste des encodages disponibles peut être paramétrée par l'administrateur du site, dans la page « paramètres » du module HotPot.

AJustement des données

Oui
la largeur des données peut être ajustée pour entrer dans les cellules du tableau. De ce fait, les lignes peuvent être très hautes.

Non
la largeur des données n'est pas ajustée.

Afficher la légende

Oui
dans le rapport principal, les questions et les réponses sont représentées par des lettres. Une légende sera créée, montrant la correspondance entre les lettres et les questions ou réponses.

Non
les questions et les réponses seront affichées intégralement, sans recourir à une légende.
