Format d'affichage

Vous pouvez choisir entre différents formats d'affichage de votre activité

Meilleur
L'activité sera affichée dans le meilleur format possible pour le navigateur de l'utilisateur.
v6+
L'activité sera affichée en format glisser-déposer pour les navigateur v6+
v6
L'activité sera affichée en format pour les navigateurs v6
v5
L'activité sera affichée en format pour les navigateurs v5
v4
L'activité sera affichée en format pour les navigateurs v4
v3
L'activité sera affichée en format pour les navigateurs v3
Flash
L'activité sera affichée à l'aide de Flash
mobile
L'activité sera affichée dans un format adéquat pour la visualisation d'appareils mobiles tels que les téléphones cellulaires.
