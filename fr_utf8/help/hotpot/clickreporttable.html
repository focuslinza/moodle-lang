Cheminement des clics

Ce tableau montre en détail les clics de chaque étudiant dans une activité Hot Potatoes, pour peu qu'il l'ai fini.

Attention : ce tableau est très fourni et peut donc être très long. Il est donc plutôt destiné à être téléchargé et exploité à l'aide d'un tableur tel que Microsoft Excel ou OpenOffice.org Calc.

Chaque ligne du tableau comporte les informations suivantes :

Quel clic ?
Ces informations déterminent de quel clic il s'agit, à quelle tentative, par quel étudiant et pour quel quiz dans quelle section de cours. Cette ligne renvoie au :

  • nom du cours, le numéro de section, le numéro de l'exercice, le nom de l'exercice, le type d'exercice, le nombre de questions ;
  • nom complet de l'étudiant, la date de connexion, l'heure de connexion, l'heure de déconnexion ;
  • numéro de la tentative, l'heure de début de la tentative, l'heure de fin de la tentative ;
  • numéro du clic, l'heure du clic, le type du clic.

Quelles sont les questions concernées ?
Ceci détaille les questions, le cas échéant, qui sont concernées par le clic :

  • les questions qui ont fait l'objet d'une validation, d'un conseil ou d'un indice lors de ce clic ;
  • la situation de chaque question après ce clic :
    • 0 signifie « bonne réponse »,
    • X signifie « mauvaise réponse »,
    • - signifie « non répondu » ;
  • les nouvelles réponses reçues lors de ce clic (les réponses sans changement ne sont pas reprises) ;
  • les questions pour lesquelles une nouvelle réponse a été enregistrée par ce clic ;
  • les questions ayant fait l'objet d'une nouvelle réponse, d'un indice ou d'un conseil, lors de ce clic.

Combien de questions sont-elles concernées ?
Ceci détaille le nombre total de questions affectées par le clic, et le cumul depuis le début du quiz :

  • le nombre de questions répondues ;
  • le nombre de questions non répondues ;
  • le nombre de questions ayant reçu une « bonne réponse » ;
  • le nombre de questions ayant reçu une « mauvaise réponse » ;
  • le nombre de questions pour lesquelles un conseil a été donné ;
  • le nombre de questions pour lesquelles un indice a été donné ;
  • le nombre de questions ayant reçu une nouvelle réponse ;
  • le nombre de questions ayant fait l'objet d'une nouvelle réponse, d'un indice ou d'un conseil.

Notes
Sont indiquées les notes reçues pour ce clic et en cumul depuis le début du quiz :

  • la note brute ;
  • la note maximale ;
  • la note, en pourcentage.
