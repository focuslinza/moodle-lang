Importer des groupes

Cet outil permet l'importation de groupes dans Moodle.

  • Chaque ligne du fichier d'importation comporte un enregistrement.
  • Chaque enregistrement est constitué d'une série de champs séparés par des virgules.
  • La première ligne du fichier contient la liste des noms des champs, qui définit le format du reste du fichier.
    • Champ requis : ces champs doivent être mentionnés dans la première ligne du fichier et définis pour chaque groupe.
      groupname
    • Champs par défaut : ces champs sont optionnels ; s'ils ne sont pas inclus, leurs valeurs sont paramétrées par la langue actuelle et le cours actuel
      idnumber, coursename, lang
    • Champ optionnels : tous ces champs sont complètement optionnels.
      description, picture, hidepicture
  • Les virgules à l'intérieur des données doivent être encodés &#44. Le script traduira automatiquement ces codes en virgules lors de l'importation.
  • Pour les champs booléens, utilisez 0 pour faux et 1 pour vrai.
  • Pour identifier le cours, on peut utiliser soit idnumber, soit coursename. Si tous les deux sont spécifiés, idnumber l'emporte sur coursename. Si aucun des deux n'est spécifié, les groupes seront ajoutés au cours actuel.
  • Le champ coursename désigne le nom abrégé du cours.
  • Remarque : si un groupe existe déjà dans la base de données de Moodle pour un cours, le script d'importation retourne le nom de ce groupe. Les utilisateurs n'ont le droit d'importer des groupes que dans les cours où ils disposent de la capacité d'édition.

Voici un exemple d'un fichier d'importation valable :

groupname,idnumber,lang,description,picture
Groupe1, Philo101, fr, ce groupe requiert plus d'attention, 0
Groupe2, Maths243, , ,
