Réglages de fenêtre

Ces paramètres peuvent être masqués/affichés en cliquant sur le bouton « Afficher les réglages » ou « Cacher les réglages.»

Habituellement, un paquetage SCORM est affiché dans une navigation normale disposant les contrôles de Moodle en haut de l'écran et la table des matières du module sur la gauche. Si vous le choisissez, vous pouvez afficher le contenu du paquetage dans une nouvelle fenêtre.

Cependant, vous devrez procéder comme ceci :

  1. Dans la section Fenêtre :, cochez l'option de liste « Nouvelle fenêtre.»
  2. Déterminez la taille de fenêtre que vous voulez voir ouvrir.

Aujourd'hui, la résolution habituelle « grand public » est considérée comme étant 1024x768. Dans la plupart des cas, il est préférable d'ouvrir à la dimension standard inférieure, comme par exemple 800 par 600.

Si vous indiquez dans ces deux paramètres une valeur de 100% alors la fenêtre popup remplira l'écran. Les valeurs en pourcentage définissent une portion proportionnelle aux dimensions de l'écran.

Si vous laissez ces paramètres vides, la nouvelle fenêtre aura exactement les mêmes dimensions que celles qui contient le lien qui l'appelle.
