Ajouter/envoyer message

Cette option vous permet de sélectionner plusieurs utilisateurs afin de leur envoyer simultanément un message.

Étendre la période d'inscription

Dans le cas où une durée d'inscription est fixée dans votre cours, grâce à cette option, vous pouvez prolonger cette période jusqu'à une année pour les étudiants sélectionnés.
