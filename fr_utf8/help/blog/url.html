URL

La plupart des blogs ont la capacité de diffuser leurs contenus sous la forme d'une URL de flux RSS. Consultez la configuration et la documentation de votre blog externe si vous n'êtes pas sûr de cette URL. Quand vous l'avez, entrez-la dans ce champ. Une fois le formulaire soumis, votre lien sera vérifié pour voir si l'URL est valide.
