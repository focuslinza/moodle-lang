Choisir une page initiale

Vous pouvez définir quel sera le contenu initial de la première page du Wiki.

Si vous voulez définir votre propre contenu, créez-le sous forme de fichiers textes, téléchargez ces fichiers dans l'espace fichiers du cours. Sélectionnez enfin l'un de ces fichiers comme fichier initial. Toutes les pages chargées deviennent le contenu de départ de chaque structure(*) de Wiki présente dans cette instance d'activité, la page sélectionnée devenant la page d'entrée.

Attention ! La page initiale devra être nommée suivant le format CamelCase, c'est-à-dire un seul mot, avec les initiales des mots liés en majuscule (par exemple ProfilPersonnel, ELearning, PortFolio, SousMarin, etc.), et sans caractères spéciaux (tiret, caractère souligné, etc.) ni extension.

(*) Rappel, une instance de Wiki peut abriter plusieurs structures parallèles selon l'action des paramètres de groupe.
