Options de liens automatiques

Dans certaines situations, vous pouvez souhaiter de ne pas activer les hyperliens automatiques sur la base des mots CamelCase. Si c'est le cas, cochez cette case pour désactiver ces transformations.

AVERTISSEMENT -- Les hyperliens CamelCase sont une fonctionnalité typique des Wikis. Les désactiver risque de provoquer des erreurs lors de l'import d'autres contenus Wikis. N'utilisez cette fonction que si vous savez parfaitement ce que vous faites.
