Mode HTML

Vous avez les possibilités suivantes :

Pas de code HTML
Toutes les balises HTML sont filtrées. Le formatage du contenu ne se fait qu'avec le marquage Wiki. Les filtres fonctionnent dans ce mode.
HTML sûr
Certaines balises HTML sont autorisées. Aucun filtre n'est actif dans ce mode. Les balises de filtrage sont ignorées !
HTML uniquement
Pas de syntaxe Wiki, que du HTML(*). Cette option permet l'utilisation de l'éditeur WYSIWYG.

* Remarque : en fait, vous pouvez utiliser la syntaxe Wiki en mode « code source » de l'éditeur WYSIWYG, mais les codes Wiki sont transformés en HTML équivalent et ne sont donc pas persistants.
