Blocage IP

Chaque entrée doit tenir sur une ligne.

Par exemple: 192.168. , 231.54.211.0/20, 231.3.56.211, 231.3.56.10-20

Vous pouvez utiliser quatre types de numérotation (vous ne pouvez pas utiliser de noms de domaine sous forme de texte comme exemple.com):

  1. Les adresses IP complètes comme192.168.10.1 qui correspondent à un ordinateur unique (ou proxy).
  2. Les adresses partielles, comme 192.168 qui correspondent à tout ce qui commence par ces chiffres.
  3. La notation CIDR, comme 231.54.211.0/20 qui vous permet de détailler les sous-réseaux.
  4. Une série d'adresses IP, comme 231.3.56.10-20 La série porte sur la dernière partie de l'adresse, c'est-à dire toutes les adresses IP de 231.3.56.10 à 231.3.56.20.

Les espaces sont ignorés.
