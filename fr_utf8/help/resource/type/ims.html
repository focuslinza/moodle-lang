Paquetages de contenu IMS

Ce type de ressource vous permet d'ajouter à votre cours des paquetages conforme aux spécifications IMS content packaging. Ces paquetages peuvent être créés et modifiés à l'aide de divers logiciels auteurs. Le contenu est affiché sous la forme de plusieurs pages web, avec une navigation entre les pages. Le logiciel auteur produit un fichier compressé ZIP, qui peut être déposé dans votre cours Moodle.

Ce fichier compressé est automatiquement décompressé par Moodle et le contenu affiché.

Différentes options existent pour afficher le contenu dans une fenêtre surgissante (popup), avec un menu de navigation ou des boutons, etc.
