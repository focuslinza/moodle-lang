Incorporer des fichiers dans un cadre

Cette option permet d'afficher un fichier dans un cadre, de sorte que la navigation de Moodle navigation reste sur la page, dans le cadre supérieur.

Cette option n'est normalement pas nécessaire pour les fichiers multimédias comme les séquences vidéo, fichiers audio ou flash. En effet, sans cette option il seront intégré à une page avec navigation.

Il est à noter que l'utilisation de cadres dans un site web peut entraver son accessibilité. Cette option est donc complètement ignorée pour les utilisateurs ayant spécifié l'option « Lecteur d'écran » dans leur profil.
