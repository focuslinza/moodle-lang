Incorporer des fichiers

Cette option permet d'afficher un fichier dans un cadre ou de l'incorporer dans une page, de sorte que la navigation de Moodle navigation reste sur la page, dans le cadre supérieur.

Cette option n'est normalement pas nécessaire pour les fichiers multimédias comme les séquences vidéo, fichiers audio ou flash. En effet, sans cette option ils seront intégrés à une page avec navigation.

Il est à remarquer que l'utilisation de balises de cadres dans un site web peut entraver son accessibilité. Cette option est donc complètement ignorée pour les utilisateurs ayant spécifié l'option « Lecteur d'écran » dans leur profil.

L'utilisation de cadres n'est pas conforme au format XHTML strict. Cependant, l'option « sans cadre » pourrait détériorer l'affichage de certains thèmes Moodle. Par conséquent, si la ressource s'affiche de façon satisfaisante sans cadre, il est conseillé de s'en tenir à cette option plutôt qu'à celle « avec cadre ».
