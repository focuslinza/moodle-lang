Balises disponibles

Les balises sont des paramètre fictifs du modèle, qui seront remplacés par des champs ou des boutons lorsque les fiches sont modifiés ou affichées.

Les champs ont le format suivant : [[nom_de_champ]]

Les boutons ont le format suivant : ##un_button##

Seuls les balises dans la listes des « Balises disponibles » peuvent être utilisées dans le modèle actuel.
