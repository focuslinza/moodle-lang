Nombre maximal de réponses dans une leçon

Cette valeur détermine le nombre maximal de réponses qu'un enseignant peut utiliser. La valeur par défaut est de 4. Si une leçon n'utilise que des questions Vrai/Faux, il est sensé de régler ce paramètre à 2.

Ce paramètre fixe également le nombre maximal de branches pouvant être utilisée dans une table de branchement.

On peut modifier sans problème cette valeur dans une leçon contenant déjà des pages. Si vous avez besoin d'ajouter une question ou une table de branchement avec de nombreux choix, il sera nécessaire de la modifier. Une fois la question ou la table de branchement inhabituelle définie, vous pourrez réduire ce paramètre à une valeur plus normale.
