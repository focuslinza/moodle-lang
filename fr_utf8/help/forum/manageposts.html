Nombre maximal de messages

Le concept de nombre maximal de messages fonctionne de la façon suivante. Les participants ne peuvent plus poster de message durant une période donnée, après en avoir écrit un certain nombre. Lorsque le nombre de messages écrits s'approche du nombre maximal, ils reçoivent une notification le leur indiquant.

Aucune notification n'est envoyée lorsque le « Nombre de messages avant notification » est de 0. Pour désactiver le blocage, on fixe à 0 le « Nombre maximal de messages.» Lorsque le blocage est désactivé, les notifications sont automatiquement désactivées également.

Ces régalges n'affectent pas les possibilités des enseignants.
