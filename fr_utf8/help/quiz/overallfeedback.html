Feedback général

Le feedback général est un texte qui montré à l'étudiant lorsqu'il a terminé une tentative d'un test. Le texte affiché peut différer suivant la note obtenue par l'étudiant.

Par exemple, si vous tapez « Bien joué » dans le premier champ de feedback, « 40% » dans la premier champ note limite et « Merci d'étudier encore cette matière » dans le deuxième champ de feedback, le message « Bien joué » sera affiché pour les étudiants obtenant une note de 40% ou plus et les autres étudiants verront l'autre message.

Les limites de note doivent être indiquées soit sous la forme d'un pourcentage, par exemple « 31.41% », soit sous la forme d'un nombre, par exemple « 7 ». Si la note maximale de votre test est de 10, une limite de 7 signifie 7 sur 10 ou plus.
