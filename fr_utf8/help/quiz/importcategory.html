Importer des catégories

Le menu Catégorie : est utilisé pour sélectionner la catégorie dans laquelle les questions importées seront placées.

Certains format d'importation (formats GIFT et XML) permettent de spécifier la catégorie à l'intérieur du fichier d'importation. Pour que l'importation des catégories ait lieu, la case à cocher Depuis le fichier doit être cochée. Si ce n'est pas le cas, les questions seront placées dans la catégorie spécifiée dans l'interface, quelles que soient les instructions du fichier.

Lorsque les catégories sont indiquées dans le fichier d'importation, elles seront créées si elles n'existent pas encore.
