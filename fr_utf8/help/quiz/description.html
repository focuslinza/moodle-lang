Description

Ce type n'est pas vraiment une question.

Il permet d'afficher un bloc de texte à l'intérieur d'un test sans que l'étudiant ait à répondre quoi que ce soit. On peut l'utiliser pour donner des directives pour les questions qui suivent.

Ce texte est afficher durant la tentative ainsi que durant la relecture du test. Si vous voulez qu'un texte ne s'affiche que durant la relecture du test, vous pouvez utiliser le feedback général.
