Téléchargement - vue d'ensemble

Vous pourriez vouloir télécharger les données affichée pour une analyse approfondie. Vous pouvez choisir parmi deux formats de fichier pour télécharger ces données.
Dans chacun des cas, les données seront affichées à l'écran, dans une feuille de calcul avec les titres appropriés, de même qu'à l'écran. Si le tableau s'étend sur plusieurs pages, alors les données de toutes les pages seront téléchargées dans un fichier unique.

Télécharger en format ODS

Produira un classeur au format .ods

Télécharger au format Excel

Produira un classeur au format .xls

Télécharger en format texte

Dans ce cas, les données seront stockées dans un fichier texte : une ligne de texte pour chaque ligne du tableau, les données étant séparées par des tabulations.
