Téléchargement des réponses

Si vous souhaitez télécharger les données du tableau à l'écran, vous avez le choix entre deux formats d'exportation.

Dans tous les cas, le fichier comportera le tableau, avec les titres appropriés, à l'instar de l'affichage à l'écran (à l'exception des photos des utilisateurs).

Le fichier téléchargé comportera toutes les données, même si celles-ci sont affichées à l'écran sur plusieurs pages. Les utilisateurs n'ayant effectué aucune tentative au test peuvent être intégrés au fichier, pour peu que vous cochiez l'option correspondante.

Format classeur Excel :

Vous obtiendrez un document au formal .xls

Format texte 

Les informations figureront dans un fichier texte classique. Une ligne pour chaque ligne du tableau, chaque champ étant séparé par une tabulation.
