Jeux de questions

Un jeu de questions est une collection de données utilisées pour créer une question (telle qu'une question calculée) comportant une variable.

Vous pouvez créer un jeu de questions « privé » pour une question particulière, ou un jeu de questions « réutilisable » qui peut être utilisé par toutes les questions d'une catégorie.

Pour créer une question recourant aux jeux de questions, vous devrez parcourir deux écrans.

Le premier écran vous permet de déterminer le jeu de questions devant remplacer chaque variable.

Le second écran vous permet d'ajouter ou supprimer des groupes de valeurs au jeu de questions. Les nombres utilisés par les questions individuelles soumises aux étudiants seront choisies ici.
