Importation de questions

Cette fonction vous permet d'importer des questions à partir de fichiers externes en format texte, déposés à l'aide d'un formulaire, ou en choisissant un fichier texte approprié existant déjà dans la zone des fichiers.

Remarque sur l'encodage des caractères

Si votre fichier d'importation comporte des caractères ne faisant pas partie du jeu de base ASCII, il doit être formaté dans l'encodage UTF-8. Il est possible de convertir ou d'exporter votre fichier en format UTF-8 à l'aide de votre éditeur de texte favori. Soyez particulièrement prudent avec les fichiers générés par les applications de Microsoft Office, qui utilisent un encodage non standard qu'il n'est pas possible de traiter correctement. Les fichiers d'importation basé sur le langage XML seront traités automatiquement pour autant que votre serveur Moodle utilise PHP 5 ou une version ultérieure.

Les formats de fichiers ci-dessous sont supportés.

Format GIFT

Le format GIFT est le plus complet des formats d'importation de questions disponible dans Moodle. Il a été conçu pour permettre aux enseignants d'écrire simplement des questions dans un fichier texte. Il permet la saisie des questions à Choix multiples, Vrai-Faux, à Réponse courte ou à Réponse numérique, ainsi que l'insertion de _____ pour le format Mot manquant. Les différents types de questions peuvent être mélangés dans un même fichier texte, et le format permet également l'insertion de commentaires, titres de questions, feedbacks et pondération des points en %. Voici quelques exemples :

 Quelle était la couleur de la jument blanche
 de Napoléon ?{~Verte ~Noire =Blanche}
 
 La jument blanche de Napoléon était {~verte =blanche ~noire}.
 
 La jument blanche de Napoléon était verte.{FALSE}
 
 Qui Napoléon a-t-il épousé ?{=Marie-Louise =Joséphine}
 
 Quelle est la date de naissance de Napoléon ?{#1769:5}

Format Aiken

Ce format simple permet d'importer des questions à choix multiple à partir d'un fichier texte.

Format XML Moodle

Ce format spécifique à Moodle importe des questions ayant été exportées auparavant dans le même format. Son utilisation est immédiate. On peut la tester facilement en exportant une catégorie dans le format XML Moodle. Ce format permet l'importation de fichiers images.

Plus d'informations au sujet du format XML Moodle.

Format Mot manquant

Ce format ne supporte que des questions à choix multiples ou à réponse courte, suivant le nombre de réponses possibles. Lorsqu'il n'y a qu'une réponse proposée, une question à réponse courte est générée. S'il y a deux ou plus de réponses proposées, une question à choix multiples est générée. Chaque réponse est précédée d'un tilde (~), sauf la réponse correcte qui est précédée d'un signe d'égalité (=). Voici un exemple pour une question à choix multiples :

Durant l'enfance, quand nous commençons à explorer les parties de notre corps, nous devenons des étudiants {=en anatomie et physiologie ~en science ~en réflexologie}, et dans un sens, nous demeurons des étudiants pour toute la vie.

Et voici un exemple pour une question à réponse courte :

Durant l'enfance, quand nous commençons à explorer les parties de notre corps, nous devenons des étudiants {=en anatomie et physiologie}, et dans un sens, nous demeurons des étudiants pour toute la vie.

Plus d'informations au sujet du format Mot manquant.

Format BlackBoard

Ce module permet d'importer des questions enregistrées dans le format « POOL » de BlackBoard. Il utilise les fonctions XML du PHP utilisé par Moodle. Il vous sera nécessaire de décompresser le fichier ZIP exporté de BlackBoard et de fournir à Moodle le fichier .dat adéquat. Cette version ne fonctionne pas avec les versions 6 et ultérieures de BlackBoard.

Plus d'informations au sujet du format BlackBoard.

Blackboard V6+

Ce format permet l'importation de questions exportées dans le format Blackboard. Il fournit un support partiel des fonctionnalités de Blackboard Version 6 et 7. Il utilise les fonctions XML du PHP utilisé par Moodle. Vous devez déposer le fichier ZIP exporté par Blackboard.

WebCT

Ce format permet l'importation de questions exportées dans le format texte de WebCT.

Plus d'informations au sujet du format WebCT.

Format Cloze

Ce type particulier de format n'importe qu'un seul type de questions, le type Cloze.

Plus d'informations au sujet du format Cloze.

Format Learnwise

Ce format permet d'importer des questions à choix multiples enregistrées en format XML Learnwise.

Format Examview

Ce format permet d'importer des questions à partir de fichier XML Examview 4. Les versions plus anciennes d'Examview ne sont pas supportées.

Les formats d'importation et d'exportation sont des ressources facilement installables (plug-in). D'autres formats sont disponibles dans la base de données « Modules et Plugins ».
