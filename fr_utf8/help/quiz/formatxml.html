Indiquer une catégorie

Pour spécifier une catégorie dans laquelle les questions suivantes seront placées, il est possible d'insérer un type de « question » dans le fichier XML. Si la catégorie indiquée n'existe pas, elle sera créée. Pour que l'importation des catégories ait lieu, la case à cocher Depuis le fichier doit être cochée. Si ce n'est pas le cas, les questions seront placées dans la catégorie spécifiée dans l'interface, quelles que soient les instructions du fichier. Vous pouvez changer cette catégorie autant de fois que vous le voulez à l'intérieur du fichier.

Voici un exemple de format :

<question type="category">
    <category>
        sciences/maths/analyse
    </category>
</question>
