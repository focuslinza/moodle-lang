Catégories de questions

Plutôt que de conserver vos questions dans une seule grande liste, vous pouvez les classer en catégories.

Chaque catégorie comporte un nom et une courte description.

Chaque catégorie peut également être « publiée », C'est à dire que la catégorie (et les questions qu'elle contient) seront disponibles pour tous les autres cours du serveur, si bien que d'autres cours pourront employer vos questions dans leurs tests.

Description des champs :

Parent : La catégorie dans laquelle celle-ci sera créée. Si aucune autre catégorie n'a encore été créée, « Racine » sera le seul choix possible.

Catégorie : Le nom de la catégorie.

Description : Une brève description de la catégorie.

Publiée : catégorie à publier immédiatement, ou non.
