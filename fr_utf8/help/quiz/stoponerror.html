Stopper en cas d'erreur

L'importation des questions a lieu en deux phases. Dans un premier temps, le fichier à importer est analysé, afin de vérifier sa validité. Dans une deuxième phase, les qeustions sont inscrites dans la base de données. Si l'option « Stopper en cas d'erreur » est cochée, lorsqu'une erreur est détectée dans le fichier, la deuxième phase ne sera pas exécutée, et aucune donnée ne sera écrite dans la base de données.

Si l'option n'est pas cochée, Moodle tentera d'écrire dans la base de données les questions valides du fichier à importer.

Par défaut, cette option est activée.
