Options d'analyse

Vous pouvez choisir quelles tentatives au test retenir dans l'analyse, en précisant les paramètres suivants :

Choix de la tentative :

Il peut être utile de n'analyser qu'une tentative pour chaque utilisateur. Cette tentative peut être la première ou la dernière tentative ou celle ayant recueuilli la meilleure note. Autrement, toutes les tentatives peuvent être retenues pour une analyse cumulative.

Ignorer les notes minimes :

Il arrive que les utilisateurs parcourent simplement les tests sans chercher à les mener à leur terme. Ces tentatives « pour voir » obtiennent généralement des notes très faibles. Elles peuvent être ignorées lors de l'analyse, en définissant une note-plancher en-deçà de laquelle les notes seront ignorées. Cette note-plancher prend la forme d'un pourcentage (de 0 à 100) de la note maximale possible dans le test.

Taille de la page :

Vous pouvez choisir combien de questions afficher par page.
