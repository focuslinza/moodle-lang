Questions Cloze à réponses intégrées

Ce type de question est très flexible. Il est analoque au format Cloze, assez répandu.

Les questions se composent d'un texte (en format Moodle) dans lequel on peut intégrer des réponses à choix multiples, des réponses courtes ou des réponses numériques.

Il n'existe à ce jour pas d'interface graphique pour créer de telles questions - vous devez les écrire directement dans la boîte de texte ou en les importer à partir d'un fichier externe.
