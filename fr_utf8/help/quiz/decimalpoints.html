Décimales

Ce paramètre vous permet de choisir le nombre de décimales à afficher dans les scores ou dans les notes. Par exemple, si vous choisissez « 0 », les nombres seront arrondis en nombres entiers.

Ces réglages n'agissent que sur l'affichage des notes, et n'altèrent pas les calculs internes et les arrondis de celles-ci.
