Note requise

Quand cette option est activée, les étudiants doivent obtenir une note à l'activité pour la terminer. Par exemple, un quiz sera marqué comme terminé dès que l'étudiant l'aura soumis.

Le résultat au quiz n'importe pas. Quelle que soit la note, l'activité sera marquée comme terminée.

Distinction entre réussite et échec

Il est possible de distinguer les notes qui signifient une réussite des notes qui signifient un échec, de telle sorte que l'activité soit marquée comme « terminée, réussie » ou « terminée, non réussie » au lieu d'un simple « terminée ». Ces résultats présentent une icône différente et un texte alternatif.

Pour utiliser cette option, vous avez besoin d'attribuer à cette activité une note pour passer :

  1. Allez sur le carnet de notes du cours en cliquant sur « Notes » dans le bloc Administration du cours.
  2. Dans le menu déroulant « Choisir une action », sélectionnez « Catégories et éléments ».
  3. Cliquez sur l'icône Modifier à côté de l'élément d'évaluation de cette activité.
  4. Cliquez sur « Afficher éléments supplémentaires ».
  5. Entrez une valeur (par exemple 5.0) dans le champ « Note pour passer ».

De cette façon, quiconque soumet le quiz se verra attribuer une icône de réussite ou d'échec. Si le quiz autorise plusieurs tentatives, l'icône sera automatiquement mise à jour à chaque nouveau score.

Il existe une limite : cela fonctionne uniquement si les notes sont immédiatement visibles par les étudiants. Les notes ne doivent jamais être cachées ni de façon permanente ni jusqu'à une certaine date. Si une note est cachée alors seul le statut standard « terminée » sera affiché - même après l'échéance.
