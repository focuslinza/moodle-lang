Obtenir un code d'autorisation par téléphone

Si la carte de crédit d'un utilisateur ne peut pas être saisie directement sur Internet, il faut obtenir par téléphone un code d'autorisation de la banque de l'utilisateur.

Certains utilisateurs ne veulent pas utiliser leur carte de crédit directement sur Internet pour des raisons de sécurité. Dans ce cas, vous devez obtenir un code d'autorisation de la banque de l'utilisateur. Dans un premier temps, demandez à votre client par téléphone, face à face ou à l'aide d'un formulaire les informations de la carte de crédit : nom de la banque, nom sur la carte, numéro de carte, date d'échéance et code de validation de la carte. Appelez ensuite le service clientèle de la banque mentionnée en donnant ces informations et demandez un code d'autorisation. Pour terminer, après avoir obtenu ce code, connectez-vous en tant que cet utilisateur pour l'inscrire. Vous pouvez aussi le donner à l'utilisateur pour qu'il s'inscrive lui-même.
