Enseignants

Vous trouverez sur cette page la liste des personnes désignées comme enseignants dans ce cours (par un administrateur du site ou un créateur de cours).

Vous pouvez utiliser ce formulaire pour donner un nom à ce rôle : professeur, enseignant, gourou, tuteur, assistant, mentor, etc. Ce titre apparaîtra dans la liste des cours du site. Si vous ne définissez pas ce titre, le nom « enseignant » leur sera attribué.

Vous pouvez également modifier l'ordre dans lequel apparaîtront les enseignants. Choisissez le nombre désiré pour chaque personne dans la colonne « Ordre ». Les changements seront affichés une fois la liste enregistrée.

Remarque : Vous pouvez cacher le nom d'un enseignant en choisissant « Caché ». Dans ce cas, le nom de l'enseignant n'apparaîtra pas dans la liste des cours, ni dans la liste des participants. Le nom de l'enseignant ne sera visible que s'il participe à des activités, par exemple à un forum.

Il est aussi possible de déterminer si un enseignant est un éditeur ou non dans le cours. Les enseignants qui ne sont pas éditeurs dans un cours sont à considérer comme des tuteurs.

  1. Ils ne peuvent pas changer la structure du cours.
  2. Ils ne peuvent pas inscrire ou désinscrire des étudiants dans le cours.
  3. Ils peuvent donner des notes.
  4. Ils peuvent participer aux activités pour donner de l'aide.

Lorsqu'ils font partie d'un groupe, ils ne peuvent effectuer les tâches 3 et 4 ci-dessus que dans ce groupe.
