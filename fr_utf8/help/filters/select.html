Menu filtre

Ce filtre vous permet de sélectionner des informations sur la base d'un choix dans un menu déroulant. Les options suivantes sont disponibles :

  • est n'importe quelle valeur - cette option désactive le filtre (c'est-à-dire que toutes les informations sont sélectionnées par le filtre)
  • est égal à - cette option sélectionne uniquement les informations égales à la valeur choisie dans le menu
  • n'est pas égal à - cette option sélectionne uniquement les informations différentes de la valeur choisie dans le menu
