Filtre de profil

Ce filtre vous permet de sélectionner des utilisateurs sur la base des valeurs des champs de leur profil utilisateur. Le filtre peut être appliqué sur un seul champ du profil ou sur la totalité des champs. Les options suivantes sont disponibles :

  • contient - cette option sélectionne les utilisateurs dont le champ spécifié contient le texte saisi (si aucun texte n'est saisi, le filtre est désactivé)
  • ne contient pas - cette option sélectionne les utilisateurs dont le champ spécifié ne contient pas le texte saisi (si aucun texte n'est saisi, le filtre est désactivé)
  • est égal à - cette option sélectionne les utilisateurs dont le champ spécifié est égal au texte saisi (si aucun texte n'est saisi, le filtre est désactivé)
  • commence par - cette option sélectionne les utilisateurs dont le champ spécifié commence par le texte saisi (si aucun texte n'est saisi, le filtre est désactivé)
  • se termine par - cette option sélectionne les utilisateurs dont le champ spécifié se termine par le texte saisi (si aucun texte n'est saisi, le filtre est désactivé)
  • est vide - cette option sélectionne les utilisateurs dont le champ spécifié est vide, mais que le champ existe (le texte saisi is ignored)
  • n'est pas défini - cette option sélectionne les utilisateurs dont le champ spécifié n'est pas défini (le texte saisi est ignoré)
  • est défini - cette option sélectionne les utilisateurs dont le champ spécifié est défini (le texte saisi est ignoré)
