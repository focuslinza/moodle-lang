A Moodle dokumentációja
--------------------

Ha a dokumentációt egy másik nyelvre kívánja lefordítani,
ajánlatos a következőket megfogadni:

 1) ne tegye az angol állományokat ebbe a könyvtárba, mert ha valamelyik hiányzik a nyelvi
    csomagjából, helyette úgyis automatikusan az angolt használja a rendszer.

 2) a fordítást az alábbi sorrendben végezze (kezdve a fontosakkal):
  
    LEGFONTOSABBAK
    |-------------------
    |
    |   - files.php
    |   - install.html
    |   - installamp.html
    |   - upgrade.html
    |   - faq.html
    |
    |  -----------------
    |
    |   - teacher.html
    |   - module_files.txt
    | 
    |  -----------------
    |   
    |   - intro.html
    |   - features.html
    |   - release.html
    |
    |  -----------------
    |
    |   - developer.html
    |   - cvs.html
    |   - future.html
    |
    |  -----------------
    |   
    |   - license.html
    |
    |-------------------
    LEGKEVÉSBÉ FONTOSAK

 3) Ne fordítsa le a credits.html-t: túl gyakran változik.
