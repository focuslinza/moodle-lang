Kaayusan ng Bintana

Ang mga kaayusang ito ay maaaring itago/ipakita sa pamamagitan ng pagklik sa buton na may pangalan na Itago ang kaayusan o Ipakita ang kaayusan.

Kadalasan, ang mga rekurso ay ipinapakita sa isang normal na nakakuwadrong window, na nasa taas ang pangnabigasyon na mga kontrol ng Moodle sa kuwadrong nasa tuktok. Kung naisin mo, maaalis mo ang kuwadrong ito at punuin ng rekurso ang bintana tulad ng isang normal na pahinang pangweb.

Magkagayunman, kung nais mong lumitaw ang rekurso sa isang bagong popup na bintana ay sundin ang mga sumusunod na hakbang:

  1. Sa seksiyon na Bintana: piliin ang 'Bagong Bintana'.
  2. Alisin ang pagkakapili sa mga opsiyon na hindi mo gustong gamitin.
    Pahihintulutan bang mabago ang laki ng bintana?

    Ang pagpapanatili sa pagkakapilì ng aytem na ito ay magpapahintulot sa mga user na mabago ang laki ng bintana na itinakda mo, at halos palaging dapat na pabayaang pilì. Magkagayunman, minsan ang rekurso ay may permanenteng laki at mas makatwiran na ipirmi ang laki ng bintana.

    Pahihintulutan bang mai-scroll ang bintana?

    Pinapahintulutan ng aytem na ito ang mga user na i-scroll ang bintana para makita ang lahat ng rekurso mo. Kung walang tsek ang aytem na ito, makikita lamang ng user kung ano ang kasya sa itinakdang laki ng bintana at hindi pahihintulutan na i-scroll ang bintana para makita ang kabuuan ng rekurso.

    Ipapakita ba ang mga link ng direktoryo?

    Pahihintulutan nito ang mga user na makita ang mga link ng direktoryo sa bintanang ito. Nagbabago ito ayon sa browser: Maari ito ang Links Bar (sa IE), ang Favourites Bar o Bookmarks bar (sa Mozilla/Firefox).

    Ipapakita ba ang bar ng lokasyon?

    Gusto mo bang makita ng mga user ang URL (address ng web) ng rekurso? Kung oo, pabayaan mong may tsek ang opsiyong ito. Kundi ay matatago ang lokasyon ng rekurso.

    Ipapakita ba ang bar ng menu?

    Upang mabigyan ng karapatan ang mga user na gamitin ang menu bar sa bintanang ito, pabayaan na walang tsek ang aytem na ito. Kapakipakinabang ito pero para sa ilang presentasyon na may larawan mas magandang tingnan na walang menu bar.

    Ipapakita ba ang toolbar?

    Ang pagpapakita ng toolbar ay magpapadali ng pagnabigasyon ng mga user mo, dahil nagbibigay ito ng Back/Forward, Home, at Refresh na mga buton.

    Ipapakita ba ang status bar?

    Ang pagpapakita ng Status bar ay magpapahintulot sa mga tao na matyagan ang impormasyon na pangbrowser, tulad ng anong parte na ng rekurso ang nailowd, o impormasyong panseguridad.

  3. Itakda ang laki ng bintana na nais mong buksan.

    Maraming tao ang gumagamit pa rin ng katamtamang resolusyon ng screen na 800x600. Sa karamihang kaso, pinakamabuti na paliitin mo muna nang kaunti dito ang bintana, tulad ng 600 taas at 480 lapad.

    Kapag pinabayaan mong blangko ang mga puwang na ito, ang bagong popup na bintana ay magiging eksaktong kasinglaki ng bintana na ikinlik nila para makuha ang popup.
