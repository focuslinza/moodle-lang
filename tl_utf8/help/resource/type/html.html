Pahinang HTML

Ang uri ng rekurso na ito ay ginagawang madali ang paggawa ng nag-iisang kumpletong pahinang pangweb sa loob ng Moodle, lalo pa kung ginagamit mo ang WYSIWYG HTML editor ng Moodle.

Ang pahina ay iniimbak sa database, hindi bilang file, at mayroon kang kalayaan na gawin ang halos lahat ng bagay na nais mo, gamit ang HTML, at puwede rin ang Javascript.
