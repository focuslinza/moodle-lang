Pag-angkat ng mga bagong tanong

Sa pamamagitan ng function na ito, puwede kang mag-import ng mga tanong mula sa mga panlabas na text file, na iaaplowd sa pamamagitan ng form na ito.

Tala hinggil sa mga encoding ng titik

Kung may di-ascii na titik ang file mo, kailangan mo munang ikumberte ang file sa UTF-8 sa pamamagitan ng paborito mong editor o ang pangluwas na software (kung magagamit ang opsiyon). Ang mga nakabatay sa XML na pang-angkat na format ay awtomatikong gagamitin, kung PHP 5 ang pinatatakbo mo.

Maraming file format na sinusuportahan:

Format na GIFT

Ang GIFT ang pinakakomprehensibong magagamit na pang-angkat na format para sa pag-aangkat ng mga tanong ng pagsusulit na Moodle mula sa isang tekstong file. Dinisenyo ito na maging madaling paraaan para sa mga gurong nagsusulat ng tanong sa isang tekstong file. Sinusuportahan nito ang Maraming-Pagpipiliang-Sagot, Tama-Mali, Maikling Sagot, Tugmaan at Denumerong tanong, gayundin ang pagsisingit ng _____ para sa "nawawalang salita" na format. Tandaan na ang mga Cloze na tanong ay hindi pa sinusuportahan sa kasalukuyan. Maaaring maghalo-halo ng iba't-ibang uri ng tanong sa isang file na teksto, at sinusuportahan din ng format ang komento na nasa linya, pangalan ng mga tanong, puna at bahagdan na timbang na marka. Nasa ibaba ang ilang halimbawa:

Sino ang inilibing sa puntod ni Grant?{~Grant ~Jefferson =walâ}

Si Grant ay {~nakabaon =inilibing ~nakatira} sa puntod ni Grant.

Si Grant ay nakalibing sa puntod ni Grant.{MALI}

Sino ang inilibing sa puntod ni Grant?{=walâ =walang sinuman}

Kailan ipinanganak si Ulysses S. Grant?{#1822}

Marami pang impo hinggil sa format na "GIFT"

Format na Aiken

Ang format na Aiken format isang napakadaling paraan ng paglikha ng maraming-pagpipiliang-sagot na tanong sa pamamagitan ng napakalinaw na format na mababasa ng tao. Narito ang halimbawa ng format:

Ano ang layunin ng first aid?
A. Upang mailigtas ang buhay, maiwasan ang dagdag na pinsala sa katawan, panatilihin ang kalusugan
B. Upang makapagbigay ng lunas na pangmediko sa sinumang nasaktan o nasugatan
C. Upang maiwasan ang pinsala sa katawan
D. Upang matulungan ang mga biktima na humihingi ng tulong
ANSWER: A

Marami pang impo hinggil sa format na "Aiken"

Nawawalang Salita

Maraming-pagpipiliang-sagot na tanong at maikling-sagot na tanong lamang ang puwede sa format na ito, depende sa dami ng sagot. Ang isang sagot ay lilikha ng maikling-sagot na tanong, samantalang ang dalawa o mahigit pang sagot ay lilikha ng maraming-pagpipiliang-sagot na tanong.

Ang bawat sagot ay pinaghihiwalay sa pamamagitan ng tilde (~), at ang tamang sagot ay minamarkahan sa unahan ng equals sign (=). Narito ang halimbawa ng maraming-pagpipiliang-sagot na tanong:

Sa sandaling pagmasdan natin ang mga bahagi ng ating katawan noong sanggol pa lamang tayo, tayo ay nagi nang mga mag-aaral ng {=anatomy at physiology ~reflexology ~siyensiya ~pag-eeksperimento}, at kung iisipin ay nananatili tayong mga mag-aaral habambuhay.

Narito naman ang halimbawa ng maikling-sagot na tanong:

Sa sandaling pagmasdan natin ang mga bahagi ng ating katawan noong sanggol pa lamang tayo, tayo ay nagi nang mga mag-aaral ng {=anatomiya at physiology}, at kung iisipin ay nananatili tayong mga mag-aaral habambuhay.

Marami pang impo hinggil sa format na "Nawawalang Salita"

AON

Katulad din ito ng Nawawalang Salitang Format, maliban na lamang sa matapos maangkat ang mga tanong, ang lahat ng Maikling-Sagot na tanong ay ikukumberte ng apat-apat sa mga Tugmaan na Tanong.

Dagdag pa, ang mga sagot sa mga maraming-pagpipiliang-sagot na tanong ay random na babalasahin habang inaangkat.

Ipinangalan ito sa isang organisasyon na nag-isponsor ng pagpapaunlad ng maraming katangian ng mga pagsusulit

Blackboard

Maaaring mag-angkat ang modyul na ito ng mga tanong na isinave sa pangluwas na format ng Blackboard. Umaasa ito sa mga XML function na nakacompile sa PHP mo. Tandaan na ang bersiyong ito ay hindi gumagana sa Blackboard Version 6 at mas bago dito.

Marami pang impo hinggil sa format na "Blackboard"

Blackboard V6+

Kayang mag-angkat ng modyul na ito ng mga tanong na isinave sa pang-angkat na format ng Blackboard. Mayroon itong limitadong suporta para sa Blackboard Version 6 at 7. Umaasa ito sa pagko-compile ng mga XML function sa iyong PHP. Kailangan mong iaplowd ang zip file na iniluwas ng Blackboard.

WebCT

Ang modyul na ito ay nakapag-aangkat ng mga tanong na isinave sa tekstong format ng WebCT.

Marami pang impo hinggil sa format na "WebCT"

Course Test Manager

Ang modyul na ito ay makapag-aangkat ng mga tanong na isinave sa Course Test Manager test bank. Umaasa ito sa iba't-ibang paraan ng pagpasok sa test bank, na nasa database na Microsoft Access, depende kung ang Moodle ay tumatakbo sa isang Windows o Linux web server.

Sa Windows, pinapahintulutan nito kayong iaplowd ang access na database tulad ng anumang file na may datos na aangkatin.

Sa Linux, kailangan mong magsaayos ng isang makinang windows sa parehong network kung saan naroon ang Course Test Manager na database at isang piraso ng software na tinatawag na ODBC Socket Server, na gumagamit ng XML sa pagsasalin ng datos papunta sa moodle na nasa Linux server.

Basahin po ang buong pantulong na file sa ibaba bago gamitin ang klase ng pag-aangkat na ito.

Marami pang impo hinggil sa format na "CTM"

Naka-embed na sagot (Cloze)

Ang pang-espesyal na gamit na format na ito ay umaangkat lamang ng isang uri ng tanong, ang Naka-embed na Sagot (tinatawag ding Cloze) na format.

Marami pang impo hinggil sa format na "Cloze"

Format na Learnwise

Ang format na ito ay nakapag-aangkat ng maraming-pagpipiliang-sagot na tanong na isinave sa format na Learnwise's XML.

Format na Examview

Makapag-aangkat ang format na ito mula sa mga file na Examview 4 XML. Tandaan na hindi sinusuportahan ang mas lumang bersiyon ng Examview.

Format na Moodle XML

Ang pang-Moodle na format na ito ay nag-aangkat ng mga tanong ng pagsusulit na naunang iniluwas sa pareho nitong format. Ang format ay etso-diretso at mas madaling ipakita sa pamamagitan ng pagluwas ng isang kategoriya sa format na Moodle XML.

Marami pang format na idadagdag, tulad ng WebCT, IMS QTI at anupamang maiaambag ng mga user ng Moodle!
