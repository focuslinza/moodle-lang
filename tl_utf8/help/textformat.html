Pagpopormat ng teksto

Sa pagsusulat ng teksto sa Moodle, marami kang anyo na mapagpipilian upang malikha ang teksto mo, depende sa kakayanan mo at sa uri ng browser na ginagamit mo.

Kalimitan ay mapapabayaan mo na lamang ang setting na ito sa default nitong halaga at gagana naman ang lahat nang ayon sa inaasahan mo. .

1. Moodle kusang-pagpormat

Pinakamahusay ang anyo na ito kapag ang ginagamit mo ay normal na mga web form sa pagpapasok ng teksto (sa halip na ang HTML editor). Itiklado lang ang teksto nang normal, tulad nang pag nagpapadala ka ng email.

Kapag isinilid mo ang teksto mo, may ilang gagawin ang Moodle upang awtomatikong maipormat ang teksto mo.

Halimbawa, ang mga URL na tulad ng http://yahoo.com o kaya'y www.yahoo.com ay makukumberte sa link.

Pananatilihin ang mga line break mo, at ang mga blangkong linya ay magsisimula ng mga bagong parapo.

Ang mga karakter na smiley tulad ng :-) ay awtomatikong makukumberte sa mga katumbas nilang larawan.

Puwede ka ring mag-embed ng HTML code kung nais mo at mananatili ito.

Dagdag na kaalaman hinggil sa Moodle kusang-pagpormat.

2. Anyong HTML

Inaakala ng anyo na ito na ang teksto ay purong HTML. Kung ginagamit mo ang HTML editor sa pag-edit ng teksto, ito ang default na anyo - lahat ng command sa toolbar ay lumilikha ng HTML para sa iyo.

Kahit hindi mo gamitin ang HTML editor, maaari kang gumamit ng HTML code sa teksto mo at lalabas din naman ito nang ayon sa gusto mong mangyari.

Hindi katulad ng Moodle kusang-pagpormat, walang awtomatikong pagpopormat na nagaganap.

3. Anyong Payak na teksto

Mahalaga ang anyo na ito kapag gusto mong maglagay ng maraming code o HTML na gusto mong madispley nang eksaktong-eksakto sa isinulat mo.

Nagkukumberte pa rin ito ng mga espasyo at bagong linya, pero wala nang iba pang gagalawin sa teksto mo.

4. Anyong Tekstong Markdown

Ginagawa ng markdown anyo na gawing madali ang pagteklado ng mga mahusay ang pagkakapormat na XHTML na pahina, sa pamamagitan lamang ng teksto na isinulat nang parang isinusulat ang isang email.

Napakahusay nito para sa pagsusulat ng malinis na pahina ng teksto na may ilang heading at ilang list pero walang gaanong link o larawan.

Dagdag na kaalaman hinggil sa Markdown na anyo
