Pag-aangkat ng mga bagong tanong

Magagamit mo ang function na ito upang mag-angkat ng mga tanong mula sa mga panlabas na file na teksto, na iaaplowd sa pamamagitan ng isang form.

May ilang format ng file na sinusuportahan:

Format na GIFT

Ang GIFT ang pinakakomprehensibong format na pang-angkat para sa pag-angkat ng mga tanong ng pagsusulit na Moodle, mula sa isang file na teksto. Dinisenyo ito na maging madali ang paraan ng pagsusulat ng mga guro ng tanong sa isang file na teksto. Sinusuportahan nito ang Maraming-Pagpipiliang-Sagot, Tama-Mali, Maikling-Sagot, Tugmaan at Denumero na mga tanong, gayundin ang pagsisingit ng _____ para sa format na "nawawalang salita". Maraming uri ng tanong na mapaghahalo sa iisang file na teksto, at sinusuportahan din ng format ang mga komento na nasa linya, pangalan ng tanong, puna at bahagdan-timbang na marka. Nasa baba ang ilang halimbawa: 

Sino ang inilibing sa puntod ni Grant?{~Grant ~Jefferson =walâ}

Si Grant ay {~nakabaon =nakalibing ~nakatira} sa puntod ni Grant.

Si Grant ay nakalibing sa puntod ni Grant.{MALI}

Sino ang inilibing sa puntod ni Grant?{=wala =walang sinuman}

Kailan ipinganak si Ulysses S. Grant?{#1822}

Marami pang impo hinggil sa format na "GIFT"

Format na Aiken

Ang format na Aiken ay isang napakasimpleng paraan ng paglikha ng maraming-pagpipiliang-sagot na tanong sa pamamagitan ng malinaw na format na puwedeng basahin ng tao. Narito ang halimbawa ng format:

Ano ang layunin ng first aid?
A. Upang magligtas ng buhay, maiwasan ang pinsala sa katawan, at mapanatilì ang kalusugan
B. Upang makapagbigay ng lunas na pangmediko sa sinumang nasaktan o nasugatang tao
C. Upang maiwasan ang dagdag na pinsala sa katawan
D. Upang matulungan ang mga biktima na humihingi ng saklolo
ANSWER: A

Marami pang impo hinggil sa format na "Aiken"

Format na Nawawalang Salita

Maraming-pagpipiliang-sagot na mga tanong lamang ang sinusuportahan ng format na ito. Ang bawat sagot ay pinaghihiwalay ng (~), at ang tamang sagot ay may equal sign (=) sa unahan. Narito ang halimbawa:

Sa sandaling simulan nating pagmasdan ang mga bahagi ng ating katawan noong sanggol pa lamang tayo ay nagsimula na tayong maging mag-aaral ng {=anatomy at physiology ~reflexology ~siyensiya ~experimento}, at kung iisipin ay nananatili tayong mag-aaral panghambambuhay.

AON

Katulad din ito ng Nawawalang Salitang Format, maliban na lamang sa matapos maangkat ang mga tanong, ang lahat ng Maikling-Sagot na tanong ay ikukumberte ng apat-apat sa mga Tugmaan na Tanong.

Dagdag pa, ang mga sagot sa mga maraming-pagpipiliang-sagot na tanong ay random na babalasahin habang inaangkat.

Ipinangalan ito sa isang organisasyon na nag-isponsor ng pagpapaunlad ng maraming katangian ng mga pagsusulit

Blackboard

Maaaring mag-angkat ang modyul na ito ng mga tanong na isinave sa pangluwas na format ng Blackboard. Umaasa ito sa mga XML function na nakacompile sa PHP mo.

Marami pang impo hinggil sa format na "Blackboard"

Course Test Manager

Ang modyul na ito ay makapag-aangkat ng mga tanong na isinave sa Course Test Manager test bank. Umaasa ito sa iba't-ibang paraan ng pagpasok sa test bank, na nasa database na Microsoft Access, depende kung ang Moodle ay tumatakbo sa isang Windows o Linux na web server.

Sa Windows, pinapahintulutan nito kayong iaplowd ang access na database tulad ng anumang file na may datos na aangkatin.

Sa Linux, kailangan mong magsaayos ng isang makinang windows sa parehong network kung saan naroon ang Course Test Manager na database at isang piraso ng software na tinatawag na ODBC Socket Server, na gumagamit ng XML sa pagsasalin ng datos papunta sa moodle na nasa Linux server.

Basahin po ang buong pantulong na file sa ibaba bago gamitin ang klase ng pag-aangkat na ito.

Pasadya

Kung may sarili kang format na kailangan mong angkatin, magagawa mo ito sa sarili mo sa pamamagitan ng pag-edit ng mod/quiz/format/custom.php

Ang kinakailangang bagong code ay napakaliit - sapat lamang para mag-parse ng isang tanong sa isang ibinigay na teksto.
