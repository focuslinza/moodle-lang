Ipakita ang Kaliwang Menu

Magpapakita ito ng listahan ng mga pahina (Mga Manghad ng Sanga) sa aralín.

At, sa pamamagitan ng pagtatakda ng marka na mas mataas sa 0, ang user na kumukuha ng Aralín ay kailangang magkaroon ng marka na katumbas o mas mataas sa marka na itinakda upang makita ang Kaliwang Menu. Magagamit ito ng mga nagdidisenyo ng Aralín na pilitin ang mga user na basahin ang buong aralín sa unang pagkuha nila. Tapos, kapag kinuha muli ng user ang Aralín matapos makuha ang kinakailangang marka, makikita na niya ang Kaliwang Menu para matulungan siya sa pagrerebyu.
