Nilalaman ng Ulat

Maaari kang magtakda ng iba't-ibang nilalaman para sa mga ulat na Hot Potatoes.

Pagpili ng uri ng ulat
Pangkalahatang Tanaw
Ang ulat na ito ay listahan ng mga pagkuha ng pagsusulit na ito. Ang mga "inabandonang" pagkuha ay kasama, pero ang mga "kasalukuyang ginagawa" na pagkuha ay hindi ipapakita. Ipinapakita nito ang larawan at pangalan ng bawat user, pati ang marka nila sa pagsusulit at iskor, oras at tagal ng bawat pagkuha. Ang mga pagkuha ay maaaring piliin at pagkatapos ay burahin kung kinakailangan.
Simpleng estadistika
Ang ulat na ito ay listahan ng lahat ng nakumpletong pagkuha ng pagsusulit na ito. Ang mga pagkuha na may markang "inabandona" o "kasalukuyang ginagawa" ay hindi isasama. Ipinapakita nito ang larawan at pangalan ng bawat user, pati ang marka nila sa pagsusulit at iskor, at ang iskor sa bawat tanong sa bawat pagkuha. Ibinibigay din nito ang katamtamang iskor sa bawat indibidwal na tanong at sa kabuuang pagsusulit.
Detalyadong estadistika
Ipinapakita ng ulat na ito ang buong detalye ng lahat ng nakumpletong pagkuha ng pagsusulit na ito. Ang mga pagkuha na may markang "inabandona" o "kasalukuyang ginagawa" ay hindi isasama. Naglalaman ang ulat ng mga sumusunod na manghad:
  • ang teybol ng mga tugon, na nagpapakita kung paano tumugon ang bawat user sa bawat tanong sa bawat pagkuha nila ng pagsusulit.
      • ang teybol ng pagsusuri ng aytem, na naglilista ng mga tugon sa bawat tanong at ng kalimitan ng bawat tugon.
Ulat ng pagsubaybay sa pagklik
Ipinapakita ng ulat na ito ang detalye ng bawat klik ng bawat mag-aaral sa mga nakumpletong pagkuha ng pagsusulit na Hot Potatoes. Maaaring maging napakalaki ng ulat at sinadya ito na madownload at matingnan sa isang program na spreadsheet, tulad ng Microsoft Excel. Ang mga detalye ng mga hanay ng ulat na ito ay matatagpuan sa iklik ang teybol ng pagsubaybay na tulong na file
Pagpili ng kurso
Ang kursong ito Ipapakita ng ulat na ito ang mga tugon ng mga kalahok sa kasalukuyang kurso
Lahat ng kurso ko
Ipapakita ng ulat na ito ang mga tugon ng mga kalahok sa lahat ng kurso na gumagamit ng pagsusulit na ito at kung saan ikaw ang guro. Paa magamit sa maraming kurso, ang pagsusulit ay dapat nasa direktoryo ng file ng Site. Ang opsiyong ito ay magagamit lamang ng mga administrador ng site.
Pagpili ng mga user
Lahat ng kalahok
Ipapakita ng ulat ang mga tugon ng mag-aaral, guro, tagalikha ng kurso at administrador. Ang mga tugon ng "bisitang" user, kung mayroon man, ay isasama rin.
Mga mag-aaral
Ipapakita ng ulat ang mga tugon ng mag-aaral na kumukuha ng kursong ito. Hindi isasama ang mga tugon ng guro sa ulat.
Iisang mag-aaral
Ipapakita ng ulat ang mga tugon ng iisang mag-aaral na pinili mula sa hinahatak-pababa na listahan ng mga pangalan ng estudyante.
Pagpili ng mga marka
Lahat ng pagkuha
Isasama sa ulat ang mga tugon mula sa lahat ng pagkuha ng bawat user
Pinakamahusay na pagkuha
Isasama lamang sa ulat ang mga tugon mula sa pagkuha na may pinakamataas na iskor ang bawat user
Unang pagkuha
Isasama lamang sa ulat ang mga tugon mula sa unang pagkuha ng bawat user
Huling pagkuha
Isasama lamang sa ulat ang mga tugon mula sa pinakabagong pagkuha ng bawat user
