Palīdzība HTML rakstīšanā

Rakstot HTML Moodle vidē, jūs varat izmantot gandrīz visus HTML tagus, lai radītu vēlamo rezultātu.

Ievērojiet, ka skriptu rakstīšana (piem. Javascript vai VB Script) ir aizliegta un skripti tiks automātiski izņemti.

Parasti jūsu rakstītais kods lapā tiks izvietots tabulas šūnā šadi:

  • nav vajadzības izmantot <HEAD> vai <BODY> tagus
  • esiet uzmanīgi ar nesapārota </TABLE> taga lietošanu, kas var sabojāt visu lapas attēlošanu.

Smaidiņi (emocijzīmes) tiks pārveidoti to grafiskajos ekvivalentos un atklāti URL tiks pārveidoti par saitēm.
