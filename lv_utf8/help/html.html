Palīdzība HTML rakstīšanā

Rakstot HTML lapā, Jūs varat izmantot visus HTML-tagus, lai radītu Jums vajadzīgos efektus.

Parasti Jūsu kods tiks izvietots tabulas šūnā, šādā veidā:

  • nevajag izmantot <HEAD> vai <BODY> tagus
  • esiet uzmanīgs ar </TABLE> taga lietošanu, kas var sabojāt visu lapas struktūru.
