Шрыфт
-----

В гэтай дырэкторыі знаходзіцца шрыфт, які будзе выкарыстоўвацца пры ўводзе графікаў.

Шрыфт, які выкарыстоўваецца павінен мець назву  - default.ttf

Калі ў гэтай папцы шрыфт адсутнічае, сістэма выкарыстоўвае па-змаўчанні  
/lang/en/fonts/default.ttf.

Мульцібайтавыя радкі патрабуюць перакадыроўкі. За гэта мусіць адказваць 
файл з назвай lang_decode.php, які ўтрымлівае функцыю lang_decode().
