Параметр плаўнасці размеркавання работ

Гэты параметр акрэслівае, ці ўсе студэнты будуць мець аднолькавую колькасць работ для ацэнкі. Тэрмін "аднолькавая колькасць работ" азначае, што кожны студэнт атрымае аднолькавую колькасць работ для ацэньвання. Калі Параметр Плаўнасці Работ ўстаноўлены на НУЛЬ, усе дасланыя работы будуць ацэненыя аднолькавую колькасць разоў. Калі ж гэты параметр устаноўлены на 1, то некаторыя работы могуць быць ацэненыя на адзін раз больш (адпаведна іншыя на адзін раз менш). Падобна, што калі гэты ўзровень будзе ўстаноўлены на 2, то будзе дазволена яшчэ большая дыспрапорцыя.

Інтуітыўна гэты параметр устанаўліваецца на нуль. Аднак жа, некаторыя студэнты не будуць мець поўнай колькасці работ для ацэнкі аж да таго часу пакуль усе здадуць свае работы. Калі гэты параметр будзе ўстаноўлены на 1, то большасць студэнтаў атрымае неабходную колькасць работ для ацэньвання і не будзе вымушана чакаць работ дасланых са спазненнем. Праблема чакання на спозненыя работы будзе яшчэ менш адчувальнай, калі параметр будзе ўстаноўлены на 2. Такім чынам, калі параметр ацэнкі работ ўдзельнікаў устаноўлены на 5, ацэньванне будзе больш плаўным, таму што некаторыя работы будуць ацэнены 4 разы, некаторыя 5 разоў, а іншыя 6 разоў. Устаноўка гэтага параметра на іншую вартасць, чым адзінка, робіць працэс ацэнкі больш эластычным.
