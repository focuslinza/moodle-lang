Escribindo HTML

Síntase libre de colocar todas as etiquetas (TAGS) de HTML que desexe para obter o efecto requirido.

O seu código colocarase dentro dunha cela de táboa, polo que:

  • Non é preciso empregar <HEAD> ou <BODY>
  • Sexa coidadoso/a en pechar as súas táboas (se é o caso) con </TABLE>, xa que a súa ausencia pode desbaratar todo o deseño.
