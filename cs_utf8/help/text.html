URL

Jakékoli slovo začínající znaky www. nebo http:// bude automaticky převedeno na odkaz, na který je možné kliknout.

Příklad: www.yahoo.com nebo http://curtin.edu

Značky HTML

Svůj text můžete zpřehlednit použitím omezené sady značek HTML:

Značky HTML Výsledek
<b> tučný text </b> tučný text
<i> text v kurzívě </i> text v kurzívě
<u> podtržený text </u> podtržený text
<font color="green"> příklad </font> příklad
