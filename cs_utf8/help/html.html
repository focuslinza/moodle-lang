Nápověda k psaní HTML

Píšete-li HTML v Moodle, můžete používat jakékoliv HTML značky (tagy) a vytvořit tak požadovaný výsledek.

Váš HTML kód bude normálně zobrazen na stránce uvnitř buňky tabulky. Z těchto důvodů:

  • nepoužívejte značky jako <HEAD> nebo <BODY>
  • pozor na neodpovídající si značky <TABLE>, které mohou rozbít HTML strukturu stránky.
