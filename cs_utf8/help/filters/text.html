Textový filtr

Tento filtr vybírá údaje na základě textu vloženého uživatelem. Nabízí následující volby:

  • obsahuje – vyberou se pouze údaje obsahující zadaný text (není-li zadán žádný text, je filtr neaktivní)
  • neobsahuje – vyberou se pouze údaje neobsahující zadaný text (není-li zadán žádný text, je filtr neaktivní)
  • je rovno – vyberou se pouze údaje přesně odpovídající zadanému textu (není-li zadán žádný text, je filtr neaktivní)
  • začíná na – vyberou se pouze údaje začínající zadaným textem (není-li zadán žádný text, je filtr neaktivní)
  • končí – vyberou se pouze údaje končící zadaným textem (není-li zadán žádný text, je filtr neaktivní)
  • je prázdné – vyberou se pouze údaje s nulovým obsahem (zadaný text se nebere v úvahu)
