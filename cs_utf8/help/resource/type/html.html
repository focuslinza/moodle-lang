Webová stránka

Tento typ studijního materiálu umožňuje přímo v Moodlu snadno vytvořit kompletní webovou stránku pomocí vestavěného HTML editoru i bez znalosti jazyka HTML.

Stránka je uložena v databázi, nikoliv jako samostatný soubor, a její možnosti jsou omezeny pouze vlastnostmi jazyka HTML, lze v ní použít také Javascript atd.
