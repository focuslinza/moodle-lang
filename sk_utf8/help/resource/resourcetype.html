Typy zdrojov

Ako zdroje môžete použiť všetko čo si viete predstaviť. Tieto sú rôzneho typu s tým, že ich delenie je podľa dosažiteľnosti obsahu:

Jednoduchý text - najjednoduchší typ. Môžete len písať (editovať) v rámci nasledovnej strany. Niektoré záležitosti, ktoré budete písať, budú automaticky formátované. Napr. príspevky do fóra. Pozri Pomoc pri písaní textu:

HTML text - pre pokročilejších študentov. Nie je vykonávané automatické formátovanie, musíte to urobiť sami s použitím HTML kódov.

Referencie - školské referencie do žurnálu alebo knihy.

Načítaný súbor - zobrazí ľubovoľný súbor, ktorý ste načítali do kurzu. Pozri sekciu "Súubory".

Web linky - A URL niekde na internet. Ak niekto klikne na túto linku (reprezentujúcu zdroj), tento bude automaticky tam presmerovaný a adresa sa zobrazí na vašom prehliadači.

Web stránka - A URL niekde na internet. Podobne ako v predchádzajúcom prípade, toto zobrazí web stránku. rozdiel je však v tom, že stránka sa zobrazí v okne kurzu, tak ako by táto bola jeho súčasťou.

Program - externý program vyžadujúci presun informácií o súčasnom používateľovi a kurze Moodle. Týmto je umožnená spolupráca externých web aplikácií s Moodle.
