Importovanie výstupov

Môžete importovať výstupy vo formáte súborov .csv. Formát textu má byť nasledovný:

Názov poľa Opis Požadované Formát
outcome_name Plný názov výstupu ÁnoString
outcome_shortnameKrátky názov výstupu Áno String
outcome_descriptionOpis výstupu Nie String
scale_nameNázov použitého rozsahu Áno String
scale_itemsČiarkami oddelený zoznam predmetov rozsahu Áno String
scale_descriptionOpis rozsahu Nie String

Tu je príklad:

outcome_name;outcome_shortname;outcome_description;scale_name;scale_items;scale_description
Účasť; Účasť; Rozsah účasti; "Malá alebo žiadna účasť, Dostatočná účasť, Plná účasť"
