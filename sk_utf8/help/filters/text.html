Textový filter

Tento filter umožňuje filtrovať informácie založené na jednoduchom textovom formulári. Má nasledovné voľby:

  • obsahuje - dané pole obsahuje zadaný text (ak nie je nič zadané, filter nie je aktívny)
  • neobsahuje - dané pole neobsahuje zadaný text (ak nie je nič zadané, filter nie je aktívny)
  • je rovnaký ako - dané pole obsahuje iba zadaný text (ak nie je nič zadané, filter nie je aktívny)
  • začína s - dané pole začína na zadaný text (ak nie je nič zadané, filter nie je aktívny)
  • končí s - dané pole končí na zadaný text (ak nie je nič zadané, filter nie je aktívny)
  • je prázdny - táto položka neumožňuje zadať žiaden text. Ak je vybraná, tak sú označené iba záznamy, ktoré v danom poli nič neobsahujú
