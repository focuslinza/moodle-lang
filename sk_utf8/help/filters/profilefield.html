Filter pre profil

Tento filter umožňuje vybrať používateľov na základe údajov z ich profilu. Môže byť aplikovaný na jediné pole profilu alebo na všetky polia. Má nasledovné možnosti:

  • obsahuje - vyberá používateľov, u ktorých dané pole obsahuje zadaný text (ak nie je nič zadané, filter nie je aktívny)
  • neobsahuje - vyberá používateľov, u ktorých dané pole neobsahuje zadaný text (ak nie je nič zadané, filter nie je aktívny)
  • je rovnaký ako - vyberá používateľov, u ktorých dané pole obsahuje iba zadaný text (ak nie je nič zadané, filter nie je aktívny)
  • začína s - vyberá používateľov, u ktorých dané pole začína na zadaný text (ak nie je nič zadané, filter nie je aktívny)
  • končí s - vyberá používateľov, u ktorých dané pole končí na zadaný text (ak nie je nič zadané, filter nie je aktívny)
  • je prázdny - vyberá používateľov, u ktorých dané pole existuje, ale je prázdne (zadaný text je ignorovaný)
  • nie je definovaný - vyberá používateľov, u ktorých dané pole neexistuje (zadaný text je ignorovaný)
  • je definovaný - vyberá používateľov, u ktorých dané pole existuje (zadaný text je ignorovaný)
