Klávesové skratky HTML editora

HTML editor pozná nasledovné klávesové skratky:

Ctrl-O Zmena fontu
Ctrl-P Zmena veľkosti fontu
Ctrl-H Zmena formátu textu (nadpisy a pod.)
Ctrl-1 až Ctrl-6 Zmena úrovne nadpisu
Ctrl-= Zmena textového jazyka pre obrazovkové čítačky. Alebo, nastavenie jazyka pre Viacjazyčný filter (tieto sa zjavia po vybraní voľby 'Multi' v rozbaľovacom menu).
Ctrl-B Tučné písmo
Ctrl-I Šikmé písmo
Ctrl-U Podčiarknuté písmo
Ctrl-S Prečiarknutý text
Ctrl-, Dolný index
Ctrl-. Horný index
Ctrl-0 Vyčistiť HTML z Wordu
Ctrl-Z Späť
Ctrl-Y Znova
Ctrl-L Ľavé zarovnanie
Ctrl-E Centrovanie
Ctrl-J Zarovnanie do bloku
Ctrl-/ Text zľava doprava
Ctrl-| Text sprava doľava
Ctrl-; Zmenšenie odsadenia
Ctrl-' Zväčšenie odsadenia
Ctrl-G Zmena farby textu
Ctrl-K Zmena farby pozadia
Ctrl-F Hľadať a nahradiť
Ctrl-` Prepnutie do zdrojového kódu HTML
Ctrl-M Prepnutie celoobrazovkového režimu
Ctrl-Alt-O Vloženie číslovaného zoznamu
Ctrl-Alt-U Vloženie nečíslovaného zoznamu
Ctrl-Alt-R Vloženie horizontálnej čiary
Ctrl-Alt-A Vytvorenie kotvy
Ctrl-Alt-L Vytvorenie odkazu
Ctrl-Alt-D Zrušenie odkazu
Ctrl-Alt-N Vypnutie automatického odkazovania
Ctrl-Alt-I Vloženie obrázku
Ctrl-Alt-T Vloženie tabuľky
Ctrl-Alt-S Vloženie emotikonu
Ctrl-Alt-C Vloženie špeciálnych znakov

Súčasné obmedzenia:

  • Nie je možné použiť skratku Ctrl-` na prepnutie medzi HTML zdrojovým kódom a WYSIWYG zobrazením.
  • Klávesové skratky pre prácu s tabuľkou nie sú (sú dostupné iba v celoobrazovkovom režime).
