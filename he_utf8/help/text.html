עזרה בנוגע לכתיבת טקסט

כתיבת טקסט במוודל עובדת פחות או יותר באופן שבו אתם מצפים שהיא תעבוד, אלא שבנוסף יש לכם את היכולת להוסיף "סמיילים", "כתובות URL" וכמה תגי HTML לתוך הטקסט שלכם.

סמיילים (פרצופונים)

    כדי לשבץ את הסמלים הקטנים הללו בתוך הטקסט שלכם עליכם פשוט להקליד את הקוד המתאים. קודים אלה, אם תטו את ראשכם שמאלה ואז תסתכלו עליהם, תראו שהם כשלעצמם דומים מעט לציורים קטנים.

כתובות URL

    כל "מילה" שמתחילה ב-www. או ב-http:// תיהפך אוטומטית לקישור שניתן ללחוץ עליו.

    לדוגמא: www.yahoo.com או: http://curtin.edu

תגי HTML

    אתם יכולים להשתמש בתת מערכת מוגבלת של תגי HTML כדי להוסיף דגשים לתוך הטקסטים שלכם.

    תגי HTMLיוצרים
    <b> bold </b> טקסט מודגש
    <i> italic </i> טקסט בהטייה
    <u> underline </u> טקסט בעל קו תחתון
    <font color="green"> example </font> לדוגמא
    <ul> <li>one</li> <li>two</li> </ul>
