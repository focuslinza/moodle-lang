הגבלת זמן

לפי ברירת המחדל לבחנים אין הגבלת זמן, מה שנותן לסטודנטים את כל הזמן שהם צריכים כדי להשלים את הבוחן.

אם אתם מפרטים הגבלת זמן, אז מספר דברים נעשים כדי לנסות ולהבטיח שהבחנים נסתיימים ומושלמים באותה מסגרת זמן:

  • תמיכת Javascript בדפדפן הופכת לחובה, מה שמאפשר לשעון העצר לעבוד באופן נכון.
  • מוצג לסטודנטים חלון צף ובו שעון העצר שסופר את הזמן אחורנית.
  • כשמסתיימת הספירה של שעון העצר (כשנגמר הזמן), הבוחן מוגש באופן אוטומטי עם אי אלו שאלות שנענו בו עד אותו הזמן.
  • אם הסטודנט מצליח לרמות ולהישאר במסגרת הבוחן למעלה מ-60 שניות מעל הזמן המוקצב, הבוחן מקבל את הציון 0 באופן אוטומטי.
