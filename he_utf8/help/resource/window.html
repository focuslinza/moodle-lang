הגדרות חלון

את ההגדרות הללו ניתן להסתיר או להציג באמצעות לחיצה על הכפתור בעל הכותרת "הסתר הגדרות" או "הצג הגדרות".

לרוב, המשאבים מוצגים בתוך חלון ממוסגר רגיל כשכלי בקרת הניווט של מוודל מוצגים במסגרת העליונה. אם תבחרו בכך, אתם יכולים להסיר את המסגרת ושהמשאב ימלא את החלון במלואו בדומה לעמוד רשת רגיל.

אך אם ברצונכם שהמשאב שלכם יופיע בחלון 'קופץ' חדש, עקבו אחר ההראות הללו:

  1. בקטע החלון: ביחרו ב-'חלון חדש'.
  2. בטלו את הבחירה באפשרויות שאינכם רוצים להשתמש בהם.
    אפשר לשנות את גודל החלון?

    אם תשאירו את הבחירה בפריט זה, למשתמשים יתאפשר לשנות את גודל החלון שאתם הגדרתם. כמעט תמיד רצוי להשאיר את הבחירה בפריט זה. אך לפעמים קורה שהמשאב עצמו הוא בגודל קבוע, ולכן זה הגיוני לנעול את גודל החלון.

    אפשר לגלול את מסך החלון?

    פריט זה מאפשר למשתמשים לגלול דרך המסך כדי לראות את המשאב כולו. אם תבטלו את הבחירה בפריט זה, המשתמשים יראו רק את מה שמתאים לגודל החלון המוגדר ולא יוכלו לגלול את המסך כדי לראות את שאר המשאב.

    הצג את קישורי התיקיות?

    זה יאפשר למשתמשים לראות בחלון זה את הקישורים לתיקיות. הגדרה זו משתנה מדפדפן לדפדפן: יכול להיות שהיא ידועה בתור סרגל הקישורים (ב-IE), סרגל המועדפים או סרגל הסימניות (ב-Mozilla/Firefox).

    הצג את סרגל המיקום?

    האם אתם רוצים שהמשתמשים שלכם יראו את כתובת ה-URL (כתובת הרשת) של המשאב? אם כן, השאירו את הבחירה באפשרות זו. אחרת, מיקום המשאב יוסתר.

    הצג את סרגל התפריטים?

    כדי לאפשר למשתמשים שלכם גישה לסרגל התפריטים בחלון זה, השאירו את הבחירה בפריט זה. לרוב, אפשרות זו היא שימושית אבל חלק מהתצוגות הגרפיות נראות טוב יותר ללא סרגל תפריטים.

    הצג את סרגל הכלים?

    הצגת סרגל הכלים תקל על המשתמשים שלכם לנווט שכן הוא מספק כפתורי קדימה\אחורה, הביתה ורענן.

    הצג את סרגל המצב?

    הצגת סרגל המצב מאפשרת למשתמשים לעקוב אחר נתוני הדפדפן, לדוגמא: כמה מהמשאב כבר עלה או מידע בטחוני.

  3. הגדירו את גודל החלון אותו אתם רוצים לפתוח.

    רוב האנשים עדיין ישתמשו ברזולוציית מסך ממוצעת של 800X600. ברוב המקרים רצוי שבאופן ראשוני תגדירו את גודל החלון שלכם מעט קטן מזה, לדוגמא: 600 לגובה ו-480 לרוחב.

    אם תשאירו את השדות הללו ריקים, אז החלון הקופץ יהיה בדיוק באותו הגודל כמו החלון עליו המשתמש לחץ כדי לקבל את החלון הקופץ.
