עיצוב טקסט

כשאתם כותבים טקסט במוודל ישנם מספר דרכי עיצוב (פורמטים) מהם אתם יכולים לבחור כדי לייצר את הטקסט שלכם, לפי המומחיות שלכם וסוג הדפדפן שבו אתם משתמשים.

לרוב אתם יכולים להשאיר את ההגדרה לערך ברירת המחדל שלה והדברים אמורים לעבוד בדרך המצופה. .

1.פורמט אוטומטי של מוודל

    פורמט זה הוא במיטבו כאשר אתם משתמשים בצורות רשת רגילות עבור הרשומה (ולא בעורך הטקסט העשיר של HTML). פשוט הקלידו את הטקסט כרגיל, כאילו אתם שולחים דוא"ל.

    כשתשמרו את הטקסט שלכם, מוודל, באופן אוטומטי, יעשה מספר דברים כדי לעצב את הטקסט עבורכם.

    לדוגמא, כתובות URL כמו:

    http://yahoo.com או אפילו: www.yahoo.com יהפכו לקישורים.

    ההפסקות שבין השורות שלכם יישמרו כפי שהן ושורות ריקות יתחילו פסקאות חדשות.

    תווי סמיילי כמו (-: יהפכו אוטומטית למקבילות הגרפיות שלהם.

    אתם יכולים אפילו לשבץ קוד HTML והוא יחזיק.

    מידע נוסף על הפורמט האוטומטי של מוודל

2. פורמט HTML

    פורמט זה מניח שהטקסט הוא HTML טהור. אם אתם משתמשים בעורך ה-HTML כדי לערוך את הטקסט, אז פורמט זה משמש כברירת המחדל - כל הפקודות בסרגל הכלים מייצרות HTML עבורכם.

    גם במידה ואינכם משתמשים בעורך הטקסט העשיר של HTML, אתם יכולים להשתמש בקוד HTML בתוך הטקסט שלכם והתוצאה אמורה לצאת בדיוק באופן שבו התכוונתם.

    בניגוד לפורמט האוטומטי של מוודל, לא מתרחשת שום פעולת עיצוב אוטומטית.

3. פורמט טקסט פשוט

    פורמט זה שימושי במידה ואתם רוצים להוסיף הרבה קוד או HTML שאתם מעוניינים שיוצג בדיוק באופן שבו כתבתם אותו.

    הוא עדיין מתרגם רווחים ושורות חדשות, אבל חוץ מזה הוא לא משנה דבר בטקסט שלכם.

4. פורמט טקסט Markdown

    פורמט Markdown מנסה להקל עליכם להקליד עמודי XHTML המעוצבים היטב באמצעות לא הרבה יותר מהדרך שבה תכתבו דוא"ל.

    הוא מאוד טוב בכל הנוגע לכתיבת עמודי טקסט נקיים בעלי כמה כותרות וכמה רשימות אבל ללא קישורים רבים או תמונות.
