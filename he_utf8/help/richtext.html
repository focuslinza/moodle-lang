עורך הטקסט העשיר של HTML מספק ממשק מעבד תמלילים שמשובץ לתוך עמוד הרשת שלכם ומאפשר לכם לערוך טקסט באופן אינטואיטיבי, ובכך לייצר קוד HTML רגיל.

בנוסף לעיצוב הטקסט, עורך זה מספק מספר תכונות נוספות שיכולות לשמש אתכם.

הדבק טקסט שמקורו ביישום אחר

    באפשרותכם לגזור ולהדביק טקסט עשיר מיישומים אחרים ב-Windows (כמו Microsoft Word) ישירות לתוך עורך זה, והעיצוב שלכם יישמר כפי שהוא. פשוט השתמשו בתפריטי הגזירה וההדבקה הרגילים שנמצאים בדפדפן שלכם (או ב- Control-C ו- Control-V).

הכנסת תמונות

    אם יש בידכם תמונות שכבר פורסמו באתר אינטרנט וניתן לגשת אליהן באמצעות כתובת URL, אתם יכולים לכלול את התמונות הללו בתוך הטקסטים שלכם באמצעות הכפתור: "הכנס תמונה".

הכנסת טבלות

    כדי להוסיף מערך לטקסטים שלכם, אתם יכולים להשתמש בכפתור: "הכנס טבלה" שנמצא בסרגל הכלים שלכם.

הכנסת קישורים /

    כדי ליצור קישור חדש, תחילה עליכם להקליד את הטקסט שברצונכם ישמש כקישור. לאחר מכן ביחרו אותו וליחצו על כפתור ה"צור קישור" בסרגל הכלים. הקלידו את כתובת ה-URL אליה אתם רוצים ליצור קישור וסיימתם!

הכנסת סמיילים (פרצופונים)

    כדי לשבץ את הסמלים הקטנים הללו לתוך הטקסט שלכם, ליחצו על סמל הסמיילי שנמצא בסרגל הכלים. על המסך יקפוץ חלון של דיאלוג שיאפשר לכם לבחור מבין אייקוני הסמיילים הבאים. (לחילופין, אתם יכולים פשוט להקליד את הקוד המתאים ישר לתוך הטקסט ומאוחר יותר, כשהטקסט שלכם יוצג, הוא יומר לאייקון המבוקש.)
