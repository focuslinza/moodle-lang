עזרה בכתיבת קוד html

כאשר כותבים HTML במערכת Moodle, אתה יכול לכתוב באופן חופשי כמעט כל תג שתחפוץ בו של HTML בכדי ליצור את המיטב שתרצה

דע די כתיבת סקריפט שהוא Java Script או VB Script אינו אפשרי, ויוסר באופן אוטומטי.

הקוד שלך יודפס בצורה נורמלית בעמוד במסגרת תאים בטבלה, כך ש:

  • אין צורך להשתמש בכל התגים מהסוג <HEAD> או <BODY>
  • היזהר להשתמש בתגים שאין שני להם כמו </TABLE> היכולים לקלקל את ההצגה של הדברים

פרצופונים (סמיילים) יומרו לשקולה הגרפית שלהם, וכתובות URL יומרו לקישורים פעילים.
