ID zenbakiak

ID zenbakia ezarrita kalifikazioak kalkulatzeko helburu-elementua identifikatzeko aukera izango duzu. Helburu-elementua ez badago inongo kalifikazio-kalkulutan sartuta edo ezin bada kalkulua aplikatu (adib. eskala kualitatiboak), ID zenbakiaren eremua zuri utzi ahal da. Jarduera batean oinarritutako kalifikazio-elementu baten ID zenbakia jarduera orriaren ezarpenetan adierazten da.
