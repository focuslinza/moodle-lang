ID zenbakia

ID zenbaki bat ezarrita jarduera identifikatzeko modua izango duzu kalifikazioak kalkulatzeko orduan. Jarduera ez badago inongo kalifikazio-kalkulutan sartuta, ID zenbakiaren eremua hutsik utzi ahal da.

ID zenbakia kalifikazio-liburuko kalifikazioen kalkulua editatzeko orrian ere ezar daiteke, nahiz eta jarduera eguneratzeko orritik bakarrik editatu ahal den.
