"Course Test Manager" software-paketetik fitxategiak inportatu

Course Test Manager (ikastaroetako galdeketen administratzailea) Course Technology delakoaren "End of Lifetime" software-pakete bat da. Gaur egun dagoeneko ez du Course Technologyk garatzen ; haren ordez beste galdeketa-administratzaile bat jarri da : ExamView. Dena den, sekula ez dute migraziorik egin, formatu hori CTMtik beste galdeketa-formatu ezagun batera esportatzeko. Horregatik idatzi dugu modulu hau.

CTMko galderak inportatzeko, aurrena ordenagailuan instalatuta eduki behar duzu (ordenagailuak Windows eduki behar du) eta ordenagailu horretan MS Accesseko datu-artxiboetan sartzeko aukera izan behar duzu. CTM datuetara sartzeko bi datu-fitxategi behar dira:

  • System.mda - Fitxategi hori CTMren instalazioaren direktorio nagusian dago; Access lengoaian "Sistemaren datu-basea" esaten zaio.
  • ctm.mdb - Datu-base honetan galdeketa-bankuko galderak daude. CTM instalatutako ikastaro bakoitzak bere datu-basea dauka, maila baxuagoko direktorioetan, CTMren instalazio-direktorioaren azpian.

Inportazio-prozesua desberdina da Linuxen eta Windowsen. Plataforma dena delakoa ere, Moodle dagoen sarean Windows sistema beharko duzu (Windows 2000 edo Windows XP), Accesseko datu-baseak zuzenean inportatzeko.

Prozesua errazagoa da Moodle Windowsen badago. CTM inportazioa erabiltzeko hau besterik ez duzu egin behar:

  1. System.mda datu-basea Moodle programara kargatu, artxiboen administratzailea medio. Ez dio axola sistem.mda ze ikastarotara kargatu, Moodle programak aurkituko du.
  2. Galdeketa sortu eta "fitxategiaren galderak inportatu"prozesuan sartu. Aukeratu duzun cmt.mdb artxiboa inportatzeko artxibo gisa kargatu. Hori behar bezala egin baduzu, beste pantaila bat agertuko zaizu, datu-basetik inportatzeko galderen azpikategoria aukeratzeko. Izan ere, CTM ebaluazio-bankuek kategoriadun galdera asko izaten dituzte maiz, liburuko kapituluetan edo ikastaroko ataletan oinarrituak.

Moodle Linuxen erabiltzen bada, ODBC Socket Server software-osagaia behar da. Linuxen CTM datu-baseak inportatzeko, aurrena sare-programa txiki hori instalatu behar duzu CTM datu-baseak dauden ordenagailuan (ordenagailu horrek Windows eduki behar du). Ez jarraitu ODBC Socket Server delako horren instalazioan aipatzen diren jarraibideak. Lan honetarako behar baino konplikatuegiak dira. Hobe duzu pauso hauek jarraitzea:

  1. Windows duen ordenagailura joan eta deskonprimitu gune horretatik jaitsitako instalazio-artxiboa.
  2. ODBCSocketServer.exe artxibo bitarra kokapen iraunkor batean kopiatu; hemen, adibidez: c:\windows\ o c:\WINNT\ (zure programa-artxiboen karpeta ere izan daiteke).
  3. ODBC Socket Server zerbitzu gisa instalatu sisteman, agindu honekin:
    <aurreko pausoan artxiboa kopiatutako direktorio osoa>ODBCSocketServer.exe /Service
    Agindua elkarrizketa-kutxan idatz dezakezu: Hasiera->Exekutatu... , edo aginduen lerrotik.
  4. Zerbitzuen administratzailea ireki, hemen: Hasiera->Konfigurazioa->Kontrol-panela->Administrazio-tresnak->Zerbitzuak, edo Ordenagailuan saguaren eskuineko botoiari klik egin, Administratu aukeratu eta, gero, eskuineko panelean, Zerbitzuak eta aplikazioak->Zerbitzuak. Zerbitzuen administratzailera sar zaitezke, baita ere, Hasiera->Exekutatu elkarrizketa-koadroan services.msc idatzita.
  5. Zerbitzuen administratzailean egin klik saguaren eskuineko botoiarekin ODBC Socket Server itemean eta aukeratu Hasi. Hori egindakoan egoera "Hasita" izango da. Inportazio-prozesua gelditu edota erabat desaktibatu dezakezu, prozesua amaitutakoan.
  6. Ziurtatu system.mda eta ctm.mdb artxiboak ordenagailuan daudela eta MS Accessen ODBC kontrolatzailea presente dagoela (normalean Windowsen instalazio estandarrean instalatzen da). Instalatuta dagoela ziurta dezakezu "Kontrolatzaileak" atalean "Kudeaketa-tresnak" sakatu eta "datu-iturriak" itema irekita. CTM non instalatuta dagoen zehaztu behar duzu eta system.mda eta ctm.mdb datu-baseak (edo inportatu nahi dituzun datu-baseak) kokatu. Datu-baseen kokapen zehatza eta Windows zerbitzariaren Hostaren IP helbidea edo izena gogoan hartu behar dituzu. Informazio-prozesuan informazio hori beharko duzu.
  7. Orain inporta ditzakezu ebaluazio-bankuak Moodle programara. Galdeketa sortu eta "fitxategiaren galderak inportatu" prozesuan sartu.
