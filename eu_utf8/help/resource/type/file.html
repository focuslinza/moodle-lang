Fitxategiak eta web orriak

    Baliabide-mota honek aukera ematen du edozein web-orri edo web publiko bateko fitxategi baterako esteka ezartzeko. Zeure ordenagailutik ikastaroko fitxategi eremura zuk igotako edozein fitxategi edo web-orrirako esteka ere ezar dezake.

    Ohiko web-orriak dauden daudenean erakusten dira baina multimedia fitxategiak era inteligentean erabiltzen dira eta web-orri baten barruan sar daitezke. Adibidez, MP3 fitxategiak berarekin dakarten erreproduktorea erabilita erakutsi daitezke, baita bideo-fitxategiak ere, flash animazioak, e.a.

    Edukia erakusteko aukera asko dago sortzen diren leihoetan, markodun leihoetan, e.a.

    Zehaztuz, zure baliabidea parametroak onartzen dituen web-aplikazioa edo bestelako edukia-mota bada, aukeran duzu baliabidera ondorengo informazioa bidaltzea: erabiltzaile-izena, posta-helbidea, zein ikastarotan dagoen matrikulatuta, e.a.
