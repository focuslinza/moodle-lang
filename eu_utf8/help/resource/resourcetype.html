Baliabide-motak

Baliabidea ikasleek kontsultatze aldera jartzen den edozein eduki da. Hauek dira baliabide-motak:

Testu laua, errazena. Orrialdea formularioan idatzi (editatu) besterik ez duzu. Idazten duzun hori automatikoki formateatu egiten da, foroetako mezuak bezalaxe. Autoformatuko testuei buruzko informazio gehiago:

HTML testua, erabiltzaile aurreratuenentzat. Testua ez da automatikoki formateatzen: eskuz formateatu behar da, HTML kodeak erabiliz. HTML testuei buruzko informazio gehiago.

Erreferentzia, aldizkari edo liburu bati buruzko erreferentzia.

Kargatutako artxiboa, ikastaroan kargatutako edozein artxibo erakusteko. Begiratu “Artxiboak” atala.

Direktorioa, direktorio osoa agertzen du (dagozkien azpidirektorioekin), ikastaroko artxiboen atalean. Ikasleek artxibo guztiak ikusi eta kontsulta ditzakete.

Web esteka, web gunearen leku baterako hiperesteka (link). Ikasleak material horretan klik egiten duenean ikastarotik kanpora bidaliko da (espezifikatutako orrialdeak nabigatzailearen leihoa beteko du).

Web orrialdea, web guneko leku baterako URLa. Aurreko baliabidean bezala, web orrialde bat agertuko da. Orrialdea, ordea, marko baten barruan agertuko da, ikastaro barruan balego bezala.

Programa – erabiltzailearen eta ikastaroaren informazioa behar duen kanpoko programa. Baliabide hori medio, internetetik kanpoko aplikazioek Moodle sistemarekin batera lan egin dezakete. Programei buruzko informazio gehiago.

Wiki testua, testua HTML erabili gabe formateatzeko. Formatua karaktere arruntekin lortzen da, adibidez: *hau letra lodia da* /hau letra etzana da/). Wikiri buruzko informazio gehiago.
