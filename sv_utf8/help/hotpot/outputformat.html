Format för output

Du kan ange olika format för visning av testet.

bästa
Testet kommer att visas i det bästa formatet med tanke på webbläsaren.
v6+
Testet kommer att visas i ett drag-och-släpp-format för webbläsare av typ v6+
v6
Testet kommer att visas i ett format för webbläsare av typ v6
v5
Testet kommer att visas i ett format för webbläsare av typ v5
Testet kommer att visas i ett format för webbläsare av typ v4
v3
Testet kommer att visas i ett format för webbläsare av typ v3
Flash
Testet kommer att visas i ett format för Flash
mobile
Testet kommer att visas i ett format som passar för att visa på små enheter som t.ex. en mobiltelefon.
