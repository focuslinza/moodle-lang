Databas-aktivitet

Databasmodulen gör det möjligt för (distans)läraren och/eller studenterna/eleverna/deltagarna/de lärande att bygga, visa och genomsöka ett arkiv av inlagda bidrag om vilket ämne som helst. Det går att välja vilket format eller vilken struktur som helst på dessa bidrag. Det kan vara bilder, filer, URLer, tal och text t.ex. Du kanske har viss vana vid liknande teknologier om du har arbetat med Microsoft Access eller Filemaker.
