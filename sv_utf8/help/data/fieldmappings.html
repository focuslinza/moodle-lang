Avbildningar av fält

Den här menyn gör det möjligt för Dig att bevara data från en befintlig databas. För att bevara data i ett fält så måste Du avbilda det och överföra det till ett annat fält där datan kommer att dyka upp. Vilket fält som helst kan även vara tomt, utan att det finns någon information i det. Alla gamla fält som inte avbildas och kopieras över till ett nytt kommer att försvinna tillsammans med alla data som det innehåller.

Du kan bara avbilda fält till fält av samma typ. Dessutom måste Du vara försiktig så att Du inte försöker att avbilda ett enda gammalt fält till flera nya fält.
