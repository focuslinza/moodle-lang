Κρυμμένα τμήματα μαθήματος

Αυτή η επιλογή σας επιτρέπει να αποφασίσετε πώς τα κρυμμένα τμήματα στο μάθημά σας θα εμφανίζονται στους σπουδαστές.
Εξ ορισμού, μια μικρή περιοχή επιδεικνύεται (συνήθως με γκρίζα γράμματα) για να δείξει που βρίσκεται το κρυμμένο εδάφιο, αν και δεν μπορούν ακόμα να δουν πραγματικά τις κρυμμένες δραστηριότητες και τα κείμενα. Αυτό είναι ιδιαίτερα χρήσιμο στην εβδομαδιαία διαμόρφωση, έτσι ώστε οι εβδομάδες χωρίς μάθημα να είναι σαφείς. Δίνει την δυνατότητα στους σπουδαστές να οργανώσουν τον χρόνο τους, γνωρίζοντας τις ημερομηνίες που θα υπάρξει κάτι καινούργιο για το μάθημα.
Εάν το επιθυμείτε, τα κρυμμένα τμήματα του μαθήματος μπορούν να μην είναι καθόλου ορατά στους σπουδαστές, έτσι ώστε αυτοί να μην γνωρίζουν ότι τμήματα της σειράς μαθημάτων είναι κρυμμένα.
