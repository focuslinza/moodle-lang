Βοήθεια στη συγγραφή κειμένου

Η σύνταξη κειμένου στο Moodle λειτουργεί σε μεγάλο βαθμό με τον τρόπο που θα περιμένατε αλλά σας δίνει επίσης τη δυνατότητα να συμπεριλάβετε "smilies", "διευθύνσεις URL" και μερικές HTML ετικέτες στο κείμενο σας.

Φατσούλες (emoticons)

Για να ενσωματώσετε αυτές τις μικρές εικόνες στο κείμενό σας, απλά πληκτρολογήστε τον σχετικό κώδικα. Οι κωδικοί αυτοί είναι οι ίδιοι μικρές εικόνες που μπορείτε να δείτε εάν στρίψετε το κεφάλι σας αριστερά κατά την εξέταση τους.

URLs

Οποιαδήποτε "λέξη" ξεκινάει με www. or http:// αυτόματα θα μετατραπεί σε σύνδεσμο στον οποίο μπορείτε να κάνετε κλικ.

Για παράδειγμα: www.yahoo.com or http://curtin.edu.

Ετικέτες HTML

Μπορείτε να χρησιμοποιήσετε ένα περιορισμένο υποσύνολο ετικετών HTML ώστε να δώσετε έμφαση στα κείμενά σας.

Ετικέτες HTMLΠαράγει
<b> bold </b> έντονο κείμενο
<i> italic </i> πλάγιο κείμενο
<u> underline </u> υπογραμμισμένο κείμενο
<font color="green"> example </font> παράδειγμα
<ul> <li>one</li> <li>two</li> </ul>
