Πακέτα περιεχομένου IMS

    Δίνεται η δυνατότητα προσθήκης πακέτων IMS (βλ. Προδιαγραφές πακέτων περιεχομένου IMS ) στο μάθημά σας.

    Απλώς φορτώστε το συμπιεσμένο πακέτο και αναπτύξτε το.

    Υπάρχουν διάφορες επιλογές για την προβολή υλικού σε 'υπερεμφανιζόμενο παράθυρο' (popup window), όπως με μενού πλοήγησης ή κουμπιά επιλογών κ.ά.

IMS Content Packages

    This resource type allows you add packages conforming to the IMS content packaging specifications to your course.

    Simply upload a zipped package and deploy.

    There are various options for displaying content in a popup window, with a navigation menu or buttons etc.
