Ρυθμίσεις παραθύρων

Αυτές οι ρυθμίσεις μπορούν να κρυφτούν ή να εμφανιστούν κάνοντας κλικ με το ποντίκι στο κουμπί ‘Αποκτυψη ρυθμίσεων’ ή στο κουμπί ‘Εμφάνιση ρυθμίσεων’ αντίστοιχα.

 

Συνήθως, οι επιλογές εμφανίζονται σε ένα κανονικό παράθυρο πλαισίου με τις επιλογές ελέγχου του Moodle στο άνω πλαίσιο. Εάν το επιλέξετε, μπορείτε να αφαιρέσετε αυτό το πλαίσιο και να έχετε το υλικό σας να γεμίζει το παράθυρο όπως σε κανονική ιστοσελίδα.

 

Εντούτοις, εάν επιθυμείτε να έχετε τις επιλογές σας να εμφανίζονται σε ένα νέο αναδυόμενο παράθυρο (pop up window) ακολουθήστε αυτά τα βήματα:

  1. Στο παράθυρο: τμήμα, επιλέξτε "νέο παράθυρο".
  2. Από-επιλέξτε τις επιλογές που δεν επιθυμείτε να χρησιμοποιήσετε.

Επιτρέψτε στο παράθυρο για να αλλάζει μέγεθος;

Η διατήρηση αυτού του στοιχείου επιλεγμένου θα επιτρέψει στους χρήστες να αλλάζουν το μέγεθος του παραθύρου που έχετε καθορίσει. Πρέπει σχεδόν πάντα να αφήνεται επιλεγμένο. Σε ορισμένες όμως περιπτώσεις το υλικό είναι σε ορισμένο μέγεθος και μπορεί να έχει νόημα να κλειδώσετε το μέγεθος παραθύρων.

Επιτρέψτε στο παράθυρο να ξετυλίγεται (scroll);

Αυτό το στοιχείο θα επιτρέψει στους χρήστες για να ξετυλίξει το παραθύρο για να δει όλο το υλικό σας. Εάν αυτό το στοιχείο δεν είναι επιλεγμένο, οι χρήστες θα δουν μόνο ό,τι χωρά στο καθορισμένο μέγεθος παραθύρων και δεν έχουν την άδεια για να ξετυλίξουν το παράθυρο για να δουν το υπόλοιπο του υλικού.

Εμφάνιση συνδέσμων καταλόγου;

Η επιλογή αυτή επιτρέπει στους χρήστες να βλέπουν τους συνδέσμους καταλόγου τους σε αυτό το παράθυρο. Η επιλογή ποικίλλει ανάλογα με το πρόγραμμα πλοήγησης: είναι ίσως η γραμμή Συνδέσμων (στον Internet Explorer), η γραμμή ‘Αγαπημένα’ ή η γραμμή Σελιδοδεικτών (στον Mozilla/Firefox).

Εμφάνιση γραμμής πλοήγησης;

Θέλετε οι χρήστες να βλέπουν το URL (ιστοδιεύθυνση) του υλικού σας; Σε αυτή την περίπτωση, αφήστε αυτήν την επιλογή ενεργοποιημένη. Διαφορετικά, η ιστοδιεύθυνση του υλικού σας σε θα εμφανίζεται.

Εμφάνιση γραμμής μενού επιλογών (menu bar);

Για να δώσετε τους χρήστες σας πρόσβαση στο μενού επιλογών σε αυτό το παράθυρο, αφήστε αυτό το στοιχείο ενεργοποιημένο. Γενικά πρόκειται για κάτι χρήσιμο αλλά μερικές γραφικές παρουσιάσεις φαίνονται καλύτερα χωρίς τη γραμμή μενού επιλογών.

Εμφάνιση γραμμής εργαλείων;

Η παρουσίαση της γραμμής εργαλείων θα διευκολύνει τους χρήστες σας δεδομένου ότι παρέχει επιλογές Πίσω/Μπρος, Αρχική σελίδα, και Ανανέωση σελίδας.

Εμφάνιση γραμμής κατάστασης;

Η παρουσίαση της γραμμής επιτρέπει στους χρήστες να ελέγχουν πληροφορίες του προγράμματος πλοήγησης, όπως την ποσότητα του υλικού που έχει φορτωθεί ή πληροφορίες ασφάλειας.

  1. Καθορίστε την ανάλυση του παραθύρου που επιθυμείτε να ανοίξετε.

Πολλοί χρησιμοποιούν ακόμα μια μέση ανάλυση οθόνης της τάξης των 800x600 στοιχείων. Στις περισσότερες περιπτώσεις είναι καλύτερα να ορίσετε αρχικά το παράθυρό σας λίγο μικρότερο από αυτό, όπως 600 στοιχεία ύψος και 480 στοιχεία πλάτος.

Εάν αφήσετε κενές τις επιλογές αυτές, το νέο αναδυόμενο παράθυρο θα είναι ακριβώς στο ίδιο μέγεθος με το παράθυρο στο οποίο κάνατε κλικ και από το οποίο προέκυψε το αναδυόμενο παράθυρο.
