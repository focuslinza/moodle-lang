Μια ειδική ομάδα συζητήσεων που ονομάζεται "Πίνακας Ανακοινώσεων" εμφανίζεται στην εβδομαδιαία και τη θεματική μορφή των μαθημάτων. Είναι ένα καλό μέρος για ανάρτηση σημειώσεων, προκειμένου να μπορούν να τις δουν όλοι οι φοιτητές. (Όλοι οι φοιτητές να είναι μέλη σε αυτή την ομάδα συζητήσεων για να λαμβάνουν όλες τις ανακοινώσεις με email).

Αυτή η ρύθμιση καθορίζει πόσα πρόσφατα αντικείμενα θα εμφανίζονται στην αρχική σελίδα του μαθήματος, στην περιοχή νέων στην κάτω δεξιά πλευρά.

Εαν ρυθμιστεί στο "0 νέα αντικείμενα" τότε η περιοχή νέων δεν εμφανίζεται καθόλου.
