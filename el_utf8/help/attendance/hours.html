Αριθμός ωρών σε μια διδακτική περίοδο

Εδώ ορίζεται ο αριθμός ωρών μιας διδακτικής περιόδου. Οι παρουσίες/απουσίες καταχωρούνται για κάθε διδακτική ώρα ξεχωριστά, ενώ δεν υπάρχει ξεχωριστός τρόπος ορισμού μέρους μιας διδακτικής ώρας.

Η επιλογή αυτή έχει περιορισμένη ανεκτικότητα όσον αφορά στην αυτόματη καταχώρηση παρουσιών/απουσιών.

Number of Hours in a Class Period

Sets how many hours this class period will have. Attendance is logged individually for each hour of class. There is no way to specify partial hours.

This option doesn't really have a lot of bearing on automatic attendance logging.
