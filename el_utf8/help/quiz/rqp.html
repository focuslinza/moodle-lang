Απομακρυσμένες Ερωτήσεις Remote Questions (RQP)

Το Πρωτόκολλο απομακρυσμένων ερωτήσεων μπορεί να χρησιμοποιηθεί για να περιλαμβάνει ερωτήσεις σε κουίζ με όλε την επεξεργασία να γίνεται σε απομακρυσμένο server. Ερωτήσεις μπορεί να περιλαμβάνονται σε οποιοδήποτε format για το οποίο ο server είναι διαθέσιμος χωρίς κάποια αλλαξή στο Moodle.
