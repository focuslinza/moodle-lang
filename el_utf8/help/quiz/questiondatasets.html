Μία σύνολο δεδομένων είναι μια συλλογή από δεδομένα που χρησιμοποιείται για να δημιουργήσει μια ερώτηση (όπως μια υπολογισμένη ερώτηση), όπου εισάγεται αντί μιας μεταβλητής μέσα στην ερώτηση.

Μπορείτε να δημιουργήσετε ένα «ιδιωτικό» σύνολο δεδομένων για μια συγκεκριμένη ερώτηση, ή ένα «επαναχρησιμοποιήσιμο» σύνολο δεδομένων που μπορούν να χρησιμοποιηθούν μέσα σε όλες τις ερωτήσεις μέσα σε μια κατηγορία.

Κατά τη δημιουργία μιας ερώτησης που χρησιμοποιεί τα σύνολα δεδομένων ,η οθόνη θα είναι χωρισμένη σε δύο .

Η πρώτη οθόνη επιτρέπει σε σας για να ορίσει ένα ιδιαίτερο σύνολο δεδομένων που αντικαθιστά κάθε μεταβλητή.

Η δεύτερη οθόνη σας επιτρέπει να προσθέσετε και να αφαιρέσετε αριθμούς στο σύνολο δεδομένων. Οι αριθμοί χρησιμοποιούνται για τις μεμονωμένες ερωτήσεις που εχουν επιλεχτεί από αυτό το σύνολο και παρουσιάζονται στους μαθητές.
