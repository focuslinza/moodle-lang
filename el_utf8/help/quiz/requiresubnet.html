Απαιτούνται διευθύνσεις δικτύου

Αυτό το πεδίο είναι προαιρετικό.

Μπορείτε να περιορίσετε την πρόσβαση για το κουίζ σε συγκεκριμένα υποδίκτυα ενός LAN ή του Internet ορίζοντας μια λίστα με διαχωρισμούς από κόμμα (,) από μερικούς ή πλήρεις αριθμούς διευθύνσεων IP.

Αυτό είναι πολύ χρησιμο για κουίζ εξέτασης, οπου επιθυμείτε να είστε σίγουροι πως συμμετέχουν μόνο άτομα από μια κεντρική αίθουσα και μόνο αυτά έχουν πρόσβαση στο κουίζ.

Για παράδειγμα: 192.168. , 231.54.211.0/20, 231.3.56.211, 231.3.56.10-20

Υπάρχουν 4 τύποι αριθμών που μπορείτε να χρησιμοποιείσετε (δε μπορείτε να χρησιμοποιείσετε domain names όπως example.com):

  1. Πλήρης IP διευθύνσεις, όπως 192.168.10.1 που θα ταιροιάζουν σε συγκεκριμένο υπολογιστή (ή proxy).
  2. Μερικές διευθύνσεις, όπως 192.168 που θα ταιροιάζουν σε οτιδήποτε ξεκινά με αυτούς τους αριθμούς.
  3. CIDR σημείωση, όπως 231.54.211.0/20 που σας επιτρέπει να ορίζετε πιο λεπτομερή υποδίκτυα.
  4. Μια κλίμακα από IP διευθύνσεις 231.3.56.10-20 Η κλίμακα αφορά ο τελεταίο νούμερο της διεύθυνσης, όπως όλες οι IP διευθύνσεις από 231.3.56.10 σε 231.3.56.20.

Τα κενά παραβλέπονταιSpaces are ignored.
