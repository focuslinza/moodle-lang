Εξαγωγή ερωτήσεων από μια Κατηγορία

Αυτή η λειτουργία επιτρέπει την εξαγωγή μιας κατηγορίας από ερωτήσεις σε ένα αρχείο κειμένου.

Σε αρκετές μορφοποίησης κειμένου κάποια πληροφορία χάνεται όταν εξάγονται ερωτήσεις. Αυτό συμβαίνει γιατί πολλές μορφοποιήσεις δεν υποστηρίζουν ολες τις δυνατότητες που υπάρχουν στις ερωτήσεις του Moodle. Δεν γίνεται πανομοιότυπη εισαγωγή και εξαγωγή ερωτήσεων. Επίσης, Κάποιοι τύποι ερωτήσεων δεν εξάγονται καθόλου. Είναι προτεινόμενος ο έλεγχος των εξαγόμενων δεδομένων πριν χρησιμοποιηθούν σε κάποιο περιβάλλον παράγωγης.

Οι μορφές που υποστηρίζονται είναι:

GIFT format

    GIFT είναι η πιο πλήρες μορφή εισαγωγής/εξαγωγής ερωτήσεων σε ένα αρχείο κειμένου. Έχει σχεδιαστεί ώστε να είναι εύκολη μέθοδο για τους δάσκαλους(δηλαδή μπορούν να γραφούν ερωτήσεις σε ένα αρχείο κειμένου). Υποστηρίζει Πολλαπλές-ερωτήσεις, Σωστό-Λάθος, Μικρές Απαντήσεις, Ταίριασμα και ερωτήσεις με Νούμερα, όπως και εισαγωγή «________» για μια λέξη που λείπει. Οι ερωτήσεις Cloze δεν υποστηρίζονται προς στιγμής. Πολλαπλοί τύποι ερωτήσεων μπορούν να ανακατευθούν σε ένα αρχείο, η μορφοποίηση υποστηρίζει σχόλια γραμμής, ονόματα ερωτήσεων, ανατροφοδότηση και βαθμούς βάση του ποσοστού βαρών. Έχουμε τα εξής παραδείγματα:

    Who's buried in Grant's tomb?{~Grant ~Jefferson =no one}

Grant is {~buried =entombed ~living} in Grant's tomb.

Grant is buried in Grant's tomb.{FALSE}

Who's buried in Grant's tomb?{=no one =nobody}

When was Ulysses S. Grant born?{#1822}

    More info about the "GIFT" format

Moodle XML μορφοποίηση

    Εξαγωγή των ερωτήσεων του κουίζ σε XML κείμενο. Μπορεί να γίνει εισαγωγή τους σε μια άλλη κατηγορία του κουίζ ή να χρησιμοποιηθούν για μια άλλη διαδικασία, όπως XSTL μετατροπές. Η XML μορφοποίηση εξάγει εικόνες που έχει επισυνάψει κάποιος σε ερωτήσεις (encoded base64).

IMS QTI 2.0

    Εξαγωγές σε IMS QTI (version 2.0) μορφή. Γίνεται παράγωγη μιας ομάδας αρχείων μέσα σε ένα 'zip' αρχείο.

    More information on the IMS QTI site (external site in new window)

XHTML

    Εξαγωγή της κατηγορίας σε μια σελίδα XHTML. Κάθε κατηγορία τοποθετείται στο δικό της <div> στοιχειό. Εάν είναι επιθυμητή η χρήση της σελίδας όπως-είναι, πρέπει να γίνει επεξεργασία του <form> στοιχείου στην αρχή <body> τομέα για την παροχή μια σωστής ενεργείας (e.g. a 'mailto').

Θα ακολουθούσουν και άλλες μορφές(π.χ. WebCT,…)
