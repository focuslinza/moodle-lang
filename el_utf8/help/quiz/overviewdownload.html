Επισκόπηση μεταφόρτωσης δεδομένων (download)

Ίσως επιθυμείτε να μεταφορτώσετε τα δεδομένα ενός πίνακα που εμφανίζεται στην οθόνη σας για περαιτέρω ανάλυση. Μπορείτε να διαλέξετε ανάμεσα σε δυο μορφές αρχείων για τα μεταφορτωμένα δεδομένα.
Και στις δυο περιπτώσεις τα δεδομένα θα παρουσιαστούν σε πίνακες με τους σχετικούς τίτλους, όπως ήταν και στην οθόνη. Αν ο πίνακας είναι σελιδοποιημένος, τότε όλα τα δεδομένα όλων των σελίδων θα μεταφορτωθούν σε ένα και μόνο αρχείο.

Μορφή αρχείων λογιστικών φύλλων τύπου Excel (Excel Spreadsheets format):

Θα δημιουργηθεί αρχείο λογιστικού φύλλου τύπου Excel ( .xls spreadsheet document).

Μορφή αρχείου κειμένου (Text format):

Στην περίπτωση αυτή τα δεδομένα θα αποθηκευτούν σε απλό αρχείο κειμένου. Κάθε σειρά του πίνακα θα αποδίδεται με μια γραμμή στο αρχείο κειμένου. Οι τιμές των δεδομένων θα χωρίζονται με διάκενα (tabstops).

Overview Download

You may want to download the data displayed in the on-screen table for further analysis. You can choose between two file formats for downloaded data.
In each case data will be presented as a table with appropriate titles, as on the screen. If the table is paged, all data for all pages will be downloaded in a single file.

Excel Spreadsheet format:

You will get an .xls spreadsheet document.

Text format

In this case, data will be stored as a regular text file. A line for each row in the table with data separated by tabstops.
