Κατέβασμα Σύνοψης

Μπορείτε να κατεβάσετε τα δεδομένα του πίνακα που εμφανίζεται στην οθόνη για περαιτέρω ανάλυση. Για το σκοπό αυτό έχετε δύο επιλογές για τη μορφή των δεδομένων.
Σε κάθε περίπτωση τα δεδομένα θα παρουσιάζονται σαν πίνακας με τις κατάλληλες επικεφαλίδες, όπως στην οθόνη. Αν ο πίνακας στην οθόνη αποτελείται από πολλές σελίδες, τα δεδομένα που θα κατεβάσετε θα περιέχουν όλες τις σελίδες αυτές με τη μορφή ενός ενιαίου πίνακα.

Μορφή λογιστικού φύλλου Microsoft Excel:

Τα δεδομένα αποθηκεύονται σε ένα έγγραφο .xls.

Μορφή αρχείου κειμένου:

Τα δεδομένα αποθηκεύονται σαν ένα αρχείο κειμένου. Οι γραμμές του πίνακα γίνονται γραμμές του αρχείου, και οι στήλες διαχωρίζονται με χαρακτήρες στηλοθέτη.
