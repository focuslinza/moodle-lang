Download Απαντήσεων

Μπορεί να επιθυμείτε να κάνετε download τα δεδομένα που προβάλονται στον πίνακα της οθόνης. Μπορείτε να επιλέξετε ανάμεσα σε 2 formats αρχείων για το download των δεδομένων.
Σε κάθε περίπτωση τα δεδομένα θα παρουσιάζονται ως πίνακας με τις κατάλληλους τίτλους, όπως στην οθόνη αλλά χώρίς τις εικόνες των χρηστών.
Εάν ο πίνακας κοπεί σε σελίδες ή ταξινομηθεί στην οθθόνη όλα τα δεδομένα για όλες τις σελίδες θα γίνονται πάντα download σε ένα μόνο αρχείο. Οι χρήστες με καμία προσπάθεια στο κουίζ θα περιλαμβάνονται στο αρχείο εάν επιλέξετε την αντίστοιχη επιλογή.

Excel Spreadsheet format:

Θα λάβετε ένα αρχείο με κατάληξη .xls.

Text format

Σε αυτή την περίπτωση τα δεδομένα θα αποθηκευτούν ως ένα απλό αρχείο κειμένου και θα περιλαμβάνουν μια γραμμή για κάθε γραμμή του πίνακα με διαχωρισμό από tabstops.
