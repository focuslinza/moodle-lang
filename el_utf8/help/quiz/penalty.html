Ποινή

Μπορείτε να ορίσετε το μέρος των βαθμών που θα αφαιρεθεί για κάθε λάθος απόκριση. Η δυνατότητα αυτή έχει αξία, αν η άσκηση τύπου κουίζ είναι οργανωμένη με τρόπο που να επιτρέπει στο διδασκόμενο να έχει επαναλαμβανόμενες αποκρίσεις σε μια ερώτηση. Η ποινή πρέπει να είναι ανάμεσα στο 0 και στο 1. Ποινή 1 σημαίνει ότι ο διδασκόμενος πρέπει να απαντήσει σωστά στην πρώτη απόκριση, για να κερδίσει τους βαθμούς της ερώτησης. Ποινή 0 σημαίνει ότι ο διδασκόμενος μπορεί να δοκιμάζει όσο συχνά θέλει, μέχρι να βρει τη σωστή απάντηση, οπόταν θα πάρει και τους βαθμούς.

Penalty factor

You can specify what fraction of the achieved score should be subtracted for each wrong response. This is only relevant if the quiz is run in adaptive mode so that the student is allowed to make repeated responses to the question. The penalty factor should be a number between 0 and 1. A penalty factor of 1 means that the student has to get the answer right in his first response to get any credit for it at all. A penalty factor of 0 means the student can try as often as he likes and still get the full marks.
