Δημιουργία πολλαπλών ερωτήσεων

Αυτή η σελίδα σας επιτρέπει να δημιουργήσετε πολλαπλές ερωτήσεις με μία κίνηση.

Αυτή τη στιγμή μπορείτε απλώς να δημιουργήσετε έναν αριθμό από τυχαίες ερωτήσεις και (προαιρετικά) να τις προσθέσετε στο τρέχον κουίζ.

Σε κάποια επόμενη έκδοση του Moodle αυτή η σελίδα θα εξελιχθεί σε έναν μεγαλύτερο οδηγό που θα έχει περισσότερες επιλογές.
