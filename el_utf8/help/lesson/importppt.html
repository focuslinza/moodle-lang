Εισαγωγή διαφανειών από Power Point HTML

ΟΔΗΓΙΕΣ

Όλες οι διαφάνειες τύπου Power Point εισάγονται ως Πίνακες - Κλάδοι (Branch Tables) με επιλογές 'Προηγούμενη' (Previous) και 'Επόμενη' (Next).

  1. Ανοίξτε το αρχείο - παρουσίαση τύπου PowerPoint.
  2. Αποθηκεύστε το ως Ιστοσελίδα (Web Page) χωρίς ειδικές ρυθμίσεις
  3. Το αποτέλεσμα πρέπει να είναι η δημιουργία ενός αρχείου τύπου htm και ενός φακέλου που περιέχει όλες τις διαφάνειες σε μορφή ιστοσελίδων (μια ιστοσελίδα για κάθε διαφάνεια).
    ΣΥΜΠΙΕΣΤΕ ΜΟΝΟ ΑΥΤΟΝ ΤΟΝ ΦΑΚΕΛΟ .
  4. Πηγαίνετε στο Moodle και προσθέστε μια νέα ενότητα.
  5. Μόλις αποθηκεύσετε τις ρυθμίσεις της ενότητας, θα δείτε τέσσερις (4) επιλογές κάτω από τον τίτλο "Τι θέλετε να κάνετε σε πρώτη φάση;" Επιλέξτε (κάντε κλικ) "Εισαγωγή από Power Point".
  6. Χρησιμοποιήστε την "Αναζήτηση...", και βρείτε το συμπιεσμένο αρχείο που πριν λίγο δημιουργήσατε. Στη συνέχεια επιλέξτε (καντε κλικ) "Φόρτωση αυτού του αρχείου".
  7. Αν όλα έχουν πάει καλά, το επόμενο παράθυρο θα έχει απλώς ένα κουμπί με την ένδειξη "Συνέχεια".

Αν η παρουσίαση Power Point εμπεριείχε εικόνες, αυτές θα έχουν αποθηκευτεί ως αρχεία μαθήματος (course files) στη θέση moddata/XY (όπου X νοείται το όνομα της διδακτικής ενότητας που έχετε ορίσει, και όπου Y είναι ένας αριθμός, συνήθως το μηδέν). Κατα τη διαδικασία της εισαγωγής δημιουργούνται αρχεία στο κατάλογο δεδομένων moodle (moodle data directory) μέσα στο φάκελο temp/lesson. Τα αρχεία αυτά δε διαγράφονται ακόμη από το importppt.php.

 

Import PowerPoint HTML

HOW TO USE

All of the PowerPoint slides get imported as Branch Tables with Previous and Next answers.

  1. Open up your PowerPoint presentation.
  2. Save it As a Web Page (no special options)
  3. The result of step 3 should be a htm file and a folder with all of the slides converted to web pages.
    ZIP THE FOLDER only.
  4. Go to your moodle site and add a new lesson.
  5. After saving the lesson settings you should see 4 options under "What would you like to do first?" Click on "Import PowerPoint"
  6. Use to the "Browse..." button to find your zip file from step 3. Then click on "Upload this file"
  7. If everything worked, the next screen should just display a continue button.

If any images were in your PowerPoint, they would have been saved as course files in moddata/XY where X is your lesson's name and Y is a number (usually 0). Also, during the import process, files are created in your moodle data directory inside temp/lesson. These files are not deleted by importppt.php as of yet.
