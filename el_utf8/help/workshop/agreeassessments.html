Συμφωνία αξιολογήσεων

Μια αξιολόγηση μπορεί να έχει τα εξής 3 χαρακτηριστικά:

  1. Κατά την φάση των αξιολογήσεων και καταχωρήσεων οι μαθητές βλέπουν τις αξιολογήσεις που έχουν γίνει από άλλους μαθητές. Δεν υπάρχει ανατροφοδότηση από τους μαθητές που κατέθεσαν την εργασία τους σε αυτούς που κάνουν την αξιολόγηση. Υπάρχει μονόδρομη ανατροφοδότηση ανάμεσα στους ατομικούς βαθμολογητές πάνω σε οποιοδήποτε κομμάτι εργασίας.
  2. Κατά την φάση υποβολής και αξιολόγησης, οι μαθητές βλέπουν τις αξιολογήσεις που έχουν γίνει από τους άλλους μαθητές και προσθέτουν τα δίκα τους σχόλια. Μπορεί να υπάρχει συμφωνία ή διαφωνία ως αναφορά τις ατομικές αξιολογήσεις. Εάν υπάρχει συμφωνία τότε ο βαθμός αυτός χρησιμοποιείται στους τελικούς υπολογισμούς των ατομικών βαθμολογίων κάποιου κομμάτιου της συνολικής εργασίας. Εάν, δεν υπάρξει συμφωνία, ο μαθητής που έκανε την αξιολόγηση μπορεί να αναθεωρήσει. Η διαδικασία αυτή μπορεί να συνεχιστεί έως ότου υπάρξει συμφωνία ή τέλος προθεσμίας. Εάν μια αξιολόγηση είναι σε κατάσταση "διαφωνίας" μετά το τέλος προθεσμίας, δε χρησιμοποιείται στο τελικό υπολογισμό. Έτσι, υπάρχει αμφίδρομη ανατροφοδότηση ανάμεσα στους ατομικούς βαθμολογητές για κάθε κομμάτι εργασίας.

Εάν η δεύτερη μέθοδος εργασίας επιλεχτεί υπάρχει η δυνατότητα απενεργοποίησης εμφάνισης της βαθμολογίας. Αρχικά, μόνο τα σχόλια εμφανίζονται στο μαθητή του οποίου η δουλεία αξιολογείται. Οι βαθμοί εμφανίζονται κατόπιν συμφωνίας (όσο αναφορά τα σχόλια). Αυτή η μέθοδος εργασίας είναι αποτελεσματική όταν είναι αναγκαία η επίτευξη συμφωνίας στις αξιολογήσεις.
