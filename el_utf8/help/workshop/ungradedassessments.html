Μη βαθμολογούμενες εργασίες

Γενικά, οι εργασίες που γίνονται από τους μαθητές, παίρνουν μια αναλογία βαθμών οι οποίες μετρούν στον τελικό βαθμό. Η βαθμολόγηση αυτών των εργασιών (και αυτών που έχουν γίνει από τον διδάσκοντα) γίνεται στο παρασκήνιο από το module εργαστηρίου. Εάν μια υποβολή αξιολογείται τρεις ή περισσότερες φορές, το module μπορεί να κάνει μια λογική εκτίμηση για την "αξία" κάθε αξιολόγησης. (Για υποβολές οι οποίες έχουν εκτιμηθεί μόνο μία ή δύο φορές, σε αυτές οι αξιολογήσεις δίνεται ο μέγιστος "βαθμός βαθμολόγησης".) Καθώς η διαδικασία βαθμολόγησης γίνεται σε κανονικά διαστήματα, ο αριθμός των μη βαθμολογημένων αξιολογήσεων είναι μηδέν. (Σημειώστε ότι στα Εργαστήρια όπου οι αξιολογήσεις μπορούν να συμφωνηθούν, οι αξιολογήσεις που δεν έχουν συμφωνηθεί πιθανόν να μην βαθμολογηθούν. Σε αυτό το τύπο εργαστηρίου ο αριθμός των μη-βαθμολογημένων αξιολογήσεων είναι πιθανόν να είναι μη-μηδενικό αλλά ο αριθμός πρέπει να μειώνεται καθώς συμφωνούνται οι αξιολογήσεις.)

Η λειτουργία αυτής της σύνδεσης είναι να επιβάλλει έναν επαν-υπολογισιμό των βαθμών βαθμολόγησης, των βαθμών που αποδίδονται στις αξιολογήσεις. Ο καθηγητής ίσως να θέλει να το κάνει αυτό όταν οποιαδήποτε από τις παρακάτω ρυθμίσεις έχει αλλάξει:

  1. Η ρύθμιση "Σύγκριση Αξιολογήσεων";
  2. Η βαρύτητα που δίνεται στις αξιολογήσεις των καθηγητών.
Το πάτημα σε αυτήν την σύνδεση ενημερώνει άμεσα τους βαθμούς βαθμολόγησης, αντικατοπτρίζοντας τις τρέχουσες τιμές που δόθηκαν σε αυτές τις παραμέτρους. Οι αλλαγές σε άλλες ρυθμίσεις του εργαστηρίου, όπως η τιμή του ίδιου του βαθμού βαθμολόγησης, δεν απαιτούν την εκτέλεση αυτού του επανυπολογισμού.
