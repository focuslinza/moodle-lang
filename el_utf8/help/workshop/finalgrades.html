H τελική βαθμολογία

Ο πίνακας στο παράθυρο παρουσιάζει την τελική βαθμολογία γενικά και αναλυτικά. Ο πρώτος κατά σειρά βαθμός αποτελεί τη βαθμολογία της αρχικής αξιολόγησης. Ο βαθμός αυτός προστίθεται στους επόμενους βαθμούς κάθε εργασίας που έχει υποβληθεί. Η τελική βαθμολογία προκύπτει από το μέσο όρο όλων των βαθμών ή από το μέσο όρο της καλύτερης εργασίας.

Ο Πίνακας Κατάταξης παρέχει τη δυνατότητα παράθεσης των καλύτερων επιδόσεων. Στις παραμέτρους της ’σκησης μπορεί κανείς να ορίσει πόσες βαθμίδες θα έχει ο Πίνακας Κατάταξης. Σε περίπτωση που ορίσουμε μηδέν (0) βαθμίδες κατάταξης, τότε ο Πίνακας Κατάταξης δεν εμφανίζεται καθόλου. Η σειρά των βαθμίδων είναι φθίνουσα (από την καλύτερη στη χειρότερη επίδοση). Στον πίνακα Κατάταξης εμφανίζεται μόνο η καλύτερη επίδοση κάθε διδασκόμενου.

The Final Grades

The table on this screen lists the overall grades and their breakdown. The first grade is the grade for the (initial) assessment. This is added to the grades given to each submission. The final grade given for this assignment will either be the average of these overall grades or the overall grade of the best submission.

The League table optionally lists the best submissions. The number of entries in the table is set as one of the Exercise parameters. If that number is set to zero then the League Table is not displayed. The table is in grade order with the best submission first. Only a student's best submission is shown in the table.
