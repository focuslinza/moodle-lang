Λήψη δεδομένων από Πίνακες

Μπορεί να θέλετε να κάνετε λήψη των δεδομένων που εμφανίζονται στην οθόνη για τραπέζι για περαιτέρω ανάλυση. Μπορείτε να επιλέξετε μεταξύ πολλών φορμάτ αρχείων για ληφθέντα δεδμοένα που φαίνονται παρακάτω.

Σε κάθε περίπτωση τα δεδομένα θα παρουσιάζονται ως ένας πίνακας με κατάλληλους τίτλους στηλών, όπως στην οθόνη. Ο κατεβασμένος πίνακας είναι :

  • μη σελιδοποιημένος, όλα τα δεδομένα για τις σελίδες θα ληφθούν σε ενιαίο αρχείο.
  • μη φιλτραρισμένος σύμφωνα με τα αρχικά του ονόματος χρήστη. (Ορισμένοι πίνακες στην οθόνη με στοιχεία σχετικά με τους χρήστες σας επιτρέπουν να επιλέξτε να εμφανίσετε τα δεδομένα μόνο για τους χρήστες με επιλεγμένα αρχικά του μικρού ονόματος/επιθέτου)
  • δεν επηρεάζονται από την ταξινόμηση κατά στήλες οπυ ένας χρήστης μπορεί να εκτελέσει στον πίνακα στην οθόνη.
