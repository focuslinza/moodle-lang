Φορτώνοντας μια εικόνα

Μπορείτε να φορτώσετε μια εικόνα από τον υπολογιστή σας σε αυτόν τον εξυπηρετητή και αυτή θα χρησιμοποιηθεί σε διάφορα σημεία για να αντιπροσωπεύσει εσάς.

Για αυτό το λόγο οι καλύτερες εικόνες είναι κοντινές φωτογραφίες του προσώπου σας, αλλά μπορείτε να χρησιμοποιήσετε όποια εικόνα επιθυμείτε.

Η εικόνα πρέπει να είναι μορφής JPG ή PNG (δηλ. τα ονόματα θα τελειώνουν σε .jpg ή .png).

Μπορείτε να αποκτήσετε μια τέτοια εικόνα χρησιμοποιώντας μία από τις τέσσερις ακόλουθες μεθόδους:

  1. Με μια ψηφιακή φωτογραφική μηχανή οι φωτογραφίες θα είναι πιθανότητα ήδη στον υπολογιστή σας και στην κατάλληλη μορφή.
  2. Μπορείτε να σαρώσετε μια εκτυπωμένη φωτογραφία. Βεβαιωθείτε πως την αποθηκεύετε σε μορφή JPG ή PNG.
  3. Αν είστε καλλιτέχνης ίσως να θέλετε να ζωγραφίσετε μια εικόνα χρησιμοποιώντας ένα πρόγραμμα ζωγραφικής.
  4. Τέλος, μπορείτε να "δανειστείτε" εικόνες από το διαδίκτυο. Το http://images.google.com είναι μια πολύ καλή αρχή. Αφού βρείτε μια εικόνα, μπορείτε να την αποθηκεύσετε με δεξί κλικ πάνω σε αυτή και μετά να επιλέξτε "Αποθήκευση εικόνας ως" ή "Save image as" από το μενου (σε διαφορετικούς υπολογιστές μπορεί να διαφέρει το μήνυμα).

Για να φορτώσετε την εικόνα στο Moodle πατήστε το κουμπί "Αναζήτηση" σε αυτή τη σελίδα και επιλέξτε ένα αρχείο εικόνας από τον υπολογιστή σας.

Σημείωση: Βεβαιωθείτε πως το αρχείο εικόνας δεν είναι μεγαλύτερο από το όριο που ορίζεται εδώ αλλιώς δε θα φορτωθεί το αρχείο.

Μετά πατήστε "Ενημέρωση του προφίλ μου" στο κάτω μέρος - το αρχείο εικόνας θα κοπεί σε τετράγωνη μορφή στα 100x100 pixels.

Όταν επιστρέψετε στη σελίδα του προφίλ σας, η εικόνα ίσως να μη φαίνεται να έχει αλλάξει. Αν συμβαίνει αυτό απλά πατήστε το κουμπί "Ανανέωση" (ή "Reload" ή "Refresh", ανάλογα με τον υπολογιστή σας) στον περιηγητή σας.
