Εικονίδιο προόδου

Τα κουτιά επιλογής εμφανίζονται κοντά σε κάποιες δραστηριότητες και αποτελούν έναν τρόπο παρακολούθησης της προόδου σας στο μάθημα.

Όταν δείτε ένα κουτί επιλογή με κουκκίδες, μπορείτε να πατήσετε επάνω του και να το γεμίσετε. (Μπορείτε να ξαναπατήσετε πάτω του για να το αδειάσετε αν αλλάξετε γνώμη)

Αυτά τα εικονίδια προορίζονται για να σας βοηθήοσυν να παρακολουθήσετε την πορεία προόδου σας και είναι εντελώς προαιρετικά. Αν θέλετε, γεμίστε το κουτί όταν νομίσετε ότι έχετε ολοκληρώσει κάτι.

Κάποια μαθήματα μπορεί να περιέχουν κενά κουτιά - κενά διαστήματα που δεν έχουν κουτί επιλογής.Δεν μπορείτε να γεμίσετε αυτά τα κουτιά εσείς, αλλά θα γεμίσουν αυτόματα σύμφωνα με συνθήκες που ορίστηκαν κατά τη δημιουργία του μαθήματος. Για παράδειγμα, ένα κουτί μπορεί να γεμίσει όταν μεταφορτώσετε ένα αρχείο. (Κάποιες φορές μπορεί να χρειαστεί να επαναφορτώσετε τη σελίδα για να δείτε αυτού του είδους την αλλαγή)

Κάποιοι διαχειριστές, όπως οι καθηγητές σας, μπορεί να μπορούν να δουν ποια κουτιά έχετε γεμίσει. Ωστόσο, αυτή η πληροφορία δε χρησιμοποιείται στην αξιολόγηση και δεν υπολογίζεται στο βαθμό του μαθήματος. Δεν χρειάζεται να συμπληρώσετε τα κουτιά αν δε το επιθυμείτε.
