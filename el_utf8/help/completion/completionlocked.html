Κλείδωμα επιλογών ολοκλήρωσης

Αν έστω και ένα άτομο έχει ολοκληρώσει μία δραστηριότητα, οι επιλογές ολοκλήρωσης "κλειδώνουν". Αυτό συμβαίνει γιατί η αλλαγή αυτών των επιλογών μπορεί να προκαλέσει απρόσμενα αποτελέσματα.

Πιθανή σύγχυση

Αν κάποιος έχει σημειώσει με το χέρι μία δραστηριότητα ως ολοκληρωμένη και εσείς στη συνέχεια ορίσετε τη δραστηριότητα ως αυτόματα ολοκληρωμένη, το κουτί της δραστηριότητας θα αδειάσει για αυτό το άτομο και θα του προκαλέσει σύγχυση.

Είναι καλύτερο να μη ξεκλειδώσετε τις επιλογές, παρά μόνο όταν είστε απολύτα σίγουροι ότι δεν θα προκληθούν προβλήματα - για παράδειγμα, αν γνωρίζεται ότι οι μαθητές δεν έχουν προς το παρόν πρόσβαση στο μάθημα, έτσι μόνο οι διαχειριστές θα είναι αυτοί που θα έχουν σημειώσει μία δραστηριότητα ως ολοκληρωμένη κάνοντας ελέγχους.

Τι συμβαίνει όταν ξεκλειδώνετε τις επιλογές

Όταν ξεκλειδώσετε τις επιλογές και πατήσετε "Αποθήκευση αλλαγών", όλες οι πληροφορίες ολοκλήρωσης για τη δραστηριότητα θα διαγραφούν και, αν είναι δυνατό, θα αναπαραχθούν σύμφωνα με τις νέες ρυθμίσεις.

  • Αν η νέα επιλογή ολοκλήρωση είναι η χειροκίνητη, σε όλους θα εμφανίζεται "μη ολοκληρωμένο" ανεξαρτήτως με τις προηγούμενες ρυθμίσεις.
  • Αν είναι αυτόματα, τότε, ανάλογα με τις ενεργοποιημένες επιλογές, ίσως είναι σε θέση να υπολογίσει μία σωστή τιμή για κάθε χρήστη.
    • Η απαίτηση "αναγνωσμένο" δεν θα λειτουργήσει - ακόμα και αν ένας μαθητής έχει αναγνώσει προηγούμενως τη δραστηριότητα, αυτή δε θα σημειωθεί ως αναγνωσμένη, μέχρι ο μαθητής να τη διαβάσει ξανά.
    • Οι περισσότερες από τις εναπομείνουσες επιλογές θα ξαναυπολογιστούν επιτυχώς.

Αν αλλάξετε τις επιλογές ολοκλήρωσης, ενώ ένας μαθητής είναι συνδεδεμένος, μπορεί να μη τις αλλαγές για κάποιο χρονικό διάστημα.
