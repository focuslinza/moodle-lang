Αλφαβητική κλίμακα

Εδώ μπορείτε να ορίσετε την αλφαβητική κλίμακα βαθμολόγησης. Αρχικά μια "προτεινόμενη" κλίμακα παρουσιάζεται και το μόνο που πρέπει να κάνετε για να τη χρησιμοποιήσετε είναι να επιλέξετε "Αποθήκευση Αλλαγών". Αν, ωστόσο, δεν σας αρέσει αυτή η κλίμακα, τότε αλλάξτε τις καταχωρήσεις που θέλετε και επιλέξτε στη συνέχεια "Αποθήκευση Αλλαγών". Μετά από αυτήν την αρχική ρύθμιση θα δείτε την τρέχουσα επιλεγμένη κλίμακα. Αφήστε μία καταχώρηση κενή για να μην συμπεριληφθεί ή να διαγραφεί από την κλίμακα βαθμολόγησης.
Σφάλματα που μπορεί να δείτε στο βαθμολόγιο:

  • Δεν έχουν οριστεί βαθμοί: Αυτό σημαίνει ότι η αρχική κλίμακα βαθμολογίας δεν έχει οριστή. Αυτό που πρέπει να κάνετε (εάν σας αρέσει η προτεινόμενη κλίμακα) είναι να επιλέξετε "Αποθήκευση Αλλαγών".
  • Δεν υπάρχει βαθμός για το Χ: Δεν έχει οριστεί αλφαβητικός βαθμός για το Χ.
