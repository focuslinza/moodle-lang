Κλίμακα βαθμολογίας με Γράμματα.

Μπορείτε να ορίσετε την κλίμακα της βαθμολογίας με "γράμματα" εδώ. Αρχικά μια "προτεινόμενη" κλίμακα παρουσιάζεται και το μόνο που πρέπει να κάνετε για να χρησιμοποιήσετε αυτή την κλίμακα είναι να κάνετε αριστερό κλικ στην επιλογή "αποθήκευση αλλαγών". Εάν εντούτοις δεν σας αρέσει αυτή η κλίμακα, τότε αλλάξτε τις καταχωρήσεις που θέλετε εσείς και κάντε αριστερό κλικ "αποθήκευση αλλαγών". Μετά από αυτήν την αρχική ρύθμιση θα δείτε την τρέχουσα επιλεγμένη κλίμακά σας. Αφήστε μία κενή εισαγωγή για να μην το συμπεριλάβετε ή για να μην διαγράψετε την κλίμακα βαθμολογίας.

Λάθη που μπορεί να δείτε στο βιβλίο βαθμολογίας:

  • Δεν μπορείτε να αποθηκεύσετε βαθμολογίες: Αυτό σημαίνει ότι η αρχική κλίμακα βαθμολογίας δεν έχει τεθεί. Αυτό που πρέπει να κάνετε (εάν σας αρέσει η προτεινόμενη κλίμακα) είναι να κάνετε αριστερό κλικ στο "αποθήκευση αλλαγών".
     

  • Δεν εμφανίζεται ένα γράμμα για τον "Χ" βαθμό: Μας δείχνει ότι δεν υπάρχει το κατάλληλο γράμμα για τον βαθμό "Χ".
