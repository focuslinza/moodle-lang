Απαγόρευση διευθύνσεων IP

Η κάθε καταχώρηση θα πρέπει να είναι μία γραμμή.

Για παράδειγμα: 192.168. , 231.54.211.0/20, 231.3.56.211, 231.3.56.10-20

Υπάρχουν τέσσερις τύποι αριθμών που μπορείτε να χρησιμοποιήσετε (δεν είναι δυνατή η χρήση ονομάτων domain όπως example.com):

  1. Ολόκληρες διευθύνσεις IP όπως 192.168.10.1 οι οποίες θα αντιστοιχούν σε έναν μοναδικό υπολογιστή (ή proxy).
  2. μερικές διευθύνσεις όπως 192.168 οι οποίες θα αντιστοιχούν σε οτιδήποτε θα ξεκινάει με τους αριθμούς αυτούς.
  3. Σήμανση CIDR όπως 231.54.211.0/20 που σας επιτρέπει να ορίσετε υποδίκτυα πιο λεπτομερώς.
  4. Ένα εύρος διευθύνσεων IP 231.3.56.10-20 Το εύρος αυτό αοφα στο τελευταίο τμήμα της διεύθυνσης, οπότε κάτι τέτοιο ορίζει όλες τις διευθύνσεις IP από 231.3.56.10 μέχρι 231.3.56.20.

Τα κενά αγνούνται.
