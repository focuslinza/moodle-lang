Τύποι εργασιών

Υπάρχει ένας αυξανόμενος αριθμός τύπων διαθέσιμων εργασιών:

Δραστηριότητα Εκτός σύνδεσης

Αυτό είναι χρήσιμο όταν μια εργασία γίνετια έξω από το Moodle. Θα μπορούσε να είναι κάτι στο web ή face-to-face.

Οι Μαθητές μπορούν να δουν μια περιγραφή της εργασίας, αλλα δεν μπορούν να κάνουν upload αρχεία ή κάτι τέτοιο. Η βαθμολόγηση λειτουργίε κανονικά, και οι μαθητές θα παίρνουν ενημέρωση για τους βαθμούς τους.

Online κείμενο

Αυτός ο τύπος εργασίας ζητά από τους χρήστες να επεξεργαστούν ένα κείμενο χρησιμοποιώντας εργαλεία επεξεργασίας. Οι διδάσκοντες μπορούν να τους βαθμολογούν online, και ακόμη να προσθέτουν σχόλια ή αλλαγές.

(Εάν είστε γνώστες παλαιότερων εκδόσεων του Moodle, Αυτός ο τύπος εργασίας κάνει την ίδια δουλειά με το παλιό Journal module)

Upload ένα αρχείο

Αυτός ο τύπος εργασίας επιτρέπει στον καθενα συμμετέχοντα να κάνει upload ένα απλό αρχείο.

Αυτό μπορεί να είναι ένα κείμενο από επεξεργαστή κειμένου (word αρχείο), ή μια εικόνα, ή ένα web site, ή κάτι που ζητατε από τους μαθητές να υποβάλουν.

Ειδικό uploading αρχείων

Αυτός ο τύπος αργασίας επιτρέπει σε καθένα συμμετέχοντα να κάνει upload ένα ή περισσότερα αρχεία σε οποιοδήποτε format. Αυτά μποεί να είναι ένα κείμενο από επεξεργαστή κειμένου (word αρχείο), ή μια εικόνα, ή ένα web site, ή κάτι που ζητατε από τους μαθητές να υποβάλουν.

Αυτός ο τύπος επιπλέον σας επιτρέπει να κάνετε upload αρχεία πολλαπλών αποκρίσεων. Αρχεία απόκρισης μπορούν να γίνουν uploaded πριν την τελική υποβολή τα οποία μπορούν να χρησιμοποιηθούν για να δώσουν στους συμμετέχοντες διαφορετικά αρχεία να εργάζονται.

Οι συμμετέχοντες επίσης μπορούν να εισάγουν σημειώσεις που θα περιγράφουν τα αρχεία, καταστάσεις προόδου, και άλλες πληροφορίες.

Η υπολοβή αυτού του τύπου των αρχείων πρέπει να ολοκληρωθεί από τον συμμετέχοντα. Μπορείτε να επαναφέρετε οποιαδήποτε εργασία σε κατάσταση πρόχειρης.
