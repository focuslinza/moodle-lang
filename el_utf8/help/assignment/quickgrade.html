Γρήγορη Βαθμολόγηση

Με τη γρήγορη βαθμολόγηση ενεργοποιημένη, μπορείτε γρήγορα να βαθμολογείτε πολλαπλές εργασίας σε μια σελίδα.

Απλά αλλάξτε τους βαθμούς και τα σχόλια και χρησιμοποιείστε το κουμπί Save για να εποθηκεύσετε όλες τις αλλαγές σας στη σελίδα.

Τα κουμπιά κανονικής βαθμολόγησης είναι στα δεξιά μπορούν να δουλέψουν επίσης σε περίπτωση που χρειάζεστε περισσότερο χώρο. Η επιλογή Γρήγορης βαθμολόγησης σόζεται και μπορεί να εφαρμοστεί σε όλες τις εργασίες σε όλλα τα μαθήματα.
