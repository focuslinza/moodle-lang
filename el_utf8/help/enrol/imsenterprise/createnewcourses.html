Αυτόματη Δημιουργία Νέων Μαθημάτων


Το plugin εγγραφής IMS Enterprise μπορεί να δημιουργήσει νέα μαθήματα για δεδομένα που θα βρει στην βάση δεδομένων του IMS αλλά όχι και για στοιχεία που θα βρει στην βάση δεδομένων του Moodle, εάν αυτή η επιλογή είναι ενεργοποιημένη.
Τα μαθήματα αρχικά εξετάζονται (ελέγχονται) από τον κωδικό αναγνώρισής τους (ID), έναν αλφαριθμητικό κωδικό μέσα στον πίνακα με τα μαθήματα του Moodle, ο οποίος μπορεί να διευκρινίσει τον κωδικό που χρησιμοποιείται για να προσδιοριστεί το μάθημα στο Σύστημα Πληροφοριών των Σπουδαστών (παραδείγματος χάριν). Εάν αυτός δεν βρεθεί (ο κωδικός αναγνώρισης), γίνεται αναζήτηση στον πίνακα των μαθημάτων για την ''σύντομη περιγραφή'', η οποία για το Moodle είναι ανάλογο προσδιοριστικό με τον κωδικό αναγνώρισης του μαθήματος... (Σε μερικά συστήματα αυτοί οι δύο τομείς μπορεί να ταυτίζονται). Μόνο όταν αυτή η αναζήτηση αποτύχει μπορεί το plugin να δημιουργήσει προαιρετικά νέα μαθήματα.
Όλα τα προσφάτως δημιουργημένα μαθήματα είναι ΚΡΥΦΑ στην αρχή. Αυτό γίνεται για να αποτρέψει τη δυνατότητα στους σπουδαστές να  περιπλανιούνται (σερφάρουν) στις κενές σελίδες ενός μαθήματος, για τις οποίες ο καθηγητής μπορεί να αγνοεί την ύπαρξή τους.
