Φωτογραφίες

Όταν δημιουργούμε λογαριασμό χρήστη, μπορούμε να προσθέσουμε και φωτογραφία σχετική με το χρήστη - συνήθως με URL προς ένα αρχείο εικόνας.

Στην περίπτωση αυτή η πλατφόρμα Moodle θα δοκιμάσει να μεταφορτώσει το αρχείο εικόνας και να βάλει την εικόνα στη αντίστοιχη θέση εντός της πλατφόρμας Moodle.

ΣΗΜΕΙΩΣΗ: Η διαδικασία αυτή θα επιβαρύνει το πρόγραμμα σε περίπτωση που ανοίξουν πολλοί λογαριασμοί χρηστών, λόγω της ανάγκης μεταφόρτωσης, μετατροπης και αποθήκευσης εικόνων. Δεν συνιστάται η ενεργοποίηση αυτής της επιλογής σε περίπτωση που αναμένεται μεγάλος αριθμός λογαριασμών χρηστών.

Photo's

When creating a new user account based on the Enterprise data, it's possible that a photo is specified for the user - usually by giving a URL to the image file.

If <photo> data is supplied, then Moodle can attempt to download the image file and turn it into the user-picture displayed within Moodle.

PLEASE NOTE: This can add a significant additional burden to the process if there are a lot of user accounts, because of the need to download, modify, and store the images. It is not recommended to activate this option in a context with a very large number of accounts to be created.
