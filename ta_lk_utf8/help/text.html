எழுதுவதில் உதவி

Moodle இல் எழுதுவது பெரும்பாலும் நீங்கள் எதிர்பார்க்கும் வகையில் இருக்கும். அத்துடன் "smilies", "URL முகவரி" HTML சீட்டுகள் போன்றனவற்றையும் உள்ளிட முடியும்.

Smilies (emoticons)

    இச்சிறிய சின்னங்களை உங்கள் உரையில் உள்ளிடுவதற்கு, அச்சின்னங்களுக்குரிய குறிகளைத் தட்டச்சுக. உங்கள் தலையை இடது புறம் திருப்பிப் பார்த்தால், இக்குறிகளே சின்னங்கள் போலத் தோன்றும்.

URLs

    www. அல்லது http:// இல் தொடங்கும் வார்த்தைகள் தானாகவே சொடுக்கக்கூடிய இணைப்பாக மாற்றப்படும்.

    உதாரணமாக: www.yahoo.com அல்லது http://curtin.edu

HTML சீட்டுகள்

    HTML சீட்டுகளின் உப தொகுதி ஒன்று உங்கள் உரையை வடிவமைக்கப் பயன்படுத்தலாம்.

    HTML சீட்டுகள்விளைவு
    <b> bold </b> தடித்த உரை
    <i> italic </i> சாய்ந்த உரை
    <u> underline </u> கீழ்க்கோடிட்ட உரை
    <font color="green"> உதாரணம் </font> உதாரணம்
    <ul> <li>ஒன்று</li> <li>இரண்டு</li> </ul>
