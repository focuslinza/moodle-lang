நிறை உரை HTML தொகுப்பியானது, உங்களுக்கு, ஒரு வார்த்தை-முறைவழியாக்கி இடைமுகம் ஒன்றை, இணையப்பக்கத்தில் தரும். அதில் தட்டச்சப்படுபவை, HTML குறிகளாக மாற்றப்படும்.

உரைகளை வடிவமைப்பதுடன், மேலும் பல வசதிகளை இத்தொகுப்பி வழங்குகிறது.

ஏனைய மென்பொருட்களிலிருந்து உரையை ஒட்டுதல்.

    Microsoft Word போன்ற ஏனைய மென்பொருட்களிலிருந்து நிறை-உரையை வெட்டி ஒட்டும்போது, வடிவமைப்பு பேணப்படும்.

படங்களை உள்ளிடல்

    உங்களிடம் ஏற்கனவே இணையத்தில் பிரசுரிக்கப்பட்ட, URL மூலம் அணுகக்கூடிய படங்கள் காணப்படுமானால், அவற்றைச் சேர்த்துக் கொள்ள, "படத்தை உள்ளிடு" பொத்தைனைச் சொடுக்குக.

அட்டவணைகளை உள்ளிடல்

    உங்கள், உரைக்கு ஒரு தளக்கோலத்தைச் சேர்க்க, கருவிப் பட்டையிலுள்ள "அட்டவணைகளை உள்ளிடு" பொத்தானைச் சொடுக்குக.

இணைப்புகளை உள்ளிடல் /

    ஒரு புதிய இணைப்பு ஒன்றை உருவாக்குவதற்கு, முதலில் இணைப்பு ஏற்படுத்தப்பட வேண்டிய உரையைத் தட்டச்சுக. பின்னர் அதனைத் தேர்ந்து, கருவிப்பட்டையிலுள்ள இணைப்புப் பொத்தானைச் சொடுக்குக. இணைப்பிற்குரிய URL ஐத் தட்டச்சுக.

smilies (emoticons) ஐ உள்ளிடல்

    இச்சிறிய சின்னங்களை உங்கள் உரையினுள் உள்ளிட, கருவிப்பட்டியிலுள்ள smiley சின்னத்தில் சொடுக்குக. பின்வரும் smiley சின்னங்கள் ஒரு உரையாடற் பெட்டியில் காட்டப்படும். அதிலிருந்து நீங்கள் தெரிந்து கொள்ளலாம். (மாற்று வழியாக, அச்சின்னங்களுக்குரிய குறியை நீங்கள் தட்டச்சலாம், அவை அனுப்பப்படும்போது, சின்னங்களாக மாற்றப்பட்டு அனுப்பப்படும். ).
