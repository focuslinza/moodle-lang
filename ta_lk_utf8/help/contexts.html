Contexts

வகிபாகங்கள் நியமிக்கப்படும் இடமே Moodle இல் Contexts ஆகும்.

உயர் context இலிருந்து தாழ்ந்த context இற்கு அனுமதிகள் வழிமுறைவரும் படிநிலை அமைப்பு இங்கு காணப்படும்.

  1. தளம் (தொகுதி)
  2. பாடநெறி வகைகள்
  3. பாடநெறி உப-வகைகள்
  4. பாடநெறிகள்
  5. கட்டங்களும் செயற்பாடுகளும்

இதையும் பார்க்கவும் வகிபாகங்கள், அனுமதிகள், வகிபாகங்களை நியமித்தல் மீறல்கள்.
