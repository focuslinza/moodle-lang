கோப்புகளும் இணையப் பக்கங்களும்

    பாடநெறிக்கோப்புகளையோ அல்லது வேறு இணயத்தளப் பக்கங்கள் மற்றும் கோப்புகளையோ இணைக்கும் ஒரு வளமாக இது அமைகின்றது.

    எளிய இணையப்பக்கங்கள் சாதாரணமாகக் காட்டப்படும், பல்லூடகக் கோப்புகள் ஒரு இணையப்பக்கத்தில் உட்பதிக்கப்பட்டுக் காட்டப்படும்.

    உள்ளடக்கமானது, புதிய சாளரத்திலோ அல்லது வேறு விதமாகவோ காட்டப்படும்.

    இதன் மூலம் உள்ளடக்கத்துக்குத் தேவையான தகவல்களும் வழங்கப்படலாம். உதாரணமாக, பயனாளர் பெயர் ஆகியன.
