Å definere en global ordbok

Systemet med ordbøker lar deg definere begreper som kan være tilgjengelig på hele nettstedet. De beskrives som globale.

For å gjøre dette må du definere hvilken ordbok som er global.

Du kan ha så mange globale ordbøker du vil, og de kan høre til hvilken klasse som helst. Alle regler som gjelder for vanlige ordbøker gjelder også for de globale. 

Merk: Kun administrator kan lage globale ordbøker.
