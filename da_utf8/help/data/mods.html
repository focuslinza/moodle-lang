Aktiviteten Database

Med databasemodulet kan lærere og/eller elever opbygge, vise og søge i en datasamling om ethvert tænkeligt emne. Formatet og strukturen af indholdet kan være nærmest hvad som helst, inklusiv billeder, filer, links, tal, tekst mm. Du kender måske teknologien fra Microsoft Access eller Filemaker databaserne.
