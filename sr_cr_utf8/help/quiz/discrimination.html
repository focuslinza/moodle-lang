Индекс разлике

Ово омогућава груби индикатор квалитета сваке ставке. Израчунава се дељењем броја студената који су по резултатима у првој трећини на овом тесту за свако питање, бројем студената који су у доњој трећини.

Нпр., ако је 30 студената решавало тест, било би по 10 студената у горњој и доњој трећини. Ако је на ставци #1, 9 из групе најбољих студената урадило тачно, али само 3 из најслабије трећине, индекс разлике би био 9/3 = 3.0. Бројеви у заградама су: (тачних одговора из најбоље трећине / тачних одговора из најслабије трећине).

Ако индекс падне испод 1.0 то значи да је више слабијих студената одговорило тачно од бољих студената. Такве ставке би требало одбацити као безвредне. Заправо, оне умањују тачност укупног резултата решавања теста.

Ако нема студената из слабије групе који су тачно одрадили ставку, именилац би био 0 што даје резултат бесконачно. Програм замењује са '10' код сваке такве ставке.
