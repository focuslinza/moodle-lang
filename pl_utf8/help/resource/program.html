Zasoby typu program

Zasób typu program jest zewnętrznym zasobem, który jest w stanie przyjąć dane podawane do niego z platformy Moodle.

Dane przekazywane są poprzez HTTP GET (jako część URL) i zawierają :

  • extern_nav: URL, z którego następuje odwołanie do zasobu
  • extern_usr: Nazwa aktualnego użytkownika
  • extern_nam: Pełne imię aktualnego użytkownika
  • extern_tim: aktualny czas
  • extern_pwd: niepowtarzalne hasło

Wszystko co opisano powyżej zostanie zmodyfikowane w najbliższej przyszłości.
