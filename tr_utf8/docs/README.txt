Moodle belgeleri
----------------

Bu belgeleri başka bir dile çevirmek istiyorsanız,
size şunları öneriyorum:

 1) Bu dizine dosyaların İngilizce sürümlerini eklemeyiniz.
    Çünkü; Çevirdiğiniz dil paketinde dosya kayıpsa
    otomatik olarak İngilizce sürümü kullanılmaktadır.

 2) Çeviriyi şu sıraya göre yapın (öncelikli olanlarla başlıyor):

    EN ÖNEMLİ
    |-------------------
    |
    |   - files.php
    |   - install.html
    |   - installamp.html
    |   - upgrade.html
    |   - faq.html
    |
    |  -----------------
    |
    |   - teacher.html
    |   - module_files.txt
    | 
    |  -----------------
    |   
    |   - intro.html
    |   - features.html
    |   - release.html
    |
    |  -----------------
    |
    |   - developer.html
    |   - cvs.html
    |   - future.html
    |
    |  -----------------
    |   
    |   - license.html
    |
    |-------------------
    AZ ÖNEMLİ

 3) Lütfen credits.html dosyasını çevirmeyiniz. Bu dosya çok değişmektedir.
