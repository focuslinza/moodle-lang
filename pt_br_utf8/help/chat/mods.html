Chat

O módulo Chat permite a realização de uma discussão textual via web em modalidade síncrona. Essa é uma maneira útil de obter um entendimento diferente de cada um e do tópico sendo discutido - o modo de utilização da sala de chat é bem diferente do de fóruns assíncronos. Este módulo contém atributos para a revisão e a administração das discussões.
