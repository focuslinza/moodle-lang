Chat

O módulo de chat permite que os participantes dialoguem em tempo real e em forma síncrona, via Web. Essa é uma forma útil de conhecer os pontos de vista dos outros sobre o tema a ser considerado - o modo como é usada uma sala de chat é bastante diferente do que um fórum que é assíncrono. O módulo de chat tem várias opções para gerir e rever as sessões de chat.
