Fotos

Quando é criada uma nova conta de usuário a partir de dados Enterprise, é possível que uma foto tenha sido especificada pelo usuário - normalmente fornecendo uma URL para o arquivo da imagem.

Se o dado <photo> é fornecido, então o Moodle pode tentar trazer o arquivo da imagem e colocá-lo como sendo a foto do usuário a ser mostrada dentro do Moodle.

FAVOR OBSERVAR: Isto pode adicionar uma sobrecarga ao processo se tiver muitas contas de usuários, por causa da necessidade de transferir, modificar e armazenar as imagens. Não é recomendado ativar esta opção em um contexto com um número muito grande de contas a serem criadas.
