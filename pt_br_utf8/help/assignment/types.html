Tipos de Tarefa

Existem alguns tipos de tarefas

Tarefas offline

- Isto é útil quando a tarefa é realizada fora do Moodle, em outro endereço web ou presencialmente. Os estudante podem ler uma descrição da tarefa, mas não podem enviar documentos. A avaliação das tarefas e a notificação dos estudantes é sempre ativa e pode ser utilizada.

Texto online

Este tipo de tarefa consiste na edição de um texto online, com o uso do editor. Os professores podem dar avaliações, adicionar comentários no texto ou modificá-lo.

(Este tipo de tarefa corresponde ao módulo Diário nas versões anteriores de Moodle)

Envio de um arquivo

- Este tipo de tarefa permite que os participantes enviem um único arquivo de qualquer tipo como imagem, documento Word, web site zipado, etc.

Envio mèultiplo de arquivos

Este tipo de tarefa permite a cada participante enviar um ou mais arquivos de qualquer tipo. Exemplos de formato de arquivo são os textos Word, imagens, web sites em arquivo zip, ou outros documentos que você indicar.

Também permite que você envie múltiplos arquivos de solução. Arquivos de solução também podem ser carregados antes da submissão, que podem ser usados para dar a cada participante um arquivo diferente para ser trabalhado.

Os participantes também podem inserir anotações descrevendo os arquivos submetidos, o status do progresso ou qualquer outra informação textual

Submissões desse tipo de tarefa devem ser finalizadas manualmente por cada participante. Você pode rever o status atual a qualquer momento, tarefas não concluídas são marcadas como Rascunho. Você pode reverter qualquer tarefa não avaliada de volta para o status de Rascunho.
