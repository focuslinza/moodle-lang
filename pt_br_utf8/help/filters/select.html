Filtro selecionado

Este filtro serve para filtrar informações e é escolhido a partir de uma lista que se abre. O filtro tem as seguintes opções:

  • é qualquer valor - esta opção desabilita o filtro (isto é: toda informação é aceita pelo filtro)
  • é igual a - este filtro permite somente informação que seja igual ao valor selecionado da lista
  • é diferente de - esta opção permite somente informação que seja diferente do valor selecionado da lista
