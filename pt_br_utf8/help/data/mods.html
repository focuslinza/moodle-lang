Atividados de Banco de Dados

O módulo de Banco de Dados permite ao professor e/ou alunos criar, mostrar e buscar um banco de entradas de registros sobre qualquer tópico. O formato e a estrutura dessas entradas podem ser quase ilimitados, incluindo imagens, arquivos, URLs, números e texto, entre outras coisas. Você pode estar familiarizado com uma tecnologia parecida na construção de bancos de dados Microsoft Access ou Filemaker.
