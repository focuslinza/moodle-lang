Mapeamento de Campos

Esse menu permite guardar os dados de um banco de dados existente. Para preservar os dados em um campo, você deve mapeá-lo para um novo campo, onde os dados aparecerão. Os campos também podem ficar em branco, para que nenhuma informação seja copiada para ele. Os campos antigos não mapeados para um novo serão perdidos e os dados, removidos.

Você só pode mapear campos de um mesmo tipo, então cada caixa de seleção terá campos diferentes. Também tenha cuidado para não tentar mapear um campo para mais de um outro campo.
