Escolhas

Este módulo permite uma atividade muito simples. O professor elabora uma pergunta com diversas opções de solução. Serve para fazer pesquisas rápidas de opinião, para estimular a reflexão sobre um tópico, para escolher entre sugestões dadas para a solução de um problema ou para obter a permissão de utilizar dados pessoais dos alunos em pesquisas do professor.
