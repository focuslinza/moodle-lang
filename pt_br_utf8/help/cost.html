Preço do Curso

Se este site foi configurado para usar um método de inscrição que solicite pagamento (como o módulo Paypal), então você pode colocar o custo aqui, sem símbolo algum, por exemplo: 19,95. A moeda corrente é ajustada automáticamente pelo plugin.

Se o campo custo estiver vazio, então a opção pagamento não é apresentada e a interface será direcionada para outros métodos de pagamento. (como chaves, ou pagamentos manuais).

Se o campo custo NÃO estiver vazio, aos estudantes que tentarem registrar-se será apresentada a opção de fazer o pagamento para entrar.

Se você incorporar TAMBÉM uma chave de acesso às configurações do curso, os estudantes terão a opção de registrar-se usando uma chave. É uma opção útil quando houver uma mistura de estudantes pagantes e não-pagantes.
