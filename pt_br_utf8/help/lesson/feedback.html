Mostrar Feedback Padrão

Se configurado com Sim, quando uma resposta não é encontrada para uma questão específica, a resposta padrão "Esta é a resposta correta" e "Esta é a resposta errada" serão usadas.

Se configurado com Não, quando uma resposta não é encontrada para uma questão específica, então nenhum feedback é mostrado. O usuário fazendo a lição será automaticamente transferido para a página da próxima Lição.
