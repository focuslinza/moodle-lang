O Número Máximo de Respostas em uma Lição

Este valor determina o número máximo de respostas que o professor pode usar. O valor padrão é 4. Por exemplo, se a lição usar sempre questões VERDADEIRO ou FALSO, é recomendável que este valor seja 2.

Este parâmetro também define o número máximo de ramificações que pode ser usado em uma Tabela de Ramificações.

É seguro mudar o valor deste parâmetro em uma lição que já tem conteúdo. Na realidade, se você quiser acrescentar uma questão com muitas alternativas ou uma Tabela de Ramificação longa, será necessário mudar este parâmetro. Depois que a questão ou Tabela de Ramificação pouco comum tiver sido acrescentada, este parâmetro pode ser reduzido para valores mais "comuns".
