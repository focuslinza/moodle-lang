IMS Content Packages

    This resource type allows you to deploy packages conforming the IMS Content Package specifications.

    Simply upload the .zip file containig the package, choose it in the FileManager and press the <Deploy> button.

    There are many options for displaying your content in popup windows, framed windows and so on.

    You can decide many aspects about how the package will be deployed table of contents, navigation buttons...)

    This resource type is highly experimental so, please, send any problem, idea or improvement to the "Resource module forum" at moodle.org.
