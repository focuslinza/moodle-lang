Filter izbire

Ta filter omogoča izbor podatkov glede na podatke v padajočem meniju. Filter ima naslednje možnosti:

  • je poljubna vrednost - onemogoči filter (t.j. filter sprejme vse podatke)
  • je enak - izbere le podatke, ki so enaki vrednosti, izbrane iz padajočega menija
  • ni enak - izbere le podatke, ki so različni od vrednosti, izbrane iz padajočega menija
