Besedni filter

Ta filter vam omogoča, da informacije filtrirate glede na prosto izbrano besedilo. Filter ima sledeče možnosti:

  • vsebuje - ta možnost dovoljuje le informacije, ki vsebujejo vnešeno besedilo (če ni vnešenega besedila, je filter izklopljen)
  • ne vsebuje - ta možnost dovoljuje le informacije, ki ne vsebujejo vnešenega besedila (če ni vnešenega besedila, je filter izklopljen)
  • je enako - ta možnost dovoljuje le informacije, ki so enake vnešenemu besedilu (če ni vnešenega besedila, je filter izklopljen)
  • se začne s/z - ta možnost dovoljuje le informacije, ki se začnejo z vnešenim besedilom (če ni vnešenega besedila, je filter izklopljen)
  • se konča s/z - ta možnost dovoljuje le informacije, ki se končajo z vnešenim besedilom (če ni vnešenega besedila, je filter izklopljen)
  • je prazno - ta možnost dovoljuje le informacije, ki so enake praznemu nizu (vnešeno besedilo se prezre)
