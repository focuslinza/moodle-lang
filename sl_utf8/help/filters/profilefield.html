Filter glede na profil

Ta filter vam omogoča izbor uporabnikov glede na vrednosti polj v njihovem profilu. Filter lahko uporabite na enem ali več poljih iz profila. Filter podpira naslednje možnosti:

  • vsebuje - izbere le tiste uporabnike, pri katerih izbrano polje vsebuje določeni tekst. Če ne vnesete teksta, filter ni aktiven.
  • ne vsebuje - izbere le tiste uporabnike, pri katerih izbrano polje ne vsebuje določenega teksta. Če ne vnesete teksta, filter ni aktiven.
  • je enako - izbere le tiste uporabnike, pri katerih je izbrano polje enako določenemu tekstu. Če ne vnesete teksta, filter ni aktiven.
  • se začne s/z - izbere le tiste uporabnike, pri katerih se izbrano polje začne z določenim tekstom. Če ne vnesete teksta, filter ni aktiven.
  • se konča s/z - izbere le tiste uporabnike, pri katerih se izbrano polje konča z določenim tekstom. Če ne vnesete teksta, filter ni aktiven.
  • je prazno - izbere le tiste uporabnike, pri katerih je izbrano polje prazno, a definirano. Če vnesete tekst, ga filter ne bo upošteval.
  • ni definiran - izbere le tiste uporabnike, pri katerih izbrano polje ni definirano. Če vnesete tekst, ga filter ne bo upošteval.
  • je definiran - izbere le tiste uporabnike, pri katerih je izbrano polje definirano. Če vnesete tekst, ga filter ne bo upošteval.
