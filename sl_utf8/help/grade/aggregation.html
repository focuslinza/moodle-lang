Agregacija kategorij

Ta menu vam omogoča izbiro strategije agregacije, ki bo uporabljena za izračunavo končne ocene vsakega udeleženca za to kategorijo. Drugačne možnosti so pojasnjene spodaj.

Ocene so najprej spremenjene v odstotkovne vrednosti (interval od 0 do 1, to se imenuje normalizacija), potem pa so agregirane z uporabo ene od funkcij spodaj in na koncu spremenjene v kategorije, povezane postavke v razponu (med Maksimalno oceno in Minimalno oceno).

Pomembno: Prazna ocena je le manjkajoči vpis v redovalnici in lahko pomeni več različnih stvari. Na primer, lahko je udeleženec, ki ni še potrdil naloge, lahko izvajalec še ni ocenil naloge ali pa je bila ocena ročno izbrisana prek skrbnika redovalnice. Svetujemo previdnost pri interpretiranju teh "praznih ocen".

Povprečja ocen
Vsota vseh ocen, deljena s številom vseh ocen.
A1 70/100, A2 20/80, A3 10/10, maksimum kategorije 100:
(0.7 + 0.25 + 1.0)/3 = 0.65 --> 65/100
Tehtano povprečje
Vsaki oceni je lahko dana teža, ki je potem uporabljena v agregaciji aritmetičnega povprečja, da vpliva na vsako postavko v celotnem povprečju.
A1 70/100 teža 10, A2 20/80 teža 5, A3 10/10 teža 3, maksimum kategorije 100:
(0.7*10 + 0.25*5 + 1.0*3)/18 = 0.625 --> 62.5/100
Preprosto tehtano povprečje
Razlika od Tehtano povprečje je, da je teža izračunana kot Maksimalna ocena - Minimalna ocena za vsako postavko. Naloga s 100 točkami ima težo 100, naloga z 10 točkami ima težo 10.
A1 70/100, A2 20/80, A3 10/10, maksimum kategorije 100:
(0.7*100 + 0.25*80 + 1.0*10)/190 = 0.526 --> 52.6/100
Povprečje ocen (z dodatnimi krediti)
Aritmetično povprečje z zvijačo. Stara, sedaj nepodprta agregacijska strategija je zagotovljena tu le za prejšnjo kompatibilnost s starimi dejavnostmi.
Mediana ocen
Srednja ocena (ali povprečje srednjih dveh ocen), kjer so ocene urejene po velikosti. Prednost pred povprečjem je ta, da nanjo ne vplivajo zunanje ocene (ocene, ki so zelo oddaljene od povprečja ).
A1 70/100, A2 20/80, A3 10/10, maksimum kategorije 100:
mediana(0.7 ; 0.25 ; 1.0) = 0.7 --> 70/100
Najmanjša ocena
Rezultat je najmanjša ocena po normalizaciji. Ponavadi se uporablja v kombinaciji z Agregiraj le neprazne ocene.
A1 70/100, A2 20/80, A3 10/10, maksimum kategorije 100:
min(0.7 ; 0.25 ; 1.0) = 0.25 --> 25/100
Najvišja ocena
Rezultat je najvišja ocena po normalizaciji.
A1 70/100, A2 20/80, A3 10/10, maksimum kategorije 100:
max(0.7 ; 0.25 ; 1.0) = 1.0 --> 100/100
Modus ocen
Modus je ocena, ki se pojavi najbolj pogosto. Bolj pogosto je uporabljen za neštevilčne ocene. Prednost pred povprečjem je, da nanj ne vplivajo zunanje ocene (ocene, ki so zelo oddaljene od povprečja). Kakorkoli, ko je več kot ena bolj najbolj pogosta ocena (le eno se obdrži) ali ko so si vse ocene različne med seboj, izgubi svoj pomen.
A1 70/100, A2 35/50, A3 20/80, A4 10/10, A5 7/10 maksimum kategorije 100:
mode(0.7 ; 0.7 ; 0.25 ; 1.0 ; 0.7) = 0.7 --> 70/100
Vsota ocen
Vsota vseh številčnih ocen. Lestvice z ocenami se ignorira. To je edini tip, ki se ne pretvori v odstotke notranje (normalizacija). Maksimalna ocena povezane postavke kategorije je izračunana avtomatsko kot maksimum vseh agregiranih postavk.
A1 70/100, A2 20/80, A3 10/10:
70 + 20 + 10 = 100/190
