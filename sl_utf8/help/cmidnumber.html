ID številka

Nastavitev ID številke omogoča identifikacijo dejavnosti pri izračunih ocen. Če dejavnost ni vključena v noben izračun ocen, se lahko polje ID številka pusti prazno.

ID številka se lahko določa tudi v redovalnici preko urejanja izračuna ocene, vendar jo je moč urejati le na strani za posodabljanje same dejavnosti.
