Sled klikov

Ta tabela prikazuje podrobnosti vsakega klika vseh udeležencev med dokončanimi poskusi v določenem kvizu Hot Potatoes.

Vedite, da je to poročilo zelo obsežno in je lahko zelo dolgo, zato je namenjen prenosu in ogledu s programom, ki podpira preglednice, kot npr. Microsoft Excel.

Vsaka vrsta v tabeli vsebuje naslednje informacije:

Kateri klik?
Sledeče podrobnosti identificirajo klik, poskus, kviz, udeleženca, odsek in predmet, na katerega se sklicuje ta vrsta:

  • Ime predmeta, Številka odseka, Številka vaje, Ime vaje, Vrsta vaje, Število vprašanj
  • Polno ime udeleženca, Datum prijave, Čas prijave, Čas odjave
  • Številka poskusa, Čas začetek poskusa, Čas konca poskusa
  • Številka klika, Čas klika, Vrsta klika

Na katera vprašanja je klik vplival?
Sledeče podrobnosti se prikažejo za vprašanja (če obstajajo), na katera je kakorkoli vplival trenuten klik:

  • vprašanja, ki so bila ob trenutnem kliku označena ali pri katerih je bil podan namig ali ključ
  • status vsakega vprašanja po trenutnem kliku
    • 0 pomeni "pravilno"
    • X pomeni "napačno"
    • - pomeni "ni bilo poskusa"
  • novi odgovori, ki so bili pridobljeni ob trenutnem kliku (odgovori, ki se niso spremenili, niso prikazani)
  • vprašanja, za katera so bili ob trenutnem kliku prejeti novi odgovori
  • vprašanja, za katera so bili ob trenutnem kliku prejeti novi odgovori ali za katere je bil podan namig ali ključ

Na koliko vprašanj je klik vplival?
Sledeče podrobnosti prikazujejo celotno število vprašanja (če so), na katera je trenuten klik vplival in celotno število odgovorov, na katere se je do tega trenutka vplivalo v kvizu:

  • število vprašanj s poskusi
  • število vprašanj brez poskusov
  • število vprašanj, pri katerih je status "pravilno"
  • število vprašanj, pri katerih je status "napačno"
  • število vprašanj, pri katerih so bili podani namigi
  • število vprašanj, pri katerih so bili podani ključi
  • število vprašanj, pri katerih so bili prejeti novi odgovori
  • število vprašanj, pri katerih so bili prejeti novi odgovori ali pri katerih so je bil podan namig ali ključ

Točke
Sledeče podrobnosti se prikažejo za do sedaj pridobljene točke pri tem kliku:

  • vmesne točke
  • najvišje točke
  • točke kot odstotek
