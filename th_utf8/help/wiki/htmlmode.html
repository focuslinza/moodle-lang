HTML Mode

คุณสามารถเลือกโหมดได้ดังนี้

ไม่ใช้ HTML
โดยการหลีกเลี่ยง HTML Tags ทั้งหมด แล้วใช้รูปแบบของ WikiWords ในโหมดนี้ Filter ทั้งหมดจะทำงาน

ใช้ HTML เป็นบางแท็ก
อนุญาตให้ใช้บาง tag ของ HTML. Filters ที่ทำงานอยู่ไม่จำเป็นต้องใช้ Tag

ใช้ HTML เท่านั้น
No WikiWords, just HTML. This option allows the use of the HTML-Editor. ไม่มี WikiWords ใช้ HTML อย่างเดียว ทางเลือกนี้ทำให้คุณใช้ HTML-Editor ได้
