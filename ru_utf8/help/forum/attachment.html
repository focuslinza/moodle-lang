Вложенные файлы

Вы можете добавить только один файл (изображение, Word-документ...) к своему сообщению в форуме. Файл будет храниться на сервере вместе с Вашим сообщением.

Файл может быть любого типа, но мы советуем использовать стандартные 3-х буквенные расширения для файлов (например, .doc для Word-документов, .jpg или .png для изображений...), для того, чтобы файлы было проще идентифицировать другим пользователям.

Если Вы изменили свое сообщение и добавили новый файл, предыдущий файл будет заменен.

Если Вы отредактировали сообщение и оставили поле с прилагаемым файлом пустым, то в сообщении будет использоваться предыдущий файл.
