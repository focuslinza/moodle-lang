"Moodle XML" Format

Dies ist ein Moodle eigenes Format zur Beschreibung von Testfragen. Sie können es verwenden, um Testfragen aus einem Moodle-System zu exportieren und es in einer anderen Moodle-Installation zu importieren. Die Struktur der Daten erkennen Sie am Besten wenn sie eine Fragekategorie exportieren und die XML-Datei öffnen. Das Moodle-XML-Format ermöglicht den Ex- und Import von eingebundenen Bldern.

Optionale Fragetyp-Plugins erweitern dieses Format.

Wechsel zwischen Kategorien

Tragen Sie eine gesonderte Dummy-Frage in die XML-Datei ein, um einen Kategorienpfad anzulegen in den die nachfolgenden Fragen eingefügt werden. Falls die Kategorien noch nicht existieren, werden sie neu angelegt. Die Option from file: (aus Datei) muss auf der Importseite aktiviert werden, sonst wird der Code komplett ignoriert. In einer XML-Dtaei können beliebig viele Kategorien angelegt werden.

Beispiel für die Beschreibung:

<question type="category">
    <category>
        tom/dick/harry
    </category>
</question>
