HTML-Seite

Diese Materialart ermöglicht es, eine vollständige Webseite innerhalb von Moodle anzulegen. Dies geht besonders einfach, wenn Sie den HTML-Editor (WYSIWYG) von Moodle verwenden.

Die Seite wird in der Datenbank und nicht als Datei abgespeichert. Sie haben dadurch eine Vielzahl von Optionen wie die Nutzung von HTML oder Javascript.
