Dateien in einem Frame einbinden

Diese Option erlaubt das Anzeigen einer Datei in einem Frame. Die Moodle-Kursnavigation bleibt dann im oberen Teil sichtbar.

Diese Option ist für Medientypen wie Filme Audiodateien, Flash-Dateien nicht nötig. Wenn sie genutzt wird, erscheint die Datei in einer navigierbaren Seite.

Der Gebrauch von Frames widerspricht den Anforderungen an barrierefreie Seitengestaltung. Bei der Nutzung von Screenreadern (Option im persönliche Profil) werden Frames ignoriert.
