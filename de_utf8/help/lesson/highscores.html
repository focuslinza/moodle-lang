Bestenliste anzeigen

Mit dieser Option können Sie eine BEsteliste für diese Lektion aktivieren. Teilnehmer/-innen die eine hohe Punktzahl erreicht haben, können einen Namen eingeben, mit dem Sie auf dieser Liste stehen wollen. Es gibt einen 'böse' Worte Filter, mit dem diese Namen überprüft werden.

Die Bestenliste wird nicht gezeigt, wenn die Option 'Übungslektion' aktiviert ist.
