Maximale Antworten/Verzweigungen

Dieser Wert legt fest, wieviele Antwortvorgaben es bei einer Frage maximal gibt. Die Grundeinstellung ist 4. Wenn Sie z.B. in Ihrer Lektion nur Richtig/Falsch-Fragen verwenden wollen, stellen Sie den Wert auf 2.

Dieser Wert legt auch die maximale Anzahl der Verzweigungen auf einer Verzweigungsseite fest.

Es ist ungefährlich, diesen Wert in einer Lektion mit bestehendem Inhalt zu ändern.

Tipp: Erstellen Sie zuerst den Inhalt und die Fragen Ihrer Lektion, und legen Sie dann diesen Wert fest. Wenn Sie eine Frage mit vielen Antwortmöglichkeiten oder eine Verzweigungsseite mit vielen Verzweigungen hinzufügen wollen, werden Sie zwangsläufig diesen Wert kurzfristig erhöhen müssen. Nachdem Sie diese 'ungewöhnliche' Frage hinzugefügt haben, können Sie den Wert wieder auf einen 'normalen' kleinen Wert zurücksetzen.
