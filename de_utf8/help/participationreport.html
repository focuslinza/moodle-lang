Aktivitäten-Bericht

Aktivitäten-Modul - Wählen Sie ein Aktivitäten-Modul aus, das im Bericht dargestellt werden soll (Forum, Test, Lektion usw.). Beachten Sie, dass nur die Aktivitäten-Module zur Auswahl stehen, die im aktuellen Kurs auch tatsächlich verwendet werden.

Rückschau - Wählen Sie den Zeitraum aus, der im Bericht erfasst werden soll.

Nur anzeigen- Wählen Sie, ob Teilnehmer/innen- oder Trainer/innen-Aktivitäten angezeigt werden sollen.

Aktivitäten anzeigen - Wählen Sie, ob Zugriffe (auf Aktivitäten), Beiträge (zu Foren) oder beides (alle Aktionen) angezeigt werden sollen.

Nachdem Sie die oben genannten Einstellungen vorgenommen haben, klicken Sie auf Start.

Nun wird eine Liste aller im Kurs vorhandenen Instanzen des gewählten Aktivitäten-Moduls erstellt. Wählen Sie eine Instanz, die Sie sich weiter ansehen wollen und klicken Sie auf Start.

Weitere Hinweise zu Zugriffen und Beiträgen finden Sie auf dem Bildschirm. Zum Beispiel:
Forum Zugriffe: Zeige Diskussion, Suche, Forum, Foren, Abonennten usw.
Forum Beitrag: Neues Diskussionsthema hinzufügen, Antworten, Diskussionsthema löschen, Diskussionsbeitrag löschen, Diskussionsthema verschieben, Diskussionsbeitrag ausschneiden, Diskussionsbeitrag bearbeiten usw.

Nutzer/innen auf die das Nur anzeigen Kriterium zutrifft (Teilnehmer/innen oder Trainer/innen) werden in einer Tabelle angezeigt (mit Name / ID-Nummer und ob und wie häufig sie die gewählte Aktion ausgeführt haben).

Sie können diesen Bericht nun auch als Sammel-E-Mail an ausgewählte Nutzer/innen versenden. Wählen Sie dazu die entsprechenden Nutzer/innen aus der Liste aus, wählen Sie Mitteilung hinzufügen / senden vom Mit ausgewählten Nutzer/innen ... Dropdown-Menü und klicken Sie auf OK.
