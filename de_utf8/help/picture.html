Hochladen eines Porträts

Sie können Ihr Porträt zum Server hochladen; dieses Bild wird an verschiedenen Stellen gezeigt.

Aus diesem Grund sollten Sie möglichst gute Bilder zu verwenden, z.B. eine Nahaufnahme Ihres Gesichtes. Aber Sie können natürlich auch jedes andere Bild verwenden.

Dieses Bild muss im JPG- oder PNG-Format vorliegen, d.h. die Dateiendung .jpg oder .png besitzen.

Eine Bilddatei können Sie mit folgenden vier Methoden erhalten:

  1. Mit einer Digitalkamera: Damit erhalten sie Ihre Bilder höchstwahrscheinlich schon im notwendigen Format.
  2. Mit einem Scanner: Sie können Papierbilder einscannen; der Scan muss dann im JPG- oder PNG-Format abgespeichert werden.
  3. Sollten Sie künsterisch veranlagt sein, können Sie mit einem entsprechenden Programm ein Bild zeichnen oder malen.
  4. Letztendlich können Sie aber auch ein Bild aus dem Netz entnehmen und einstellen. (Z.B. mit Hilfe der Google-Bildsuche www.google.de) Dabei sollten Sie selbstverständlich prüfen, ob das Bild urheberrechtlich geschützt ist.
.

Um das Bild zu hochladen, klicken Sie den Button "Durchsuchen" auf dieser Seite an und wählen dann das Bild von Ihrer Festplatte aus.

ANMERKUNG: Überprüfen Sie, ob die Datei nicht größer ist als die maximal zulässige Größe für den Upload, sonst wird sie nicht hochgeladen.

Dann klicken Sie "Aktualisieren" am unteren Ende der Seite - die Datei wird dabei zu einem Quadrat im Seitenverhältnis 100x100 Pixeln verkleinert.

Wenn Sie zurück gehen auf Ihre Profilseite und das neue Bild noch nicht erscheint, klicken Sie bitte auf "Aktualisieren" (F5-Taste) in Ihrem Browser. Manchmal ist es auch notwendig, das Cache-Verzeichnis des Browsers zu leeren, damit das neue Bild erscheint.
