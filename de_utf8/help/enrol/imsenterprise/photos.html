Fotos

Beim Erstellen eines neuen Nutzerzugangs mittels einer IMS-Datei kann ein Foto des Nutzers angegeben werden. Das Foto wird normalerweise in Form einer URL auf die Bilddatei übergeben.

Wenn in der IMS-Datei für einen Nutzer ein <photo> Element angegeben ist, kann Moodle diese Bilddatei auf den Server laden.

HINWEIS: Der Import einer großen Zahl solcher Bilddateien stellt hohe Anforderungen an die Leistung des Server. Jedes Bild wird heruntergeladen, umgewandelt und gespeichert. Es ist NICHT empfehlenswert, auf diesem Wege eine große Zahl von Bildern zu importieren.
