Fotos

Beim Erstellen eines Nutzer-Accounts auf der Grundlage von Enterprisedaten kann ein Fotos des Nutzers übernommen werden. Dieser wird in Form einer URL auf die Bilddatei übergeben.

Moodle kann diese Blddatei in den moodle Nutzerbilder-Bereich übernehmen.

HINWEIS: Der Import einer großen Zahl solcher Bilddateien stellt eine hohe Anforderung an den Server da. Jedes Bild wird heruntergeladen, umgewandelt und gespeichert. Es ist nicht empfehlenswert, auf diesem Wege eine große Zahl von Bildern zu importieren.
