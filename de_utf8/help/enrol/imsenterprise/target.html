Targets

Ein IMS Enterprise Datensatz kann für viele verschiedene Zwecke ("targets") - verschiedene Lernmanagementsysteme oder andere Anwendungen genutzt werden. In der Enterprise Datei kann festgelegt werden dass Funktionen für eine oder mehrere Funktionen bereitgestellt werden. Sie werden im <target> Tag definiert innerhalb des <properties> Tag.

Zumeist werden Sie sich damit gar nicht befassen müssen. Lassen Sie die Einstellung leer und moodle verarbeitet die Daten unabhängig ob ein Target Wert definiert ist oder nicht. Andernfalls tragen Sie den exakten Wert, der im Targetfeld definiert ist und der von moodle berücksichtigt werden soll hier ein.
