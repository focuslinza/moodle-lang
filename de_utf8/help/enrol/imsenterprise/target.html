Targets

Ein IMS Enterprise Datensatz kann für viele verschiedene Zwecke ("targets") - verschiedene Lernmanagementsysteme oder andere Anwendungen - genutzt werden. In der IMS-Datei kann festgelegt werden, dass Funktionen für eine oder mehrere Zwecke ("targets") bereitgestellt werden. Sie werden im <target> Element innerhalb eines <properties> Elements definiert.

In den meisten Fällen werden Sie sich damit gar nicht befassen müssen. Lassen Sie die Einstellung deaktiviert, und Moodle verarbeitet die Daten unabhängig davon, ob ein <target> Element definiert ist oder nicht. Andernfalls tragen Sie den exakten Wert, der im <target> Element definiert ist und der von Moodle berücksichtigt werden soll, hier ein.
