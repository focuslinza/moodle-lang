reCAPTCHA - Captcha-Schutz für die Registrierung

Beschreibung

CAPTCHA ist ein Programm, das erkennt, ob die Eingaben von einem Menschen oder einem Computerprogramm vorgenommen werden. Damit wird verhindert, dass Bot-Programme sich Zugang zu einer Anwendung verschaffen und z.B. Werbespam verbreiten. Dazu wird ein zufällig generierter Buchstabencode grafisch dargestellt und muss als Bestätigung abgetippt werden. Computerprogramme können auch mit Texterkennungslogik diese Codes in der Regel nicht identifizieren. Das hier verwandte Captcha-Programm bietet alternativ zur Bilddarstellung eine auditive Darstellung des Code an, um auch sehbehinderten Nutzern den Zugang zu ermöglichen.

Anleitung

Geben Sie die Zeichen in dem Kasten in das Eingabefeld ein. Beachten Sie die Reihenfolge und Leerzeichen.

Wenn Sie die Zeichen nicht sicher erkennen, können Sie diese erraten oder einen neuen Code mit Klick auf "Ein anderes CAPTCHA anzeigen" anfordern.

Falls Sie die Zeichen nicht lesen können, können Sie sich einen Code vorlesen lassen. Klicken Sie auf den "Audio CAPTCHA" Link und tippen Sie den gehörten Code ein.
