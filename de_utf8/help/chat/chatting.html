Den Chat verwenden

Das Chat-Modul enthält einige Features, die das Chatten vereinfachen.

Smileys
Alle Smileys (emoticons), die in Moodle benutzt werden können, lassen sich auch im Chat verwenden und werden korrekt angezeigt. Tastatur-Smileys werden durch Bild-Smileys ersetzt, z.B., :-) =
Links
Internetadressen werden automatisch in Links umgewandelt.
Personalisierung
Sie können eine Zeile mit "/me" oder ":" beginnen. Dieser Text wird durch Ihren Namen ersetzt. Wenn Ihr Name beispielsweise Jürgen ist und Sie geben ein: ":lacht!" oder "/me lacht!", dann wird jeder sehen: "Jürgen lacht!"
Tonsignale
Sie können Tonsignale (beeps) an andere Teilnehmer/innen senden, wenn Sie den Signal-Link oben in der Nähe Ihres Namens anklicken. Wenn Sie "beep all" schreiben, senden Sie den Ton an alle Teilnehmer/innen im Chat.
HTML
Sie können HTML-Code verwenden, um in Ihren Text Bilder einzufügen, Farben zu ändern oder Sounds abzuspielen.
