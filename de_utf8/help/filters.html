Filterkonfiguration

In der Filterkonfiguration legen Sie fest, welche Filter bei der Anzeige von Seiteninhalten und in welcher Reihenfolge angewandt werden. Für einige Filter stehen hier zusätzliche Einstellungsoptionen zur Verfügung. Die Filter beeinflussen die Darstellung der meisten Textseiten, kurz bevor sie an den Browser der Nutzer/innen übergeben werden.

Filter sind erweiterbare Ressourcen (Plugins). Unter http://download.moodle.org/modules/filters.php finden Sie eine Reihe weiterer Filter für Ihr System.

Wählen Sie Filter aus, indem Sie das Augensymbol anklicken und sie damit sichtbar machen. Wenn Sie mehr als einen Filter aktiviert haben, können Sie die Reihenfolge der Filter mit den Aufwärts- und Abwärtspfeilen festlegen.

Algebraisches Zeichensystem

    Dieser Filter ermöglicht es, in Verbindung mit TeX (s.u.) Formeln im TeX-Format darzustellen.

TeX-Zeichensatz

    TeX-Beschreibungen werden vorwiegend zur Darstellung mathematischer Formeln innerhalb von Moodle verwendet. Z.B. $$ sqrt(a+b) $$ (die $$ sind die Kennzeichnung von Anfang und Ende des Formats).

    Moodle verwendet externe Programme, um aus diesen Zeichen ein Bild zu erzeugen. Der Filter sucht zunächst einen LaTeX-Umwandler ( wie Ghostscript). Wenn dieser nicht zur Verfügung steht, verwendet er ein einfacheres Programm miktex, das in Moodle integriert ist. LaTex ermöglicht es, alle Funktionen zu verwenden, miktex unterstützt nur die mathematische Zeichendarstellung.

    Ghostscript ist für Windows erhältlich, ebenso LaTeX (eine gängige Variante von LaTeX für Windows nennt sich MikTeX).

    Auf der Einstellungsseite können Sie die Pfade zu den LaTeX- und Ghostscript-Programmen beschreiben, zusammen mit LaTeX-Vorzeichen.

Automatische Verlinkung zur Datenbank

    Der Filter durchsucht Texte nach Begriffen in der Datenbank (bzw. in Datenbanken). Gefundene Begriffe werden hervorgehoben und mit einem Link zum betreffenden Datenbank-Eintrag versehen.

Automatische Verlinkung zum Glossar

    Der Filter durchsucht Texte nach Schlüsselbegriffen im Glossar (bzw. in Glossaren) und erzeugt jeweils einen Link auf die Begriffserklärung.

Automatische Verlinkung zu den Namen der Arbeitsmaterialien

    Der Filter durchsucht die Texte nach den Titeln der Arbeitsmaterialien im Kurs. Gefundene Begriffe werden hervorgehoben und mit einem Link zum betreffenden Arbeitsmaterial versehen.

Automatische Verlinkung zu Wiki-Seiten

    Der Filter durchsucht Texte nach den Namen von Wiki-Seiten und verlinkt dorthin.

Automatische Verlinkung zu den Aktivitätsnamen

    Die gleiche Funktion wie oben für die Namen von Lernaktivitäten.

Wortzensur

    Der Filter "Wortzensur" durchsucht die Texte nach Tabu-Wörtern aus einer vordefinierten Liste. Diese steht im Sprachpaket für jede Sprache separat zur Verfügung. Auf der Einstellungsseite können Sie Ihre eigene Liste zusammenstellen. Nutzer/innen sehen diese Begriffe als schwarzen Block. Der Originalbegriff wird per Mouseover angezeigt.

Spamschutz

    Der Spamschutz-Filter durchsucht den Text nach E-Mail-Adressen. Gefundene E-Mail-Adressen werden so verborgen, dass Programme, die das Internet systematisch nach E-Mail-Adressen durchforsten, diese nicht erkennen. Diese Funktion reduziert das Spam-Risiko auf Moodle-Seiten, die öffentlich zugänglich sind.

Multimedia-Plugins

    Der Filter durchsucht die Texte nach den Namen von Multimedia-Ressourcen im Kurs. Gefundene Begriffe werden hervorgehoben und mit einem Link zur betreffenden Ressource versehen.

Mehrsprachiger Inhalt

    Dieser Filter erlaubt es, Texte in verschiedenen Sprachen anzuzeigen. Es wird die Sprache angezeigt, die als bevorzugte Sprache im Nutzerprofil eingetragen ist. Erstellen Sie für die Verwendung dieses Filters ihre Texte zunächst in mehreren Sprachen im gleichen Dokument. Danach schließen Sie jede Sprache mit dem folgenden tag ein: 

          <lang lang="XX">Ihr_Inhalt</lang>

Tidy

    Der Tidy-Filter prüft Texte auf HTML-Code-Elemente. Gefundene HTML-Codes werden geprüft und so ergänzt, dass sie gültigen HTML-Code enthalten. Die Tidy-Funktion muss für diesen Filter in Ihrem PHP-System aktiviert sein.
