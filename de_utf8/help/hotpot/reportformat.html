Bericht-Format

Sie können festlegen, in welchem Format der Bericht für den Hot Potatoes Test angezeigt wird.

Format

HTML
Der Bericht wird im HTML-Format generiert, das für die Anzeige in einem Browser geeignet ist.

Excel
Der Bericht wird als ".xls" Datei generiert, die in einem Tabellenkalkulationsprogramm (z.B. Microsoft Excel) geöffnet werden kann.

Text
Der Bericht wird als Textdatei generiert, die in einem Editor geöffnet werden kann.

Codierung

Es kann sein, dass Ihr Tabellenkalkulationsprogramm eine bestimmte Codierung erfordert. Sie können die benötigte Codierung aus dem entsprechenden Auswahlmenü auswählen.
Z.B. erfordert Microsoft Excel eine "SJIS" Codierung für Japanisch.
Die Liste der möglichen Codierungen wird von der Administration in den Einstellungen für das "Hot Potatoes Test"-Modul festgelegt.

Daten umbrechen

Ja
Daten, die aus einer langen Zeichenkette bestehen, werden umgebrochen, damit sie in die Tabellenspalten passen. Dadurch können Zeilen sehr "hoch" werden.

Nein
Daten werden nicht umgebrochen.

Legende anzeigen

Ja
In den Berichtstypen "Ausführliche Statistik" und "Ausführlicher Zugriffsbericht" werden Fragen und Antworten durch Buchstaben repräsentiert. Es wird eine Legende generiert, die anzeigt, welche Buchstaben für welche Fragen bzw. Antworten stehen.

Nein
Fragen und Antworten erscheinen im Bericht in voller Länge. Es wird keine Legende generiert.
