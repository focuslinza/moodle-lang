Moodle XML-Format-Dateien importieren

Kategorienwechel

Sie können spezielle Dummy-Fragen in eine XML-Datei einfügen, um die Kategorie festzulegen, in die die nachfolgenden Fragen eingeordnet werden sollen. Falls eine oder mehrere Kategorien nicht existieren, so werden sie neu angelegt. Beachten Sie, dass die Option aus der Datei: in den Einstellungen markiert sein muss. Andernfalls wird der Code vollständig ignoriert. Sie können in der XML-Datei beliebig oft zwischen den Kategorien wechseln.

Beispiel für das Format:

<question type="category">
    <category>
        tom/dick/harry
    </category>
</question>
