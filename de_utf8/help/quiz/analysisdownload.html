Ergebnisanalysen downloaden

Sie können die auf dem Bildschirm dargestellten Tabellenübersichten herunterladen, um weitere Auswertungen vorzunehmen. Wählen Sie zwischen drei Dateiformaten.
In jedem Fall werden die Daten als Tabelle gespeichert, wie auf dem Bildschirm. Auch wenn die Tabelle auf dem Bildschirm in mehrere Seiten aufgeteilt ist, wird sie in der Datei als eine Tabelle zusammengefasst.

Excel-Tabellenformat:

Sie erhalten ein Tabellendokument mit der Dateiendung .xls.

OpenOffice-Writer-Format:

Sie erhalten eine Tabellendatei als Open-Office-Datei mit der Endung .sxw.
Dieses Format steht nur zur Verfügung, wenn die PHP-Writer-Library im Verzeichnis /moodle/lib installiert wurde. Fragen Sie den/die Administrator/in.

Textformat

Die Daten werden in einer Textdatei gespeichert. Für jede Zeile der Tabelle wird eine Textzeile angelegt. Die Werte werden durch Tabs getrennt.
