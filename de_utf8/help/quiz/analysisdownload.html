Analysen downloaden

Sie können die Daten der Tabellen auf dem Bildschirm herunterladen, um damit weitere Auswertungen vorzunehmen. Wählen Sie zwischen den folgenden Dateiformaten.
In jedem Fall werden die Daten als Tabelle, wie auf dem Bildschirm gezeigt, gespeichert. Wenn die Tabelle auf dem Bildschirm in mehrere Seiten aufgeteilt ist, wird sie in der Datei als eine Tabelle dargestellt.

Excel-Tabellenformat:

.xls-Tabellendokument.

OpenOffice-Writer-Format:

Tabellendatei als OpenOffice-.sxw-Datei .
Dieses Format steht nur zur Verfügung, wenn die PHP-Writer-Library im Verzeichnis /moodle/lib installiert wurde. Fragen Sie den/die Administrator/in.

Textformat

Die Daten werden in einer Textdatei gspeichert. Für jede Zeile der Tabelle wird eine Textzeile angelegt. Die Werte werden durch Tabs getrennt.
