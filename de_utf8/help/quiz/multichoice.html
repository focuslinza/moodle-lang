Multiple-Choice-Fragen

Um auf eine Frage (die auch ein Bild enthalten kann) einzugehen, wählen die Teilnehmer/innen aus verschiedenen vorgegebenen Antworten aus. Es gibt zwei Arten von Multiple-Choice-Fragen, und zwar solche mit einer richtigen Antwort und solche mit mehrere richtigen Antworten.

Bei Fragen mit einer Antwort darf genau eine Antwort ausgewählt werden. Normalerweise sollten alle Bewertungen für solch eine Frage positiv sein.

Bei Fragen mit mehreren Antworten dürfen eine oder auch mehrere Antworten ausgewählt werden - jede Antwort könnte eine positive oder negative Bewertung haben, so dass die Auswahl ALLER Möglichkeiten nicht unbedingt in einer guten Bewertung endet. Wenn die Bewertung negativ ist, wird diese Frage mit 0 gewertet.
Achtung: Es ist möglich, Fragen zu erzeugen, deren Wertungsergebnis über 100% liegen könnte.

Zu guter Letzt sollte jede Antwort ein Feedback enthalten - dieses Feedback wird den Teilnehmer/innen neben jeder gegebenen Antwort angezeigt (wenn der Test selber so eingestellt ist, dass er ein Feedback anzeigt).
