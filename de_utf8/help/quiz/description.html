Beschreibung

Dieser Fragetyp ist nicht wirklich eine Frage.

Hier wird nur ein Text angezeigt, der keinerlei Antworten erfordert.

Dieser Typ kann zum Beispiel benutzt werden, um einen beschreibenden Text anzuzeigen, der sich auf eine folgende Fragengruppe bezieht.

Beachten Sie, dass, sobald Sie Fragen "mischen", auch diese Erklärungen gemischt werden.
