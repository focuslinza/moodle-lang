Beschreibung

Dieser Fragentyp enthält keine wirkliche Frage, denn er bringt lediglich einen Text zur Anzeige, der keinerlei Antworten erwartet.

Dieser Fragentyp kann benutzt werden, um beispielsweise einen erklärenden Text anzuzeigen, der sich auf eine nachfolgende Fragengruppe bezieht.

Anmerkung: Sobald Sie die Fragen "mischen", werden auch diese Erklärungen gemischt, was unter Umständen unerwünscht oder vielleicht sogar absolut unpassend wäre.
