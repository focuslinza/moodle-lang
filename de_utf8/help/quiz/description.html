Beschreibung

Dieser Fragentyp enthält keine Frage, er bringt lediglich einen Text zur Anzeige, der keinerlei Antworten verlangt.

Dieser Typ kann zum Beispiel benutzt werden, um einen erklärenden Text anzuzeigen, der sich auf eine folgende Fragengruppe bezieht.

Bitte beachten Sie: Sobald Sie Fragen "mischen", werden auch diese Erklärungen gemischt.
