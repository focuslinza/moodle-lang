Fragekategorien

Schnell sammelt sich eine große Anzahl von Fragen in Tests an. Damit diese nicht in einer einzigen langen Liste verwaltet werden müssen, können Kategorien für verschiedene Fragen angelegt werden.

Jede Kategorie erhält einen Namen und eine Kurzbeschreibung.

Jede Kategorie kann zudem veröffentlicht werden. Damit ist die Kategorie (mit sämtlichen enthaltenen Fragen) in allen Kursen dieser Installation nutzbar.

Feldbeschreibung

Hauptkategorie: Die Kategorie, der diese Kategorie untergeordnet werden soll. Wenn keine anderen Kategorien vorhanden sind, wird die neue Kategorie unterhalb der obersten Ebene eingefügt.

Kategorie: Name der Kategorie.

Kategorienbeschreibung: Kurzbeschreibung des Inhalts der Kategorie.

Veröffentlichen: Sollen die Fragen aus dieser Kategorie auch in anderen Kursen zur Verfügung stehen? Wenn Sie in Ihren Kursen auch Teilnehmer/innen Trainerrechte zuweisen, haben auch diese Einblick in Ihre Testfragen.
