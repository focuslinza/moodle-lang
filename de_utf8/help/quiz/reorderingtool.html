Zeilennummern anzeigen

Diese Option fügt Felder mit Zeilennummern am linken Rand der Fragenliste ein. Sie werden in Zehnerschritten vergeben, damit Sie Platz für das nachträgliche Einfügen von Fragen haben. Sie können die Zeilennummern ändern, falls Sie die Fragen umsortieren möchten. Nach dem Speichern der Einträge werden die Fragen in der Reihenfolge der von Ihnen neu vergebenen Zahlen sortiert. Beachten Sie in diesem Zusammenhang die Option "Fragen mischen?" in den Test-Einstellungen. Sie sollte auf "Nein" gestellt sein, wenn Sie Einfluss auf die Reihenfolge nehmen wollen.

Sie müssen nicht unbedingt ganzzahlige Nummern eintragen. Wenn Sie Dezimalzahlen (mit einem Dezimalpunkt) eintragen, wird auch das bei der Sortierung berücksichtigt.

Seitenumbrüche erhalten ebenfalls eine Zeilennummer, damit Sie auch den Seitenumbruch festlegen können. Wenn Sie die Funktion "Seitenumbrüche anzeigen" deaktiviert haben, werden statt der Seitenumbrüche Lücken bei der Zeilennummerierung angezeigt.
