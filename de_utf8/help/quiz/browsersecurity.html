Browsersicherheit

Diese Funktion erlaubt es, bestimmte Brwoserfunktionen für Teilnehmer/innen zu deaktivieren. Damit kann das Nachschlagen auf anderen Seiten verhindert werden. In manchen Situationen kann dies ein Betragsversuch sein, in anderen kann eine solche Ooption jedoch durchaus gewollt sein.

Bedenken Sie, dass es sich hierbei nicht primär um eine technische Problematik und eine technische Lösung handelt. Schon immer haben Prüfungsteilnehmer Wege gesucht und gefunden, um Prüfungsstoff zu kopieren und einzufügen. Im Zeitalter von Computer und Internet haben sich hierfür neue Wege ergeben. Zugleich haben Trainer/innen jedoch auch andere Möglichkeiten gefunden, diese aufzudecken. Die hier offerierte Lösung ist nicht vollständig sicher, sie erschwert aber den Betrugsversuch, macht zugleich jedoch auch bestimmte Funktionen weniger einfach.

Es gibt eine ganze Reihe weiterer Möglichkeiten den Betrug bei Tests zu erschweren:

  • Verwenden Sie eine möglichst große Testfragendatenbank und die Funktion der Zufallsfragenauswahl. Verschiedene Teilnehmer/innen sehen dann ähnliche aber unterschiedliche Fragen.
  • Verwenden Sie Funktion, um Antwortoptionen in zufälliger Reihenfolge anzeigen zu lassen.
  • Stellen Sie Fragen bei denen nicht einfach Fakten abgefragt werden, sondern nachgedacht werden muß, um eine Antwort zu finden.

    • Mit diesen Vorüberlegungen im Hinterkopf beschreiben wir nun die Funktionen.

    Keine

    Keine Sicherheitsfunktionen werden aktiviert.

    Vollbildmodus mit einigen JavaScript Sicherheitsbeschränkungen

    • Der Teststart erfordert,dass Javascript im Browser des Teilnehmers aktiv ist.
    • Der Test öffnet sich im Vollbildmodus und überdeckt alle anderen Fenster und Navigationsmöglichkeiten.
    • Copy and paste ist zumindest eingeschränkt.

    Zusatzfunktionen mit dem Safe Exam Browser

    Diese Funktion erfordert zuvor eine Aktivierung durch den Administrator.

    Der Safe Exam Browser ist ein speziell angepasster Browser und muß zunächst auf dem Teilnehmer-PC installiert worden sein. Die Einschränkungen ähneln den Funktionen des Vollbildschirms (s.o.). Da es sich um einen eigenen Browser handelt sind die Funktionen jedoch weitaus wirksamer. Wenn Sie die Funktion verwenden:

    • Teilnehmer/innen können nur mit dem Safe Exam Browser auf den Test zugreifen.
    • Der Test öffnet sich im Vollbildschirmmodus ohne Navigationsoption.
    • Das Fenster kann nur geschlossen werden wenn der Test abgeschlossen iwrd.
    • Tastaturkürztel wie Ctrl+Alt+Del, Alt+F4, F1, Ctrl+P, Printscreen, sind unter Windows deaktiviert.
    • Copy and paste sowie das Kontextmenu (rechte Maustaste) sind deaktiviert.
    • Der Wechsel zu anderen Programmen wird unterbunden.
    • Der Aufruf anderer Webseiten wird unterbunden.
