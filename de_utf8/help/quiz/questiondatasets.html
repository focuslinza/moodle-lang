Datasets für Fragen anlegen

Ein Dataset ist die Sammlung von Werten, die bei der Erstellung von Fragen genutzt wird, um eine Variable zu füllen (wie bei der berechneten Frage).

Erstellen Sie ein "privates" Dataset nur für eine einzelne Frage, oder ein "wiederverwertbares" Dataset, das in allen Fragen einer Kateorie verwandt werden kann.

Beim Erstellen einer Frage, die ein Dataset verwendet sehen Sie zwei Bildschirmseiten.

Auf der ersten Seite legen Sie die Datasets an, die danach mit Variablen gefüllt werden.

Auf der zweiten Seite werden die Werte in die Datasets eingetragen. Aus dieserSeite werden die Werte für die Fragen genommen, die den Teilnehmer/innen angezeigt werden.
