Freitext-Fragen

Als Antwort auf eine solche Frage (die auch eine Abbildung enthalten kann) wird ein frei formulierter Text erwartet. Drei Felder können beim Anlegen einer Frage ausgefüllt werden: der Titel der Frage, der Wortlaut der Frage selbst und ein Feedback-Text.

Für die Freitext-Frage erhalten die Teilnehmer/innen erst dann eine Bewertung, wenn sie von Trainer/innen manuell bewertet wurde. Dabei steht ein Textfeld für ein individuelles Feedback und die Zuweisung einer Wertung zur Verfügung.

Normalerweise können die Teilnehmer/innen für die Antwort den Rich-Text-Editor verwenden. Wenn allerdings mehr als eine Freitext-Frage auf der Seite vorhanden ist, wird der Editor nur für die erste Frage dieses Typs bereitgestellt.
