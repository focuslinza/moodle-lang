Überblick Download

Vielleicht möchten Sie die Ergebnisdaten, die auf dem Bildschirm angezeigt werden, herunterladen. Sie können dann zwischen zwei Dateiformaten wählen.
In jedem Fall werden die Daten als Tabelle dargestellt, wie auf dem Bildschirm. Auch wenn sich die Tabelle über mehrere Bildschirmseiten erstreckt, wird sie in einer einzigen Datei zusammengefasst.

Excel-Format:

Sie erhalten eine .xls-Datei.

Text-Format

Die Daten werden als Textdatei gespeichert: eine Zeile für jede Tabellenzeile, getrennt von Tabulatorzeichen.
