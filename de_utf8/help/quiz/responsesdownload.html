Antworten downloaden

Sie können die Ergebnisdaten in zwei unterschiedlichen Formaten herunterladen.
Die Daten werden ohne die Bilder der Teilnehmer/innen herunter geladen.
Wenn die Tabelle am Bildschirm über mehrere Seiten verteilt angezeigt wird, wird sie in der Datei als eine einzige Tabelle angezeigt.

Excel Datei:

Sie erhalten eine .xls Datei.

Textformat

Daten werden als einfache Textdatei gespeichert. Für jede Tabellenzeile wird eine Textzeile erstellt. Die Werte werden durch einen Tabulatorschritt getrennt.
