Antworten downloaden

Sie können die Ergebnisdaten in zwei unterschiedlichen Formaten herunterladen.
Die Daten werden als Tabelle wie auf dem Bildschirm, aber ohne die Bilder der Teilnehmer/innen, heruntergeladen.
Auch wenn die Tabelle sich auf dem Bildschirm über mehrere Seiten erstreckt, wird sie in der Datei komplett heruntergeladen.

Excel-Datei:

Sie erhalten eine .xls-Datei.

Textformat:

Die Daten werden als einfache Textdatei gespeichert. Für jede Tabellenzeile wird eine Textzeile erstellt. Die Werte werden durch einen Tabulatorschritt getrennt.
