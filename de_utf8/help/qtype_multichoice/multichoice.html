Multiple-Choice-Fragen

Als Antwort auf eine Frage stehen verschiedene Antwortmöglichkeiten zur Verfügung. Die/der Teilnehmer/in kann eine (Single-Choice) oder mehrere (Multiple-Coice) der Antworten ankreuzen. Die Frage selber kann durch ein Bild oder eine Grafik ergänzt werden.

Single-Choice-Fragen erlauben es, genau eine Antwort auszuwählen.

Multiple-Choice erlaubt der/dem Teilnehmer/in mehrere Optionen anzukreuzen. Die einzelnen Antwortoptionen können mit Bewertungen von -100 % bis + 100% versehen werden. Damit wird verhindert, dass beim Ankreuzen aller Antwortoptionen die Höchstpunktzahl erreicht wird. Die Summe aller positive Bewertungen muss 100 % sein.

Zusätzlich sollten Testfragen Feedbackinforamtionen enthalten. Dieses wird den Teilnehmer/innen nach dem Test angezeigt. Zu welchem Zeitpunkt wird in den Einstellungen zum Test festgelegt. Feedback kann für jede einzelne Antwortoption, für falsche/richtige/teilrichtige Antworten oder generell zur Frage angelegt werden.
