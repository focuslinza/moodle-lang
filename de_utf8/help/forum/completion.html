Forum als Arbeitsauftrag beenden (Teilnehmersicht)

Im Form können drei verschiedene Optionen gewählt werden, die bei der Einschätzung, ob die Bearbeitung abgeschlossen ist beachtet werden.

  • Nutzer/in muss Diskussionsthemen beginnen oder vorhandene beantworten: Der Nutzer muss eine festgelegte Zahl von eigenen Beiträgen verfassen. Es wird nicht unterschieden, ob es sich um Antworten oder neue Themen handelt.
  • Nutzer muss neue Themen anlegen: Der Nutzer muss eine festgelegte Anzahl neuer Themen im Forum anlegen.
  • Nutzer muss Antworten verfassen: Der Nutzer muss eine festgelegte Anzahl von Antworten im Forum zu bestehenden Themen verfassen.

Falls gewünscht, können mehrere Optionen ausgewählt werden. Dann müssen alle Anforderungen erfüllt sein, damit die Forenaktivität als erfüllt gewertet wird. Beispiel: wenn Sie eingeben '2 neue Themen' und '10 Themen oder Antworten' dann wird jede Kombination entsprechend gewertet die dies beinhaltet (z.B. 10 neue Themen; 2 neue Themen und 8 Antworten oder irgendetwas dazwischen)
