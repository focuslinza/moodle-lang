Anhang

Sie haben die Option, EINE Datei als Anhang zu einem Glossareintrag hinzuzufügen. Die Datei wird auf den Server geladen und mit dem Eintrag gespeichert.

Dies ermöglicht, ein Bild oder eine Textdatei im Glossar zur Verfügung zu stellen.

Die Datei kann ein beliebiges Dateiformat haben. Achten Sie darauf, dass Sie die Internet-üblichen 3-Buchstaben-Dateiendungen verwenden (z.B. .doc für Word-Dokumente, .jpg oder .png für Bilder usw.). Damit erleichtern Sie anderen Nutzer/innen das Herunterladen bzw. Anzeigen der Anhänge in ihrem Browser.

Wenn Sie einen vorhandenen Eintrag überarbeiten und einen neuen Dateianhang hochladen, steht die ursprünglich angehängte Datei nicht mehr zur Verfügung.

Wenn Sie einen Eintrag mit Dateianhang überarbeiten und das Feld für den Dateianhang leer lassen, bleibt die ursprünglich angehängte Datei erhalten.
