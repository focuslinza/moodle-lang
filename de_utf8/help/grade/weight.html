Kategorien gewichten

Sie können hier die Gewichtung für eine Kategorie vornehmen, die niedrigsten X Resultate herausnehmen, Bonuspunkte zu einer Kategorie hinzufügen oder eine Kategorie ausblenden und aus der Gesamtberechnung herausnehmen.

  • Gewichtung: Gewichtung der Bewertungen der Kategorie, Angabe in Prozent der Gesamtbewertung. Die Summe aller Gewichtungen (unten angezeigt) muss 100 ergeben. Wenn der Wert abweicht, wird er rot angezeigt.
  • Die X niedrigsten herausnehmen: Mit dieser Option werden die X schlechtesten Bewertungen eines/r Teilnehmer/in bei der Gesamtbewertung ignoriert. Die Gesamtpunktzahl für eine Kategorie sollte immer denselben Wert haben, damit es nicht zu unkalkulierbaren Ergebnissen kommt.
  • Bonuspunkte: Hier können Sie Extrapunkte vergeben, ohne dass die Gesamtpunktzahl der Kategorie beeinflusst wird. Damit korrigieren Sie nachträglich eine unfaire oder aus anderen Gründen zu korrigierende Bewertung einer Aufgabenstellung. Die Punkte werden jedem Teilnehmer/jeder Teilnehmerin zugeordnet. Wenn Sie Einzelnen Bonuspunkte geben wollen, fügen Sie eine neue Lernaktivität hinzu und wählen sie "Extrapunkte" unter "Kategorien einrichten".
  • Verborgen: Mit dem Anklicken wird die Kategorie ausgeblendet und bei der Berechnung nicht berücksichtigt. Dies ist eine einfache Möglichkeit, Aufgaben nur dann in die Bewertungsübersicht aufzunehmen, wenn sie auch tatsächlich bewertet wurden.
    Da bewertete Aufgaben, die Sie keiner Kategorie zugeordnet haben, automatisch als "Nicht kategorisiert" eingeordnet werden, können Sie z.B. diese Kategorie verbergen und die Aufgaben dann, wenn sie in die Gesamtbewertung einfließen sollen, in eine andere Kategorie verschieben. Dann können auch die Teilnehmer/innen die Bewertung sehen.
