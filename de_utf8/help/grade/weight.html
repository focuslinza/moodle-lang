Gewichtung

Sie können hier die Gewichtung für eine Kategorie vornehmen, die niedrigsten X Resultate herausnehmen, Bonuspunkte zu einer Kategorie hinzufügen oder eine Kategorie ausblenden und aus der Gesamtberechnung herausnehmen.

  • Gewichtung: Gewichtung der Bewertungen der Kategorie. Angabe in Prozent der Gesamtbewertung. Die Summe aller Gewichtungen (unten angezeigt) muß 100 ergeben. Wenn der Wert abweicht wird er rot angezeigt.
  • Niedrigsten Bewertungenn herausnehmen: Hiermit werden die X schlechtesten Bewertungen eines/r Teilnehmer/in bei der Gesamtbewertung ignoriert. The point totals for a category should be all the same value or unpredictable results may occur.
  • Bonuspunkte: Vergeben Sie Extrapunkte. Sie beeinflussen die Gesamtpunktzahl der Kategorie nicht. Damit korrigieren Sie nachträglich eine unfaire oder aus andere Gründen zu korrigierende Bewertung einer Aufgabenstellung Die Punkte werden allen Teilnehmer/innen zugeordnet. Wenn Sie Einzelnen Extrabewertungen geben wollen, fügen Sie eine neue Bewertung hinzu und wählen sie Extrakredit unter "Kategorien einrichten".
  • Verborgen: Mit dem Anklicken wird die Kategorie ausgeblendet und bei der Berechnung nicht weiter berücksichtigt. Damit können Sie Werte nach der Bewertung im Berechnung sichtbar machen Bewertungen, die keiner Kategorie hinzugefügt wurden werden automatisch als nicht kategoriesiert/eingeordnet. Wenn Sie diese Kategorie als 'verborgen' markieren. Nach der Berechnung der Gesamtbewertung können Sie wieder einer Kategorie zugeordnet und für die Teilnehmer/innen sichtbar gemacht werden.
