Importieren von Lernzielen

Lernziele können aus einer .csv Datei importiert werden. Das Format muss wie folgt gestaltet sein:

Feldname Beschreibung Erforderlich Format
outcome_name Vollständiger Titel des Lernziels JaString
outcome_shortnameKurzbezeichnung des Lernziels Ja String
outcome_descriptionBeschreibung des Lernziels Nein String
scale_nameName der verwendeten Bewertungsskala Ja String
scale_itemsKommaseparierte Liste der Bewertungspunkte Ja String
scale_descriptionBeschreibung der Bewertungsskala Nein String

Dies ist ein Beispiel:

outcome_name;outcome_shortname;outcome_description;scale_name;scale_items;scale_description
Beteiligung;beteiligung;;Beteiligungsbewertung;"wenig oder keine Beteiligung, Befriedigende Beteiligung, Sehr gute Beteiligung";
