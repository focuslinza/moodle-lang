Nutzerschüssel

Die Veröffentlichung von Bewertungen ist eine Option zum Export von Noten und den Aufruf derselben über eine URL-Adresse ohne sich dazu einzuloggen. Ein Nutzerschlüssel wird zufällig aus unterschiedlichen Zeichen erstellt und ist an die jeweilige URL gebunden.

Um Bewertungen zu veröffentlichen wählen Sie "Neuen Nutzerschlüssel" erstellen. Wenn sich das Formular öffnet, wurde bereits ein Nutzerschlüssel angelegt. Sollten früher bereits Nutzerschlüssel angelegt worden sein, können diese ausgewählt werden.
