Einstellungen für das Notenbuch

Diese Einstellungen können Sie für das Notenbuch vornehmen:

  • Erweiterte Funktionen nutzen: de-/aktiviert weitere Funktionen für das Notenbuch. Die Normalansicht zeigt die Einzelnoten und eine Summe. Mit den erweiterten Funktionen können Sie Kategorien bilden und besondere Gewichtungen vornehmen.
  • Gewichtete Noten anzeigen: legt fest, ob Prozentwerte gezeigt werden oder nicht.
  • Punkte anzeigen: legt fest, ob Punkte angezeigt werden oder nicht. Die Anzeige für Teilnehmer/innen kann unabhängig von der Trainer/innensicht festgelegt werden.
  • Prozentwerte anzeigen: legt fest, ob Prozentwerte angezeigt werden oder nicht. Die Anzeige für Teilnehmer/innen kann unabhängig von der Trainer/innensicht festgelegt werden.
  • Anzeige einer Gesamtnote: legt fest, ob eine Gesamtnote (z.B. Schulnote, Buchstaben-Note) angezeigt werden soll.
  • Gesamtnote/Buchstabennote: legt fest wie die Gesamtnote ermittelt wird. In manchen Bereichen werden 100er Punktskalen in Schulnoten umgerechnet oder andere Punktwerte in Buchstabennoten umgerechnet.
  • Kopfzeilen wiederholen: legt fest wie oft Kopfzeilen wiederholt werden (erhöht die Übersicht bei langen Tabellen/vielen Teilnehmer/innen.
  • Verborgene Werte anzeigen: die Einstellung bewirkt, dass Trainer/innen als verborgen deklarierte Werte in der Übersicht sehen können. Die Werte werden dann auch in die Gesamtnpote mit einberechnet. Teilnehmer/innen können die Werte nicht einsehen.
