Bewertungsskala

Sie können hier eine eigene Notenskala definieren (z.B. Schulnoten von 1 bis 6, Buchstabennoten von A bis E usw.). Sie können auch Änderungen an der vorhandenen Skala vornehmen und sie dann speichern. Diese Einstellungen können Sie jederzeit überarbeiten. Sie können Werte auch frei lassen oder löschen.
Fehlermeldungen:

  • Keine Bewertungen eingestellt: Es ist keine Bewertungsskala definiert worden. Sie müssen die vorhandene Skala (falls sie Ihnen zusagt) in diesem Fall nur noch abspeichern.
  • Kein Notenwert für X: Dem Ergebnis ist kein Wert zugeordnet.
