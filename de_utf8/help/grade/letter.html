Bewertungsskala

Sie können hier eine eigene Notenskala definieren (z.B. Schulnoten von 1 bis 6, Buchstabennoten von A bis E usw.). Sie können auch Änderungen an der vorhandenen Skala vornehmen und sie dann speichern.
Diese Einstellungen können Sie jederzeit überarbeiten.

  • Zeile löschen: Lassen Sie ein Feld frei und speichern Sie, um die Zeile aus der Skala zu löschen.
  • Zeile hinzufügen: Füllen Sie die letzte Zeile und speichern Sie, um Platz für eine weitere Note zu erhalten.
Fehlermeldungen:

  • Keine Bewertungen eingestellt: Es ist keine Bewertungsskala definiert worden. Sie müssen die vorhandene Skala (falls sie Ihnen zusagt) in diesem Fall nur noch abspeichern.
  • Kein Notenwert für X: Dem Ergebnis ist kein Wert zugeordnet.
