Icons für den Bearbeitungsfortschritt

Neben einigen Lernaktivität befinden sich anklickbare Icons. Damit können Sie kennzeichnen, was Sie im Kurs bereits bearbeitet haben und was noch nicht.

Wenn Sie ein Symbol mit einer gepunkteten Linie (Häckchen) neben dem Namen der Aktivität sehen, können Sie darauf klicken,um es als fertig bearbeitet zu markieren. Nochmaliges daraufklicken ändert die Einstellung wieder.

Mit dieser Funktion können Sie gut einen Überblick darüber behalten, was bereits bearbeitet wurde und wo Sie noch etwas tun müssen/wollen.

Einige Aktivitäten enthalten vielleicht ein leeres Kästchen. Diese werden automatisch umgestellt wenn Sie die Anforderungen erfüllt haben. Dies kann heißen, dass Sie einen Test abgeschlossen oder eine bestimmte Bewertung erzielt haben müssen. Wenn dies nicht gleich angezeigt wird, kann es sein, dass Sie die Seite neu laden müssen.

Ihre Trainer/innen haben Einsicht, welche Aktivitäten von Ihnen als 'bearbeitet' gekennzeichnet wurden. Die manuelle Markierung einer Aktivität hat keine Auswirkung auf die Bewertung im Kurs.
