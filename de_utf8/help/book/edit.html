Kapitel bearbeiten

Formulieren Sie kurze Kapitelüberschriften. Diese Überschriften werden in der Inhaltsübersicht im linken Menü verwendet.

Sie können bis zu zwei Unterebenen zu den Kapiteln anlegen.

Falls Sie Eingabeprobleme mit dem WYSIWYG-Editor haben, versuchen Sie einen anderen Browser (z.B. Mozilla Firefox) zu benutzen oder schalten Sie den Editor für die Eingabe von Texten aus. Diese Ausschaltmöglichkeit finden Sie in Ihrem persönlichen Profil.
