Inhaltsübersicht überspringen

Wenn Sie ein Lernpaket mit einem einzigen enthaltenen Lernobjekt hinzufügen, dann können Sie wählen, ob die Inhaltsübersicht automatisch übersprungen werden soll, wenn Teilnehmer/innen auf die SCORM-Aktivität in der Kursseite klicken.

Zur Auswahl stehen:

  • Nie: Die Inhaltsübersicht wird nie übersprungen.
  • Erster Zugang: Die Inhaltsübersicht wird nur beim ersten Mal übersprungen.
  • Immer: Die Inhaltsübersicht wird immer übersprungen.
