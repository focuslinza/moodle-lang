Was ist ein Lernpaket?

Ein Lernpaket ist eine Datei mit der Dateiendung zip (oder pif), die Dateien mit SCORM- oder AICC-Kursdefinitionen enthalten.

Ein SCORM-Lernpaket muss im Basisverzeichnis eine Datei mit Namen imsmanifest.xml enthalten. Sie definiert die Kursstruktur des Lernpakets, den Ablageort f√ºr Ressourcen und andere Dinge.

Ein AICC-Lernpaket besteht aus verschiedenen Dateien (von 4 bis 7) mit verschiedenen Dateiendungen. Die Dateiendungen haben folgende Bedeutung:

  • CRS - Kursbeschreibung (erforderlich)
  • AU - Assignable Unit file (erforderlich)
  • DES - Descriptor file (erforderlich)
  • CST - Kursstruktur (erforderlich)
  • ORE - Objective Relationship file (optional)
  • PRE - Voraussetzungen (optional)
  • CMP - Completition Requirements file (optional)
