Tipuri "Temă de casă"

Sunt diferite feluri de teme de casă

Activitate offline - această opţiune este folositoare când tema de casă este rezolvată în afara site-ului. Studenţii pot vedea o descriere a temei de casă dar nu pot încărca pe server nici un fel de fişier. Profesorii pot acorda totuşi note, iar studenţii le pot afla.

Încarcă un singur fişier pe server - acest tip de temă de casă permite tuturor studenţilor să încarce un singur fişier, indiferent de extensia sa. Aşadar, fişierul poate fi un document Word, o imagine, un întreg site comprimat sau orice alt tip de fişier cerut de profesor. Profesorul poate acorda note online pentru tema de casă.
