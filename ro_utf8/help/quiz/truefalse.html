Întrebări Adevărat/Fals

Ca răspuns la o întrebare (care poate să includă o imagine) cel examinat alege una dintre cele două variante de răspuns: True sau False.

Dacă feedback-ul este activat, atunci mesajul de feedback este arătat celui examinat imediat sub răspuns. De exemplu, dacă răspunsul corect este "False", dar el răspunde "True" (greşind) atunci feedback-ul de la "True" este arătat.
