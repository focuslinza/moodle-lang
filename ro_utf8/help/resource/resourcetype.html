Tipul resursei

Resursele se refera la orice tip de fişier (.doc, .txt, .pps, etc).

Text simplu - cel mai accesibil tip. Tastezi textul direct în formele care apar. Unele cuvinte pe care le scrii vor fi formatate automat (ex. intervenţiile din forum). Ajutor în redactarea textului

HTML text -Pentru utilizatori mai avansaţi. Formatarea nu se va face automat, este nevoie să o faci manual folosind tag-uri HTML.

Referinţă - O referinţă bibliografică.

Fişiere încărcate - Arată toate fişierele pe care le-ai încarcat pentru acest curs. Uită-te la secţiunea 'Fişiere'.

Link web - Un URL pentru o locaţie de web. Oricine va da click pe aceast link va fi scos afară din curs (pagina la care e făcut link-ul va apărea în fereastra browserului).

Pagină Web - Un URL pentru o locaţie de web. Ca şi tipul anterior acesta va lansa o pagină de web. Spre deosebire de tipul anterior, totuşi, pagina va apărea în interiorul ferestrei, ca şi cum ea ar fi inclusă în curs.

Program - un program extern care cere să se treacă în el informatţi despre studenţii şi cursurile Moodle. Acesta permite ca aplicaţii web externe să coopereze cu Moodle.
