Suggerimenti sulla redazione di testi

Redigere un testo in Moodle è molto semplice ed intuitivo. Avete anche la possibilità di aggiungere al testo le "faccine", "indirizzi URL" ed alcuni tipi di tag HTML

Faccine (emoticon)

Per aggiungere faccine al testo, è sufficiente inserire il codice corrispondente alla faccina desiderata. I codici delle faccine assomigliano a volti stilizzati: per accorgervene guardate i codici della faccine piegando la vostra testa verso sinistra.

URL

Qualsiasi parola che inizi per www. oppure http:// sarà automaticamente trasformata in un link attivo.

Ad esemepio: www.yahoo.com oppure http://curtin.edu.

Tag HTML

Potete formattare il testo usando alcuni tipi di tag HTML.

Tag HTMLRisultato
<b> grassetto </b> grassetto
<i> italico </i> italico
<u> sottolineato </u> sottolineato
<font color="green"> esempio </font> esempio
<ul> <li>uno</li> <li>due</li> </ul>
