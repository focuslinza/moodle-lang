Tipi di Compito

C'è un numero crescente di tipi di compiti disponibili:

Offline

Questo è utile quando il compito è svolto al di fuori di Moodle. Potrebbe essere presente da qualche parte sul web o svolto in presenza.

Gli studenti possono vedere una descrizione del compito, ma non possono caricare file o altro. Comunque i docenti possono dare un voto agli studenti, e gli stessi possono controllare i propri voti online.

Online

Questo tipo di compito chiede agli utenti di preparare un testo, utilizzando i normali strumenti di editing. I docenti possono valutarli online, e anche aggiungere commenti in linea e modifiche.

(Se siete familiari con le vecchie versioni di Moodle, questo tipo di compito permettere di fare le stesse cose che permetteva il vecchio modulo Diario.)

Consegna semplice

Questo tipo di compito permette a tutti gli studenti di caricare un singolo file, di qualunque tipo.

Il file può essere un documento Word, o un'immagine, o un sito web zippato, o qualunque cosa docente chiede di produrre e poi caricare.

Consegna multipla

Questo tipo di compito permette a ogni partecipante di caricare uno o più file di qualunque formato. I file possono essere documenti Word, o immagini, o siti web zippati, o qualunque cosa il docente chiede di produrre e poi caricare.

Questo tipo permette anche al docente di rispondere con file multipli. I file di risposta possono essere caricati anche prima delle consegne degli studenti e questo può essere utile per fornire a ogni partecipante diversi file sul quale lavorare.

I partecipanti possono anche indicare delle note descrittive per i file, stato di avanzamento o qualunque altra informazione testuale.

Il completamento della consegna di questo tipo di compito deve essere segnalato esplicitamente dal partecipante. Al docente è comunque possibile vedere lo stato del compito di ciascuno in qualunque momento e i compiti non terminati sono segnati come Bozza. Il docente può anche riportare allo stato di Bozza qualunque compito non ancora valutato.
