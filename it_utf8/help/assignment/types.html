Tipi di Compito

C'è un numero crescente di tipi di compiti disponibili:

Attività offline

Questo è utile quando il compito è svolto fuori di Moodle. Potrebbe essere qualcos'altro sul web o svolto in presenza.

Gli studenti possono vedere una descrizione del compito, ma non possono caricare files o altro. Comunque i docenti possono dare un voto agli studenti, e gli stessi possono controllare i propri voti.

Testo online

Questo tipo di compito chiede agli utenti di modificare un testo, utilizzando i normali strumenti di modifica. I docenti possono valutarli online, e anche aggiungere commenti in linea e modifiche.

(Se siete familiari con le vecchie versioni di Moodle, questo tipo di compito permettere di fare le stesse cose che permetteva il vecchio modulo Diario.)

Caricamento di un file

Questo tipo di compito permette a tutti gli studenti di caricare un singolo file, di qualunque tipo.

Il file può essere un documento Word, o un'immagine, o un sito web zippato, o qualunque cosa docente chiede di produrre e poi caricare.

Caricamento files avanzato

Questo tipo di compito permette a ogni partecipante di caricare uno o più file di qualunque formato. Il file può essere un documento Word, o un'immagine, o un sito web zippato, o qualunque cosa docente chiede di produrre e poi caricare.

Questo tipo permette anche al docente di rispondere con file multipli. I file di risposta possono essere caricati anche prima della sottomissione questo può essere utilizzato per fornire a ogni partecipante diversi file sul quale lavorare.

I partecipanti possono anche indicare delle note descrittive per i file, stato di progresso o qualunque altra informazione testuale.

La sottomissione di questo tipo di compito deve essere manualmente terminata dal partecipante. Al docente è comunque possibile vedere lo stato in qualunque momento, i compiti non terminati sono segnati come Bozza. Il docente può riportare allo stato di Bozza qualunque compito non valutato.
