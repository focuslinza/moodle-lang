File di domande in formato "Moodle XML"

Questo formato nativo di Moodle permette di importare domande già esportate nello stesso formato. Il formato è semplice e chiaro e se ne può avere la dimostrazione esportando una categoria di domande in formato Moodle XML. Il formato XML è in grado di importare immagini.

E' anche possibile estendere le funzionalità di questo formato tramite Plugin opzionali per nuovi tipi di domanda.

Switch categoria

Nel file XML è possibile inserire una domanda fittizia speciale per specificare la categoria di appartenenza delle domande. Se la categoria o le categorie non esistono, saranno create automaticamente. Notare che l'opzione da file deve essere selezionata sulla pagina delle impostazioni altrimenti questa codifica sarà ignorata. Si può cambiare categoria nel file XML tutte le volte che si vuole.

Esempio:

<question type="category">
    <category>
        tom/dick/harry
    </category>
</question>
