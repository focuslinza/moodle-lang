Mostra Feedback di default

Se impostata a Si, quando non è definita una replica per una particolare domanda, verrà utilizzata le risposta default "Risposta corretta" o "Risposta sbagliata".

Se impostata a No, quando non è definita una replica per una particolare domanda, non sarà mostrato alcun feedback. L'utente che utilizza la Lezione sarà automaticamente diretto alla prossima pagina della Lezione.
