Finestra Pop-up per file o pagina web

Questo produrrà una finestra pop-up all'inizio di una lezione per un file (ad esempio un file mp3) o per una pagina web. Inoltre, su ogni pagina della lezione, sarà presente un link che, se necessario, riapre la finestra pop-up.

Opzionalmente può essere inserito un pulsante "Chiudi" in fondo al pop-up come pure l'altezza e la larghezza della finestra.
