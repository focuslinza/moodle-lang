Importare pagine HTML generate da PowerPoint

Istruzioni per l'uso:

Tutte le diapositive di Power Point possono essere importate come tabelle di diramazione complete di bottoni di navigazione avanti e indietro, operando come di seguito:

  1. Aprire la presentazione in PowerPoint
  2. Salvarla come Pagina Web (senza indicare alcuna opzione)
  3. Il risultato dell'operazione precedente è un file con estensione .html e una cartella contenente tutte le diapositive convertite in pagine web: ZIPPARE SOLO QUESTA CARTELLA
  4. Aggiungere in Moodle una nuova lezione
  5. Dopo aver salvato le impostazioni relative alla lezione appariranno 4 possibilità nel riquadro "Cosa vuoi fare prima?": cliccare su "Importa PowerPoint"
  6. Tramite il bottone "Sfoglia..." selezionare il file .zip generato al punto 3 e cliccare sul bottone "Trasferisci questo file".
  7. Se tutto funziona, la prossima videata visualizzerà il bottone "Continua"

Se sono presenti delle immagini nella presentazione PowerPoint, dovrebbero essere salvate come file del corso nella cartella moddata/XY dove X è il nome della lezione e Y un numero (solitamente 0). Inoltre, durante il processo di importazione, vengono creati dei file nella cartella \moodledata\temp\lesson. Questi file ad oggi, non vengono cancellati da importppt.php.
