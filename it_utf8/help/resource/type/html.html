Pagina web

La pagina web permette di creare con facilità all'interno di Moodle una pagina HTML, consentendo di usare l'editor visuale HTML di Moodle (WYSIWYG).

La pagina web creata con questo tipo di risorsa viene salvata nel Database (e non in un file) ed è possibile usare la maggior parte delle caratteristiche di un pagina HTML, incluso il Javascript.
