Pacchetto IMS

    Questa risorsa permette di pubblicare un pacchetto di contenuti conforme alle specifiche IMS.

    Tramite il bottone "Scegli o aggiorna file" è possibile caricare e/o scegliere il file .zip che contiene il pacchetto di contenuti.

    La visualizzazione dei contenuti può essere gestita via frame nella stessa finestra di Moodle o tramite una nuova finestra di popup. E' inoltre possibile decidere come il pacchetto verrà pubblicato, scegliendo se visualizzare l'indice e i pulsanti di navigazione.
