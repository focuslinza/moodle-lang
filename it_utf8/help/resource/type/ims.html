Pacchetto IMS

I pacchetti software IMS possono essere creati e modificati usando una varietà di strumenti software per l'authoring di contenuti. Questi contenuti sono di solito mostrati su più pagine con la possibilità di navigare tra le stesse. Questi software producono un file zip, che può essere caricato nel proprio corso in Moodle.

Il file zip è unzippato automaticamente su Moodle, ed è mostrato il contenuto del pacchetto.

Ci sono varie opzioni per mostrare tale contenuto in una finestra popup, con un menu di navigazione o pulsanti, ecc.
