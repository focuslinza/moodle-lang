Link a file o siti web

Questo tipo di risorsa permette di linkare qualunque pagina web o qualunque file sul web pubblico. Permette anche di collegare pagine web o file che sono stati precedentemente caricati nell'area file del corso dal proprio computer.

Le normali pagine web sono visualizzate così come sono, mentre i file multimediali sono trattati più intelligentemente e possono essere inseriti in una pagina web. Per esempio, i file MP3 saranno mostrati utilizzando uno streaming player predisposto internamente, come pure i file video, le animazioni flash e così via.

Esistono molte opzioni per visualizzare il proprio contenuto in finestre popup, in finestre a frame e così via.

In particolare, se la propria risorsa è un'applicazione web o altro tipo di contenuto capace di accettare parametri, si può decidere di passare le informazioni alla risorsa, come per esempio, il nome utente, la sua email, il nome del corso in cui si trova, e così via.
