Box.net API Key

Recatevi su enabled.box.net e create un account da developer. Dopo esservi utenticati, nella pagina My projects create un nuovo progetto.

L'unica impostazione fondamentale per Moodle è la url di callback, che dovrebbe essere $CFG->wwwroot/portfolio/add.php?postcontrol=1.

Box.net aggiungerà all'URL la parte rilevante del token di autenticazione (individuando se contiene ? e quant'altro).
