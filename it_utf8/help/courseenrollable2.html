Possibilità di iscrizione

Questa impostazione determina se gli studenti posso iscriversi al corso usando il plugin di iscrizione interattiva default (per es. Iscrizione interna) oppure se riceveranno il messaggio "In questo momento non è possibile iscriversi al corso".

Questa impostazione non ha effetto per i plugin di iscrizione non-interattiva (per es. Iscrizione tramite un database esterno). Si tenga anche presente, che un utente per ottenere il ruolo di studente (e cioè essere iscritto) tramite il link Ruoli presente nel blocco di Amministrazione del corso.
