Categoria di importazione

Il menu drop-down Categoria: serve per selezionare la categoria nella quale le domande importate dovranno andare.

Alcuni formati di importazione (GIFT e XML) permettono di specificare la categoria all'interno del file di importazione. Perch√© questo avvenga deve essere marcato il box Categoria da file, altrimenti le domande andranno nella categoria selezionata senza riguardo alle istruzioni presenti nel file. I file esportati da Moodle possono anche contenere il contesto di categoria inserito nella stringa della categoria. Deselezionare il box Contesto da file per forzare l'uso del contesto default.

Quando le categorie sono specificate dentro un file di importazione, se le stesse ancora non esistono saranno automaticamente create.
