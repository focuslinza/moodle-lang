Stop in caso di errore

Le domande sono importate in due passi. Primo, il file di input è sottoposto a 'parsing' e ne viene controllata la validità formale. Secondo, viene scritto nel database. Se questa opzione è selezionata (default) e viene rilevato un errore durante il 'parsing' allora non verrà scritto nulla nel database.

Se l'opzione non è selezionata, si cercherà di scrivere nel database tutte le domande risultate valide.
