Download del riepilogo

Si può voler scaricare sul proprio computer i dati in tabella mostrati sullo schermo per ulteriori analisi. SI può scegliere tra due formati di file per i dati scaricati.
In ogni caso i dati verranno mostrati in una tabella con titoli appropriati, come sullo schermo. Se la tabella è divisa in più pagine, tutti i dati di tutte le pagine saranno scaricati in un singolo file.

Formato foglio elettronico Excel:

Si otterrà un documento .xls .

Formato Testo

In questo caso i dati saranno memorizzati come un normale file di testo. Una riga per ogni riga della tabella con i dati separati da caratteri di tabulazione.
