Importazione files in formato "Moodle XML"

Switch Categoria

Può essere inserita una speciale domanda fittizia nel file XML per specificare il percorso della categoria nella quale le domande che seguono dovranno essere inserite. Se la categoria o le categorie non esistono, saranno create. Da notare che l'opzione da file, sulla pagina delle opzioni, deve essere marcata o questa categoria sarà completamente ignorata. Le categorie possono essere cambiate nel file xml tutte le volte che si vuole.

Un esempio del formato è il seguente:

<question type="category">
    <category>
        tom/dick/harry
    </category>
</question>
