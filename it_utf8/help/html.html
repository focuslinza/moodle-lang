Scrivere in HTML

Quando si scrive in HTML si è liberi di inserire qualsiasi tag HTML per produrre gli effetti voluti

Da notare che lo scripting (cioè Javascript o Script VB) non è consentito e sarà automaticamente rimosso.

Il codice scritto sarà collocato all'interno di una cella di tabella, per cui:

  • non è necessario utilizzare i tag <HEAD> o <BODY>
  • prestare attenzione durante l'inserimento di tabelle alla chiusura delle stesse con </TABLE>, per non distorcere l'impaginazione del corso.

Le faccine (emoticon) saranno convertite nella grafica equivalente, e gli indirizzi URL saranno convertiti in link.
