Calcoli

I calcoli per il registro di valutazioni seguono la stessa sintassi delle formule/funzioni dei fogli di calcolo. Essi cominciano con un segno uguale (=), e usano i soliti operatori matematici e le stesse funzioni per produrre un singolo output numerico. Questo output è poi utilizzato come valore calcolato per l'elemento di valutazione che si sta modificando.

Di seguito è riportata una lista delle funzioni supportate dal calcolo. La virgola (,) è usata per separare gli argomenti all'interno delle parentesi.

  • average([[item1]], [[item2]]...): Restituisce la media di una lista di argomenti
  • max([[item1]], [[item2]]...): Restituisce il massimo di una lista di argomenti
  • min([[item1]], [[item2]]...): Restituisce il minimo di una lista di argomenti
  • mod(dividend, divisor): Calcola il resto di una divisione
  • pi(): Restituisce il valore del numero Pi
  • power(base, power): Innalza un numero alla potenza di un altro
  • round(number, count): Arrotonda un numero con una determinata precisione
  • sum([[item1]], [[item2]]...): Restituisce la somma di tutti gli argomenti

E' anche supportato il seguente elenco di funzioni matematiche:

  • sin
  • sinh
  • arcsin
  • asin
  • arcsinh
  • asinh
  • cos
  • cosh
  • arccos
  • acos
  • arccosh
  • acosh
  • tan
  • tanh
  • arctan
  • atan
  • arctanh
  • atanh
  • sqrt
  • abs
  • ln
  • log
  • exp

Si possono includere i valori degli elementi di valutazione usando il loro numero id come riferimento nelle formule. Il numero id è contenuto in due coppie di parentesi quadre, per evitare confusione quando il numero id contiene caratteri non comuni. Per esempio, se si ha un elemento di valutazione con numero id uguale a Quiz.3, nelle formule ci si riferirà a tale elemento con [[Quiz.3]].

Sotto al campo Calcolo c'è una ramificazione del proprio corso con le sue categorie ed elementi di valutazione. A fianco al totale di ogni elemento o categoria è mostrato il numero id che può essere usato nei calcoli (già incluso nelle parentesi quadre). Comunque, poiché il numero id è opzionale, alcuni elementi potrebbero non averne ancora uno. Questi elementi hanno allora un campo di input che permette di inserire direttamente un nuovo numero id, in modo tale da poterlo usare nei calcoli senza necessità di lasciare la stessa pagina. Comunque in questo modo non è possibile modificare i numeri id esistenti. Se questo è necessario bisognerà andare a modificare direttamente l'elemento di valutazione.

Appena saranno stati assegnati i numeri id necessari, si potrà cliccare su "Aggiungi numeri id", e la pagina sarà ricaricata e mostrerà la stessa ramificazione del corso con i numeri id appena assegnati. Si è allora liberi di usare questi riferimenti nei propri calcoli.

Segue una lista di esempi pratici di calcoli che possono essere usati nel proprio registro di valutazioni:

  • =average([[Quiz.1]], [[Quiz.4]], [[Assignment.1]])
  • =average(max([[Quiz.1]], [[Quiz.4]], [[Assignment.1]]), min([[Quiz.1]], [[Quiz.4]], [[Assignment.1]]))
