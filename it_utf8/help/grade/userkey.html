Chiave utente

La pubblicazione delle valutazioni è un modo di esportare le stesse attraverso un URL che rende non necessario essere loggati in Moodle. Una chiave utente è una serie di caratteri generati a caso che è appesa all'URL per motivi di sicurezza.

Per pubblicare le valutazioni, selezionare "Crea una nuova chiave utente". Quando il form è inviato, viene creata una nuova chiave utente. Alternativamente può essere selezionata una chiave creata precedentemente.
