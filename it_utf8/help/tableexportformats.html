Download Dati Tabella

Si possono scaricare i dati mostrati sullo schermo per ulteriori analisi. Per i dati scaricati si può scegliere uno tra i molti formati sotto elencati.

In ogni caso i dati saranno presentati sotto forma di tabella con appropriati titoli di colonna, come sullo schermo. La tabella scaricata è:

  • non impaginata, tutti i dati di tutte le pagine saranno scaricati in un singolo file.
  • non filtrata per nome utente. (Alcune tabelle di dati riguardanti gli utenti permettono sullo schermo di mostrare solo i dati di utenti con iniziali definite per cognome/nome).
  • non modificata dall'ordinamento per colonne che un utente può fare su una tabella visualizzata sullo schermo.

Foglio elettronico Microsoft Excel.

Si otterrà un file .xls .

Foglio elettronico OpenDocument (ODS).

Si otterrà un file .ods . Questo formato può essere aperto con OpenOffice, KOffice, Google Docs, NeoOffice, Zoho, IBM Lotus Symphony e Corel WordPerfect Office X4.

Un file di testo con valori separati da tab

In questo caso si otterrà un normale file di testo. Una riga per ogni riga di tabella con i dati separati da tabstops. Può essere aperto da diversi programmi applicativi.

Un file di testo CSV (Comma Separated Values)

In questo caso si otterrà un normale file di testo. Una riga per ogni riga di tabella con i dati separati da virgole. Può essere aperto da diversi programmi applicativi.

Un documento XHTML non impaginato

In questo caso si otterrà un documento xhtml salvabile, stampabile senza gli headers e i footers che si hanno su una normale pagina Moodle.
