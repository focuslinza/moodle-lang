Opzioni di visualizzazione

Sono disponibili diverse opzioni di visualizzazione a seconda dei browser utilizzati:

best
Il quiz verrà visualizzato nel formato migliore per il browser.

v6+
Il quiz verrà visualizzato nel formato "drag & drop" per browser v6+

v6
Il quiz verrà visualizzato per browser v6

v5
Il quiz verrà visualizzato nel formato per browser v5

v4
Il quiz verrà visualizzato nel formato per browser v4

v3
Il quiz verrà visualizzato nel formato per browser v3

Flash
Il quiz verrà visualizzato con Flash

mobile
Il quiz verrà visualizzato in un formato adatto alla visualizzazione su palmari o cellulari
