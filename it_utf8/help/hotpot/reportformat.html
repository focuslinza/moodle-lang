Formato dei rapporti

E' possibile specificare diversi formati per l'emissione dei rapporti di Hot Potatoes.

HTML
Il rapporto verrà generato in formato HTML adatto per la visualizzazione con browser web
Excel
Il rapporto generato sarà un file con estensione ".xls" utilizzabile con un programma per fogli elettronici come Microsoft Excel
Testo
Il rapporto generato sarà un file con estensione ".txt" utilizzabile con editor di testo
Codifica
Se il programma per la gestione del foglio elettronico lo richiede, è possibile dichiarare un elenco di codifiche con cui generare il rapporto. Per esempio, Microsoft Excel richiede la codifica "SJIS" per il Giapponese. La lista delle codifiche possibili può essere dichiarata dall'amministratore del sito seguendo il percorso Configurazione>Moduli>Quiz Hot Potatoes.
Wrap data
Si
Informazioni estese saranno adattate per essere completamente visibili all'interno delle celle. L'altezza di alcune righe potrebbe aumentare di molto.
No
Le informazioni non verranno adattate.
Visualizzare la legenda
Si
Nelle statistiche dettagliate le risposte alle domande vengono rappresentate con le lettere dell'alfabeto. La legenda generata mostra l'associazione tra la lettera e la risposta.
No
Domande e risposte appariranno nel rapporto e non verrà generata la legenda.
