Estensione iscrizione

Questa pagina permette di allungare o ridurre la durata dell'iscrizione per i partecipanti al corso selezionati. Il nuovo periodo di iscrizione è calcolato in base al riferimento per la data di partenza e all'estensione selezionata.

Possibili riferimenti per le date di partenza sono:

  • Data di inizio del corso
  • Iscrizioni aperte - la data di inizio dell'iscrizione del partecipante (questa opzione è possibile solamente per utenti con una nota data di inizio iscrizione)
  • Iscrizioni chiuse - la data di fine dell'iscrizione del partecipante (questa opzione è possibile solamente per utenti con una nota data di fine iscrizione)
  • Oggi (questa opzione è possibile solo se la data di inizio corso è passata)
  • Data di inizio iscrizioni al corso - la data di inizio delle iscrizioni presa dalle impostazioni del corso (questa opzione è possibile solo per i corsi con una data di inizio iscrizioni impostata)
  • Data di fine iscrizioni al corso - la data di fine delle iscrizioni presa dalle impostazioni del corso (questa opzione è possibile solo per i corsi con una data di fine iscrizioni impostata)

Se la durata dell'estensione è impostata a "Nessun cambiamento" o 'Illimitata', allora la data di riferimento per la partenza è ignorata.
