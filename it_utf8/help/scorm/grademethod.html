Metodo di valutazione

I risultati delle attività dei pacchetti SCORM/AICC mostrati nella pagina delle valutazioni possono essere valutati in diversi modi:

  • Situazione dei learning objects
    Questa modalità mostra il numero di Learning Object completati/passati per l'attività. Il punteggio massimo è il numero di Learning Object nel pacchetto.
  • Il voto più alto
    La pagina delle valutazione mostrerà il punteggio massimo ottenuto dagli utenti in tutti Learning Object passati.
  • Media dei voti
    Se si sceglie questa modalità Moodle calcolerà la media di tutti i punteggi.
  • Somma dei voti
    Con questo metodo tutti i punteggi verranno sommati.
