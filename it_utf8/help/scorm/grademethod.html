Metodo di valutazione

I risultati delle attività dei pacchetti SCORM/AICC mostrati nella pagina delle valutazioni possono essere memorizzati in diversi modi:

  • Learning object
    Questa modalità mostra il numero di Learning object completati/superati. Il punteggio massimo sarà il numero di Learning Object che compongono il pacchetto.
  • Voto migliore
    La pagina delle valutazioni mostrerà il punteggio massimo ottenuto dagli utenti in tutti Learning Object superati.
  • Media dei voti
    Se si sceglie questa modalità Moodle calcolerà la media di tutti i punteggi ottenuti.
  • Somma dei voti
    Con questo metodo tutti i punteggi ottenuti verranno sommati.
