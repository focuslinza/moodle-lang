File del pacchetto

Il pacchetto è un file particolare con estensione zip (o pif) che contiene dei file validi di definizione SCORM o AICC.

Un pacchetto SCORM deve contenere nella radice dello zip un file chiamato imsmanifest.xml che definisce la struttura dei contenuti SCORM, la posizione delle risorse ed altre informazioni.

Un pacchetto AICC è definito da diversi file (da 4 a 7) con nome comune ed estensioni definite. Di segiuto trovate il significato delle estensioni:

  • CRS - Course Description file (obbligatorio)
  • AU - Assignable Unit file (obbligatorio)
  • DES - Descriptor file (obbligatorio)
  • CST - Course Structure file (obbligatorio)
  • ORE - Objective Relationship file (opzionale)
  • PRE - Prerequisites file (opzionale)
  • CMP - Completition Requirements file (opzionale)
