Il file del pacchetto

Il pacchetto è un file particolare con estensione zip (o pif) che contiene dei file di definizione SCORM o AICC validi.

Un pacchetto SCORM deve contenere nella radice dello zip un file chiamato imsmanifest.xml che definisce la struttura del corso SCORM, la posizione risorse e molte altre cose.

Un pacchetto AICC è definito da diversi file (da 4 a 7) con nome comune ed estensioni definite. Qui trovate il significato delle estensioni:

  • CRS - Course Description file (obbligatorio)
  • AU - Assignable Unit file (obbligatorio)
  • DES - Descriptor file (obbligatorio)
  • CST - Course Structure file (obbligatorio)
  • ORE - Objective Relationship file (opzionale)
  • PRE - Prerequisites file (opzionale)
  • CMP - Completition Requirements file (opzionale)
