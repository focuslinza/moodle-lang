패키지 파일들

이 패키지는 zip(혹은 pif)확장자를 가진 특별한 파일로서 이에는 AICC 혹은 SCORM 과정에 적합한 정의 파일들을 포함하고 있다.

A SCORM 패키지에는 imsmanifest.xml라 부르는 파일이 들어있는데, 이에는 SCORM과정의 구조, 자료의 위치, 그리고 다른 여러 것들이 정의되어 있다.

An AICC 이는 (넷에서 일곱개 까지의) 미리 정해진 확장자에 따른 여러 파일에 의해 정의된다. 확장자의 의미는 다음과 같다.

  • CRS - 배움터 설명 파일(필수)
  • AU - Assignable Unit file (필수)
  • DES - Descriptor file (필수)
  • CST - 과정 구조 파일 (필수)
  • ORE - 객체 관련 파일 (선택)
  • PRE - Prerequisites file (선택)
  • CMP - Completition Requirements file (선택)
