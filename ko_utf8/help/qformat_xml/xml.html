"Moodle XML" 형식

본 무들 전용 형식은 이전에 동일한 형식으로 내보냈던 퀴즈 문항들을 가져온다. 이 형식은 간단하면서도 Moodle의 XML 형식으로 내보낸 범주를 가장 잘 표현해 준다. XML 형식은 이미지 파일도 가져올 수 있다.

여타의 문형 플러그인도 이 형식을 확장할 수 있다.

범주 변경

자신이 문항을 저장하고 싶은 경로를 지정하려 한다면, XML 파일 안에 특별한 더미 문항을 삽입할 수도 있다. 만일 범주들이 존재하지 않을 경우에는 새로 만든다. 만일 선택 화면에서 파일로부터 ..추출 옵션에 표시가 되어 있다면 이 코드는 완벽하게 무시될 것임에 유의하라. XML 파일 내에서는 원하는 대로 범주를 변경할 수 있다.

다음은 그 예이다.

<question type="category">
    <category>
        tom/dick/harry
    </category>
</question>
