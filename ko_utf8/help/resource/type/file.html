파일과 웹페이지

이 자원의 유형은 공개된 어떠한 웹문서나 파일을 연결할 수 있게 해 줍니다. 또 이 유형은 자신의 컴퓨터에 있는 어떠한 자료라도 배움터에 올려놓고 연결하여 사용할 수 있게 합니다.

정상적인 웹 문서는 문제없이 쉽게 표시되며, 동영상 자료의 경우 좀 더 지능적으로 웹페이지에 포함되어 보여질 수 있습니다. 예를들어 MP3 파일은 내장된 프로그램을 통해서 연주되며 영화나 동영상파일 역시 관련 구동프로그램을 구동하여 시연됩니다.

내용을 표시하기 위한 방법에는 팝업창으로 표현하는 방법, 동일 화면에 나타내는 방법등 다양한 설정이 있습니다.

특히 자료가 웹을 기반으로 하는 프로그램 혹은 따로 자료 입력을 요구하는 경우에는, 사용자명, 이메일 주소, 현재 등재되어 있는 배움터 등의 자료를 전송 가능하게 설정할 수 있다.
