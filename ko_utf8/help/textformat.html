문서의 형식

무들에서 문서를 작성할 때에는 여러분의 능력과 쓰고 있는 브라우저에 따라 다양한 문서를 만들어 낼 수 있다.

대체로 현재 설정되어 있는 그대로 써도 여러분들이 원하는 작업을 할 수 있을 것이다..

1. 무들 형식

이 형식은 일반적인(전문적인 HTML편집기 대신) 웹 양식을 써서 입력할 때에 가장 좋다. 마치 이메일을 보내듯 그냥 일반 문서처럼 입력하면 된다.

문서를 저장할 때, 무들은 자동적으로 몇 가지 필요한 작업을 대신 해 준다.

예들 들면 아래와 같은 URLs은

http://yahoo.com 혹은 그냥 www.yahoo.com 도 자동으로 링크를 만들어 준다.

줄바꿈은 그대로 유지가 되고, 빈 줄은 새로운 문단의 시작을 의미하게 된다.

:-) 와 같은 이모티콘은 자동으로 이미지로 변환하여 표시해 준다.

심지어는 HTML 코드를 집어넣어 원하는 그대로 표현할 수도 있다.

2. HTML 형식

이는 문서가 완벽한 HTML 문서임을 가정한다. 만일 여러분이 문서 작성을 하는데 HTML 편집기를 사용한다면, 이것이 기본 형식이 된다. 도구상자에 있는 모든 명령어가 여러분이 HTML 문서를 작성하는데 도움을 줄 것이다.

만일 여러분이 전문적인 HTML편집기를 사용하지 않더라도, 문서에 HTML코드를 직접 쓸 수 있고, 그것은 여러분이 의도한 대로 정확히 표시될 것이다.

무들 형식과는 다르게, 자동적으로 형식화 하지는 않는다.

3. 일상적 문서 형식

이 형식은 매우 많은 양의 문서를 포함하거나, 정확하게 HTML 코드를 작성하는 대로 보이게 하고 싶은 경우에 유용하다.

이 형식은 여전히 공백과 새로운 줄을 인식하지만, 여러분이 작성한 대로 표시된다.

4. Markdown 문서 형식

Markdown 형식은 이메일을 작성하는 정도의 문서 작성을 통해서 쉽게 잘 형식화된 XHTML페이지를 만들 수 있게 한다.

이것은 몇 개 안되는 기호를 가지고 깔끔한 문서 화면을 만드는데 매우 유용하지만 그림이나 링크가 많은 것에는 적합하지 않다.
