HTML 편집 단축키

다음의 기능은 단축키를 통해서 구현할 수 있다.

Ctrl-O 글꼴 변경
Ctrl-P 글자 크기 변경
Ctrl-H 문서 형태 변경(표제 등)
Ctrl-1 to Ctrl-6 제목 수준 변경
Ctrl-B 고딕체로
Ctrl-I 이탤릭체로
Ctrl-U 밑줄
Ctrl-S 문자 가운데 줄(삭제표시)
Ctrl-, 아래첨자
Ctrl-. 윗첨자
Ctrl-0 Clean Word HTML
Ctrl-Z 실행취소
Ctrl-Y Redo
Ctrl-L 왼쪽 정렬
Ctrl-E 중앙 정렬
Ctrl-J 양쪽 배분
Ctrl-/ 왼쪽에서 오른쪽으로 쓰기
Ctrl-| 오른쪽에서 왼쪽으로 쓰기
Ctrl-; 내어쓰기
Ctrl-' 들여쓰기
Ctrl-G 글자 색 변경 colour
Ctrl-K 배경색 변경
Ctrl-F 찾아 바꾸기
Ctrl-` HTML 소스코드 보기
Ctrl-M 전체 편집 화면으로
Ctrl-Alt-O 순서 있는 목록 삽입
Ctrl-Alt-U 순서 없는 목록 삽입
Ctrl-Alt-R 수평선 삽입
Ctrl-Alt-A 닺 생성
Ctrl-Alt-L 링크 생성
Ctrl-Alt-D 링크 해제
Ctrl-Alt-N 자동 링크 해제
Ctrl-Alt-I 그림 삽입
Ctrl-Alt-T 표 삽입
Ctrl-Alt-S 이모티콘 삽입
Ctrl-Alt-C 특수 문자 삽입
