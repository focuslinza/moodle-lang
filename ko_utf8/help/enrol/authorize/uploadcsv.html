Authorize.net CSV 파일 올리가

만일 eCheck (ACH) 지불을 활성화시켰으면 반드시 이 작업을 수행해야만 합니다.

  1. authorize.net 계정으로 로그인 한다.
  2. 지불에 대하여 REPORT 나 SEARCH를 실행시킨다.
  3. 만일 지불 검색을 하였다면 기간을 선택한다.
  4. 'Download to File'버튼이 나타난다.
  5. 버튼을 클릭한 후, 'Include Column Headings'에 체크하고 내려받을 파일 유형 'Expanded Fields/Comma Separated' 혹은 'Expanded Fields with CAVV Response/Comma Separated' 을 선택한다.
  6. 좀 더 자세한 사항을 알고 싶을 때는 도움말 링크를 클릭한다.
  7. Submit 버튼을 클릭한다.
  8. 일단 파일 생성이 끝나면 파일을 저장할 경로를 묻는 팝업창이 뜬다.
  9. 저장할 위치가 나타날 때까지 팝업창을 띄어 둔다. 작업을 계속하기 위해 오른쪽 상단의 화면 최소화 아이콘을 클릭하여 화면을 최소화 할 수는 있다. 만일 팝업창을 닫아버리게 되면 모든 작업이 취소되고 파일을 내려받지 못하게 된다.
  10. 파일을 다 내려받고 내 컴퓨터에 저장이 끝나면 화면 오른쪽 상단의 X표시를 클릭하여 팝업창을 닫는다.
  11. 마지막으로 사용자들이 echeck를 이용하여 지불을 할 수 있도록, 이 CSV 파일을 무들에 올린다.
