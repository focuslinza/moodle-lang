Features

Moodle은 역동적으로 진화하는 프로그램입니다. 본 페이지에서는 무들이 담고 있는 여러 기능 중 주요한 몇 가지를 소개 합니다.

전체적인 디자인

  • 사회 구성주의자의 교육 철학을 구현(협동학습, 활동, 비판적 반영 등)
  • 면대면 학습 상황을 지원할 뿐만아니라 100% 온라인 수업에 적합
  • 간편하고 효율적이며 가볍고 호환성이 큰 브라우저 인터페이스
  • PHP를 지원하는 어떤 환경하에서도 손쉽게 설치 가능. 데이터베이스 단 하나로 처리(공유 가능)
  • 전반적인 데이터베이스 요약 설명을 기반으로 현존하는 모든 대형 데이터베이스를 지원(초기 테이블 설정 제외)
  • 손님 계정의 접속 가능성을 포함해, 서버의 모든 과정에 대한 설명을 보여주는 배움터 목록
  • 범주화되고 찾기 쉬운 배움터 - 하나의 Moodle 사이트로 수 천여 교과 과정을 운영 가능
  • 완벽하고 강력한 보안에 중점. 제반 서식, 데이터의 유효성, 쿠기의 암호화 등이 점검됨
  • (자원, 포럼의 게시물 등) 대다수의 문서영역은 내장된 WYSIWYG HTML 편집기를 통해 편집 가능

사이트 관리

  • 설치시 지정된 한사람의 관리자에 의해 사이트가 관리됨
  • 관리자는 플러그 인 "테마"를 이용하여 사이트의 색, 글자꼴, 전체적인 모습 등을 실정에 맞게 고칠 수 있음
  • 플러그 인 활동 묘듈을 이미 설치된 Moodle에 첨가 가능
  • 플러그 인 언어팩으로 어떤 국가의 언어든 전반적인 지역화 가능. 언어팩은 내장된 편집기를 이용하여 수정 가능. 현재 43개국 이상의 언어팩이 있음.
  • 개인의 용도에 따라 자신에 맞도록 쉽게 수정할 수 있는 GPL license 하의 PHP 코드로 작성됨

사용자 관리

  • 높은 수준의 보안을 유지하는 가운데 관리자의 업무 부담 최소화를 목표로 함
  • 플러그인 인증 모듈을 통하여 인증 기능을 확장, 쉽게 기존 시스템 성능을 최대로 활용 가능
  • 표준 이메일 기능 : 학생 개인 접속 계정 생성 가능. 이메일로 인증
  • LDAP 적용가능 : LDAP서버를 이용한 접속 계정 점검. 관리자는 사용되는 항목을 지정할 수 있음
  • IMAP, POP3, NNTP: 메일 서버 혹은 뉴스 서버에 대해 계정 접속을 점검할 수 있음. SSL, TLS 지원
  • 외부 데이터베이스 : 두 항목이상 포함하는 어떠한 외부 데이터베이스도 외부 인증 정보로 사용 가능
  • 개인은 전 서버에 걸쳐 단 하나의 계정으로 충분 - 각 계정은 다른 접속 가능
  • 관리자 계정으로는 배움터의 생성 및 배움터 사용자의 교사 배정을 함
  • 배움터를 만든 사람의 계정으로는 단지 과정을 운영하고 소속한 사람들을 가르침
  • (임시교사의 경우)교사는 편집 권한을 제한받을 수 있고 그 경우 배움터를 함부로 고칠 수 없음
  • 보안 - "참여키"를 활용하여 허용된 학생이 아닌 경우 배움터에 함부로 접속하는 것을 제한할 수 있음. "참여키"는 면대면 상황에서 일러줄 수도 있고 이메일 등을 통해 전달할 수도 있음
  • 교사가 원하면 수동으로 학생들의 출결을 조정할 수 있음
  • 교사는 수동으로 학생을 퇴출을 시킬 수 있으며, 또 일정 기간동안 활동을 하지 않는 경우(관리자에 의해 조정됨) 자동적으로 퇴출될 수도 있음
  • 학생들은 자신들의 사진, 소개, 이메일주소 등을 포함하는 신상명세를 작성할 수 있으며 이를 공개하거나 공개하지 않을 수 있음
  • 모든 사용자는 자신이 쓰는 시간대를 지정할 수 있으며, 모든 시각(게시 날자, 약속 도래일 등)들은 무들이 알아서 개인의 시간대에 맞게 변경해 줌
  • 모든 사용자는 무들 인터페이스에 맞게 자신의 언어(한국어, 영어, 불어, 독일어, 스페인어 등)를 지정할 수 있음

배움터(과정) 관리

  • 정식 교사는 자신이 생성한 배움터에 대하여 다른 교사들의 활동을 제한하는 것을 포함해, 전적인 조정권한을 가짐
  • 주별, 주제별, 사회 토론형 중에서 배움터 유형을 선택 가능
  • 다양한 활동 - 포럼, 퀴즈, 자원, 선택, 설문, 숙제, 채팅, 워크숍 -의 유연한 조합 가능
  • 공동체의 이해가 용이하도록 배움터의 갱신 내용이 배움터 초기 화면에 게시됨
  • (자원, 포럼의 게시물 등) 대다수의 문서영역은 내장된 WYSIWYG HTML 편집기를 통해 편집 가능
  • 포럼, 퀴즈 그리고 숙제 등에 대한 모든 점수는 한 화면을 통해 볼 수 있으며 스프레드쉬트 형식으로 내려 받을 수 있음
  • 완벽한 사용자 관리기록 및 추적 보고 - 각 학생들의 게시글 등은 물론, 각 모듈에 대한 상세한 활동 보고(마지막 접속, 페이지 뷰 등)가 그래프 및 문서로 제공됨
  • 이메일 통합 - 게시글, 교사의 피드백 등이 이메일을 통해 텍스트나 HTML 형태로 전달될 수 있음
  • 적절한 등급 척도 - 교사는 포럼이나 숙제에 대하여 스스로 실정에 맞게 평가 척도를 정의하여 사용 가능
  • 교과 과정은 백업 기능을 이용하여 하나의 압축파일로 만들 수 있음. 추후 어떤 무들 서버에서도 재복구 가능

숙제 모듈

  • 숙제는 제출 기한 및 최대 등급을 지정할 수 있음
  • 학생들은 서버에 어떤 파일의 형태로든 숙제를 올릴 수 있으며 올린 숙제에는 올린 날짜가 명기됨
  • 늦게 제출한 숙제도 허용될 수 있으나, 얼마나 늦게 냈는지에 대한 정보가 교사에게 제공됨
  • 각각의 숙제에 대하여 전체 반에 대한 등급과 평이 하나의 양식에 만들어져 보고됨
  • 교사의 피드백은 각 학생의 숙제 페이지에 첨부됨과 동시에, 학생들에게 이메일로 통보됨
  • 교사는 등급판정 후 (재등급을 위해)숙제를 다시 받을 것인지에 대해 선택할 수 있음

채팅 모듈

  • 자연스럽고 즉각적인 문서의 반응을 보이는 대화창
  • 대화창에서 대화 상대자의 사진을 볼 수 있음
  • 상대방의 URL, 이모티콘, 이미지, 내장 HTML 등을 지원
  • 학생들이 나중에 보거나 활용할 수 있도록 모든 세션이 기록 가능

설문 모듈

  • 설문 조사와 비슷. 어떤 사안에 대해 투표를 하거나, (research consent 같은)학생들의 반응을 조사할 때 활용
  • 교사는 표를 통해 누가 어떤 선택을 했는지 즉각적으로 볼 수 있음
  • 학생들이 최근의 결과를 그래프를 통해 볼 수 있도록 조정 가능

포럼 모듈

  • 다양한 형태의 포럼(교사만이 주제를 올릴 수 있는 형태, 배움터 소식, 모든 사람에게 허용되는 형태, 한 사람이 하나의 주제를 갖는 형태)을 활용할 수 있음
  • 게시글에 쓴 사람의 사진이 첨부됨
  • 토론은 게시판 형, 중첩형, 새문서 먼저, 예전글 먼저 등 다양한 형태로 게시될 수 있음
  • 개개의 포럼에서 구독자들이 원하면 게시글이 이메일로 전달될 수 있으며, 교사가 강제적으로 구독하게도 할 수도 있음
  • (공지글과 같은) 포럼의 게시물은 답장을 받지 못하도록 조치할 수 있음
  • 토론 글타래는 교사에 의해 다른 포럼으로 쉽게 이동될 수 있음
  • 첨부된 이미지는 온라인 상에서 쉽게 볼 수 있음
  • 포럼 등급매기기를 실시할 경우, 그 기한을 조절할 수 있음

퀴즈 모듈

  • 교사들은 각기 다른 퀴즈에서 재 사용 가능한 문항의 데이터베이스를 구축 가능
  • 질문들은 접근하기 쉽게 범주화시켜 저장할 수 있으며, 각 범주는 "공유"할 수 있어 사이트의 어떤 배움터에서도 활용 가능
  • 퀴즈 결과는 자동으로 점수가 매겨지며 문제가 바뀌었을 때 재 점수 부여 가능
  • 퀴즈 사용 시간 및 특정 접속 영역에서만 퀴즈를 풀 수 있게 제한 가능
  • 교사 옵션을 통해 학생들이 여러번 퀴즈에 도전하게 할 수도 있고 답안에 대한 피드백을 해 줄 수도 있음
  • 퀴즈의 문제와 답안을 뒤섞어 학생들의 부정행위를 줄일 수 있음
  • 문제는 HTML 이나 그림을 사용할 수 있음
  • 외부 문서파일을 불러들여 문항을 만들 수 있음
  • 원한다면 문제풀이 시도를 누적시킬 수 있으며, 단계별로 마칠 수도 있음
  • 선다형 문형은 답이 하나일 경우는 물론 답이 여러 개 일 경우도 처리할 수 있음
  • 단답형(단어 혹은 문장) 문항도 처리
  • OX 문형 처리 가능
  • 짝짓기 문형 제공
  • 무작위 문형 가능
  • 오차를 처리할 수 있는 계산 문제 가능
  • 완성형(괄호넣기) 문항 작성 가능
  • 문자와 그래픽을 혼용해서 작성 가능

자원 모듈

  • 워드문서, 파워포인트, 플래시, 비디오, 사운드 등 거의 모든 디지탈 내용을 출력 지원
  • 파일들은 서버에 업로드 및 관리 가능하며, or files can be created on the fly using web forms (text or HTML)
  • 웹에 게시된 외부의 내용도 연결 가능하거나 자연스럽게 배움터에 포함 가능
  • 외부의 웹 응용프로그램도 데이터를 전송하는 방법으로 연결 가능

설문조사 모듈

  • 온라인 강의 분석의 도구로서 검증된 내장된 설문(COLLES, ATTLS)
  • 많은 그래프를 포함한 항상 활용가능한 온라인 설문 보고서. 데이터는 엑셀 스프레드 쉬트 혹은 CVS 문서 파일로 내려받을 수 있음.
  • 완성이 안된 조사는 분석되지 않도록 설계됨
  • 학급 평균과 비교한 학생들 자료만 피드백으로 제공

웍크숍 모듈

  • 문서에 대한 동료평가가 허용될 수 있으며 교사는 평가 등급을 관리 가능
  • 가능한 평가 척도에 대한 광범위한 지원
  • 학생들이 등급을 메기는 연습을 하도록 예제 문서 제공 가능
  • 유연성있는 다양한 옵션
